# แปลง OneNote เป็น PDF และบันทึกเป็น Stream – Aspose.Note

## บทนำ

ในบทแนะนำนี้คุณจะได้เรียนรู้วิธี **แปลง OneNote เป็น PDF** และ **บันทึกสตรีม PDF** ลงในหน่วยความจำโดยใช้ Aspose.Note สำหรับ Java การสตรีม PDF ให้คุณควบคุมได้เต็มที่ว่าผลลัพธ์จะไปที่ไหน—ไม่ว่าจะต้อง **สร้าง PDF จาก OneNote** สำหรับบริการเว็บ, เก็บไว้ในฐานข้อมูล, หรือส่งต่อให้ส่วนประกอบอื่นโดยไม่ต้องใช้ไฟล์ระบบ เราจะอธิบายขั้นตอนทั้งหมด ตั้งแต่การโหลดไฟล์ OneNote ไปจนถึงการส่งออกเป็นสตรีม PDF เพื่อให้คุณสามารถผสานความสามารถนี้เข้าไปในแอปพลิเคชัน Java ของคุณได้อย่างมั่นใจ

## คำตอบอย่างรวดเร็ว
- **อะไรหมายถึง “convert OneNote to PDF”?** มันจะแปลงไฟล์ OneNote `.one` ให้เป็นเอกสาร PDF และเขียนผลลัพธ์ลงในสตรีมแทนการสร้างไฟล์จริง  
- **ทำไมต้องใช้สตรีม?** สตรีมช่วยให้คุณจัดการข้อมูลในหน่วยความจำ เหมาะสำหรับบริการเว็บ, API หรือเมื่อคุณต้องการหลีกเลี่ยงไฟล์ชั่วคราว  
- **รูปแบบ Aspose.Note ที่ใช้คืออะไร?** enum `SaveFormat.Pdf` บอกไลบรารีให้ส่งออกเป็น PDF  
- **ต้องมีลิขสิทธิ์สำหรับการใช้งานในผลิตภัณฑ์หรือไม่?** ใช่—Aspose.Note ต้องการลิขสิทธิ์ที่ถูกต้องสำหรับการใช้งานเชิงพาณิชย์  
- **ฉันสามารถส่งออกเป็นรูปแบบอื่นได้หรือไม่?** แน่นอน—ใช้ค่า `SaveFormat` อื่น ๆ เช่น `Docx`, `Html`, `Png` เป็นต้น  

## การแปลง OneNote เป็น PDF คืออะไร?

การแปลง OneNote เป็น PDF หมายถึงการนำเนื้อหาที่หลากหลายและหลายหน้าในโน้ตบุ๊ก OneNote มาทำให้เป็นเอกสาร PDF ที่พกพาได้ ซึ่งมีประโยชน์เมื่อคุณต้องการเวอร์ชันอ่าน‑อย่างเดียวที่สามารถดูได้ทั่วโลก หรือเมื่อต้องฝังเนื้อหาเข้าไปในกระบวนการทำงานอื่น ๆ เช่น อีเมล รายงาน หรือการเก็บถาวร

## ทำไมต้องสตรีม PDF ใน Java?

การสตรีม PDF ใน Java ช่วยหลีกเลี่ยงการเขียนไฟล์ชั่วคราวลงดิสก์ ทำให้คุณสามารถ:

* ส่ง PDF ตรงไปยังการตอบสนอง HTTP
* เก็บอาร์เรย์ไบต์ในคอลัมน์ BLOB ของฐานข้อมูล
* เชื่อมต่อผลลัพธ์กับไลบรารีอื่น (เช่น การเข้ารหัสด้วย Aspose.PDF) โดยไม่ต้องทำ I/O ระหว่างขั้นตอน

## นำเข้าแพ็กเกจ

ก่อนอื่นให้ทำการนำเข้าคลาสที่จำเป็น การจัดการ import ให้เป็นระเบียบช่วยให้โค้ดอ่านง่ายและบำรุงรักษาได้ดีขึ้น.

```java
import java.io.ByteArrayOutputStream;
import java.io.IOException;
import com.aspose.note.Document;
import com.aspose.note.SaveFormat;
```

## ขั้นตอนที่ 1: โหลดเอกสาร OneNote

โหลดไฟล์ OneNote ต้นฉบับเข้าไปในอ็อบเจ็กต์ `Aspose.Note` `Document` แทนที่พาธตัวอย่างด้วยตำแหน่งจริงของไฟล์ `.one` ของคุณ.

```java
String dataDir = "Your Document Directory";
Document doc = new Document(dataDir + "Sample1.one");
```

## ขั้นตอนที่ 2: บันทึกเอกสารเป็นสตรีม

ต่อไปเราจะส่งออกเอกสารที่โหลดแล้วเป็น PDF และเขียนลงใน `ByteArrayOutputStream` สตรีมนี้สามารถส่งตรงไปยังไคลเอนต์ เก็บในฐานข้อมูล หรือทำการประมวลผลต่อได้.

```java
ByteArrayOutputStream dstStream = new ByteArrayOutputStream();
doc.save(dstStream, SaveFormat.Pdf);
```

### วิธี **ส่งออกอาร์เรย์ไบต์ PDF** ไปยังปลายทางอื่น

หากคุณต้องการ PDF เป็นอาร์เรย์ไบต์ เพียงเรียก `dstStream.toByteArray()` สำหรับการตอบสนองเว็บ ให้เขียนอาร์เรย์ไบต์ลงในสตรีมเอาต์พุตของ HTTP วิธีเดียวกันใช้ได้กับรูปแบบอื่น—แค่เปลี่ยน `SaveFormat.Pdf` เป็นค่า enum ที่ต้องการ.

## ปัญหาที่พบบ่อยและวิธีแก้

- **OutOfMemoryError** – เมื่อจัดการไฟล์ OneNote ขนาดใหญ่มาก ควรใช้ `FileOutputStream` เพื่อเขียนโดยตรงลงดิสก์แทนการเก็บทั้งหมดในหน่วยความจำ.  
- **Missing fonts** – PDF อาจสูญเสียฟอนต์ที่กำหนดเองหากไม่ได้ติดตั้งบนเซิร์ฟเวอร์ ใช้ `FontSettings` เพื่อฝังฟอนต์หากจำเป็น.  
- **License not found** – ตรวจสอบให้แน่ใจว่าไฟล์ลิขสิทธิ์ถูกโหลดก่อนเรียกใช้ API ของ Aspose.Note มิฉะนั้นจะเห็นลายน้ำรุ่นทดลอง.
