## How to Change Style in OneNote
การเปลี่ยนสไตล์ใน OneNote ง่ายเพียงเรียกใช้เมธอดบางอย่างบนอ็อบเจ็กต์ `RichText` ด้านล่างเราจะสรุปแนวคิดหลักก่อนที่จะลงลึกในสองบทเรียนโดยละเอียดที่ตามมา

### Core Concepts
- **Text Formatting:** ชื่อฟอนต์, ขนาด, สี, และการไฮไลท์ถูกควบคุมผ่านคุณสมบัติของ `TextRun`.  
- **Paragraph Formatting:** สไตล์ย่อหน้าเริ่มต้น, การจัดแนว, และการเยื้องสามารถตั้งค่าที่ระดับส่วนได้.  
- **Styling Scope:** สามารถใช้สไตล์กับรันเดียว, ย่อหน้าเต็ม, หรือทั่วทั้งเอกสารได้.

**Q:** *Can I apply the same style to multiple paragraphs at once?*  
A: ได้, คุณสามารถวนลูปผ่านคอลเลกชันของย่อหน้าและกำหนดอ็อบเจ็กต์ `ParagraphStyle` ที่ใช้ร่วมกัน

**Q:** *Is it possible to change the default font for the entire notebook?*  
A: แน่นอน. ตั้งค่าสไตล์ย่อหน้าเริ่มต้นที่ระดับสมุดบันทึกและทุกหน้าที่สร้างใหม่จะสืบทอดค่าเหล่านั้น

**Q:** *Do these APIs work with encrypted OneNote files?*  
A: API สามารถเปิดไฟล์ที่มีการป้องกันด้วยรหัสผ่านได้เมื่อคุณให้รหัสผ่านถอดรหัสในระหว่างการโหลด

**Q:** *What happens if I set a font size that exceeds OneNote’s limits?*  
A: Aspose.Note จะปรับขนาดให้เป็นค่าที่ใกล้เคียงที่สุดที่รองรับ เพื่อให้เอกสารยังคงเป็นที่ถูกต้อง

**Q:** *Can I preview the styled note before saving?*  
A: ได้, คุณสามารถเรนเดอร์เอกสารเป็น HTML หรือ PDF เพื่อดูผลลัพธ์แบบทันที
