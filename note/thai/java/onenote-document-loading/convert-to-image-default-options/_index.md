ก่อนเริ่มทำงาน ตรวจสอบให้แน่ใจว่าติดตั้งและตั้งค่าตามนี้แล้ว:

### Aspose.Note for Java
1. **ดาวน์โหลด** ไลบรารีจาก [หน้าดาวน์โหลด Aspose.Note for Java](https://releases.aspose.com/note/java/)

## นำเข้าแพ็กเกจ
ขั้นตอนแรกคือการนำเข้าคลาสที่จำเป็นสำหรับการโหลดไฟล์ OneNote และบันทึกเป็นภาพ

```java
import java.io.IOException;
import com.aspose.note.Document;
import com.aspose.note.LoadOptions;
import com.aspose.note.SaveFormat;
```

## คู่มือแบบขั้นตอน

### ขั้นตอนที่ 1: โหลดเอกสาร OneNote
โหลดไฟล์ `.one` ต้นฉบับเข้าสู่วัตถุ `Aspose.Note` `Document` ตัวสร้าง `LoadOptions` ที่ไม่มีพารามิเตอร์บ่งบอกให้ไลบรารีใช้พฤติกรรมการโหลดเริ่มต้น

```java
// Load the document into Aspose.Note.
String dataDir = "Your Document Directory";
Document oneFile = new Document(dataDir + "Sample1.one", new LoadOptions());
```

> **เคล็ดลับ:** ให้ `dataDir` ชี้ไปยังพาธแบบเต็มหรือใช้ `Paths.get(...)` เพื่อความเข้ากันได้ข้ามแพลตฟอร์มที่ดียิ่งขึ้น

### ขั้นตอนที่ 2: บันทึกเอกสารเป็นภาพ
เรียกเมธอด `save` ระบุชื่อไฟล์ผลลัพธ์ที่ต้องการและเลือกรูปแบบภาพผ่าน `SaveFormat` ตัวอย่างด้านล่างบันทึกหน้าแรกเป็น GIF แต่คุณสามารถเปลี่ยน `SaveFormat.Gif` เป็น `SaveFormat.Png`, `SaveFormat.Jpeg` ฯลฯ เพื่อ **แปลง OneNote เป็น PNG** หรือรูปแบบอื่นได้

```java
// Save the document as Gif.
oneFile.save(dataDir + "ConvertToImageUsingDefaultOptions_out.gif", SaveFormat.Gif);
```

**สิ่งที่เกิดขึ้นเบื้องหลัง**  
Aspose.Note จะเรนเดอร์แต่ละหน้าเป็นบิตแมพ แล้วเข้ารหัสด้วยโค้ดเดคภาพที่เลือก เนื่องจากเราใช้ตัวเลือกเริ่มต้น ไลบรารีจะกำหนดขนาดหน้า, DPI, และความลึกสีโดยอัตโนมัติ

## คำถามที่พบบ่อย

**ถาม: ฉันสามารถแปลงโน้ตบุ๊ก OneNote ที่มีหลายหน้าเป็นภาพแยกแต่ละหน้าได้หรือไม่?**  
ตอบ: ได้. ทำการวนลูป `oneFile.getPages()` แล้วเรียก `save` สำหรับแต่ละหน้า พร้อมกำหนดชื่อไฟล์ที่ไม่ซ้ำกัน

**ถาม: จะเปลี่ยนความละเอียดของภาพได้อย่างไร?**  
ตอบ: ใช้ `ImageSaveOptions` เพื่อกำหนด DPI ก่อนบันทึก: `ImageSaveOptions options = new ImageSaveOptions(SaveFormat.Png); options.setResolution(300); oneFile.save("out.png", options);` วิธีนี้เป็นวิธีแนะนำเพื่อ **ตั้งค่าความละเอียดภาพใน Java**

**ถาม: สามารถแปลง OneNote โดยตรงเป็น PDF แทนภาพได้หรือไม่?**  
ตอบ: แน่นอน. แค่เปลี่ยน `SaveFormat.Gif` เป็น `SaveFormat.Pdf` เพื่อสร้างเอกสาร PDF

**ถาม: เวอร์ชันทดลองมีลายน้ำบนภาพผลลัพธ์หรือไม่?**  
ตอบ: ไม่มี. เวอร์ชันทดลองสร้างภาพคุณภาพเต็มโดยไม่มีลายน้ำ; ต้องมีไลเซนส์เท่านั้นสำหรับการใช้งานเชิงพาณิชย์

**ถาม: รูปแบบภาพใดให้ขนาดไฟล์เล็กที่สุด?**  
ตอบ: GIF และ JPEG มักให้ไฟล์ขนาดเล็กกว่า PNG, แต่ควรเลือกตามความต้องการคุณภาพ—PNG ไม่เสียคุณภาพ, JPEG มีการบีบอัดสูญเสีย

## คำถามที่พบบ่อยเพิ่มเติม

**ถาม: ฉันสามารถแปลง OneNote เป็น PDF ในขั้นตอนเดียวกันได้หรือไม่?**  
ตอบ: ได้—เพียงเปลี่ยน `SaveFormat` เป็น `SaveFormat.Pdf` ไลบรารีจะสร้างไฟล์ PDF ของโน้ตบุ๊ก

**ถาม: จะตั้งค่าความละเอียดของภาพใน Java อย่างไรเมื่อบันทึกหลายหน้า?**  
ตอบ: สร้างอินสแตนซ์ `ImageSaveOptions`, ตั้งค่า DPI ที่ต้องการ, แล้วส่งผ่านไปยังเมธอด `save` ของแต่ละหน้า วิธีนี้ทำให้คุณ **ตั้งค่าความละเอียดภาพใน Java** อย่างสม่ำเสมอสำหรับไฟล์ผลลัพธ์ทั้งหมด

**ถาม: มีเคล็ดลับการเพิ่มประสิทธิภาพสำหรับการแปลงโน้ตบุ๊กจำนวนมากเป็นชุดหรือไม่?**  
ตอบ: ประมวลผลโน้ตบุ๊กต่อเนื่อง, ใช้วัตถุ `ImageSaveOptions` เดียวกันซ้ำ, และทำลาย (`dispose`) `Document` หลังบันทึกแต่ละไฟล์เพื่อคืนหน่วยความจำ
