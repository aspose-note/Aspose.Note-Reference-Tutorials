# แปลงสมุดบันทึกเป็นภาพแบนใน OneNote - Aspose.Note

## บทนำ

ในบทแนะนำนี้คุณจะได้เรียนรู้วิธี **แปลง OneNote เป็น PNG** โดยการเปลี่ยนสมุดบันทึกทั้งหมดให้เป็นภาพแบนเดียวโดยใช้ Aspose.Note สำหรับ Java วิธีนี้เหมาะอย่างยิ่งเมื่อคุณต้องการแชร์สมุดบันทึกเป็นภาพคงที่ ฝังลงในรายงาน หรือเก็บเป็นเอกสารโดยไม่สูญเสียรายละเอียดภาพใดๆ

## คำตอบด่วน
- **‘flatten notebook’ หมายถึงอะไร?** มันรวมองค์ประกอบทั้งหมดของหน้าเป็นภาพราสเตอร์เดียว.  
- **รูปแบบใดที่ใช้?** PNG เป็นค่าเริ่มต้น ให้คุณภาพไม่มีการสูญเสีย.  
- **ฉันสามารถเปลี่ยน DPI ได้หรือไม่?** ได้, ใช้ `setResolution` บน `ImageSaveOptions`.  
- **ต้องการไลเซนส์หรือไม่?** รุ่นทดลองฟรีใช้ได้สำหรับการประเมิน; จำเป็นต้องมีไลเซนส์เชิงพาณิชย์สำหรับการใช้งานจริง.  
- **สนับสนุนบนระบบปฏิบัติการทั้งหมดหรือไม่?** API ของ Java ทำงานได้ทุกที่ที่ Java ทำงาน.

## การแปลง OneNote เป็น PNG คืออะไร?

การแปลง OneNote เป็น PNG จะสร้างภาพบิตแมพของทุกหน้าภายในสมุดบันทึก โดยคงข้อความ, การวาด, และวัตถุที่ฝังอยู่ในไฟล์ภาพเดียว วิธีนี้มีประโยชน์อย่างยิ่งสำหรับเอกสาร, การนำเสนอ, หรือการเก็บข้อมูลเพื่อการปฏิบัติตามกฎระเบียบ

## ทำไมต้องแปลง OneNote เป็น PNG ด้วย Aspose.Note?

- **ความแม่นยำเต็มรูปแบบ** – ทุกองค์ประกอบภาพถูกเก็บไว้ทั้งหมด  
- **ผลลัพธ์เป็นไฟล์เดียว** – ไม่จำเป็นต้องจัดการไฟล์หลายหน้า  
- **ความละเอียดที่ปรับได้** – ปรับ DPI ให้ตรงตามความต้องการคุณภาพ  
- **ไม่ต้องติดตั้ง Office** – ทำงานอย่างอิสระจาก Microsoft OneNote  

## ข้อกำหนดเบื้องต้น

ก่อนเริ่ม, โปรดตรวจสอบว่าคุณมีสิ่งต่อไปนี้:

1. Java Development Kit (JDK) ติดตั้งบนระบบของคุณ.  
2. ไลบรารี Aspose.Note สำหรับ Java ดาวน์โหลดและตั้งค่าในโปรเจค Java ของคุณ คุณสามารถดาวน์โหลดไลบรารีได้จาก [ที่นี่](https://releases.aspose.com/note/java/).  
3. ความรู้พื้นฐานของการเขียนโปรแกรม Java.

## นำเข้าแพ็กเกจ

เพื่อเริ่มต้น, คุณต้องนำเข้าแพ็กเกจที่จำเป็นจาก Aspose.Note สำหรับ Java:

```java
import java.io.IOException;
import com.aspose.note.ImageSaveOptions;
import com.aspose.note.Notebook;
import com.aspose.note.NotebookImageSaveOptions;
import com.aspose.note.SaveFormat;
```

## ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรีเอกสาร

แรกสุด, กำหนดไดเรกทอรีที่ไฟล์สมุดบันทึกของคุณอยู่:

```java
String dataDir = "Your Document Directory";
```

แทนที่ `"Your Document Directory"` ด้วยพาธไปยังไฟล์สมุดบันทึกของคุณ

## ขั้นตอนที่ 2: โหลดสมุดบันทึก

ต่อไป, โหลดไฟล์สมุดบันทึกโดยใช้คลาส `Notebook`:

```java
Notebook notebook = new Notebook(dataDir + "test.onetoc2");
```

ตรวจสอบให้แน่ใจว่าได้แทนที่ `"test.onetoc2"` ด้วยชื่อไฟล์สมุดบันทึกของคุณ

## ขั้นตอนที่ 3: ตั้งค่าตัวเลือกการบันทึกภาพ

ตอนนี้, ตั้งค่าตัวเลือกสำหรับบันทึกสมุดบันทึกเป็นภาพ เราจะระบุรูปแบบการบันทึกเป็น PNG และตั้งความละเอียดเป็น 400 DPI:

```java
NotebookImageSaveOptions saveOptions = new NotebookImageSaveOptions(SaveFormat.Png);
ImageSaveOptions documentSaveOptions = saveOptions.getDocumentSaveOptions();
documentSaveOptions.setResolution(400);
```

คุณสามารถปรับความละเอียดตามความต้องการของคุณ นี่คือจุดที่คุณ **ตั้งค่าความละเอียดของภาพ** เพื่อควบคุมคุณภาพของผลลัพธ์

## ขั้นตอนที่ 4: แบนสมุดบันทึก

เพื่อให้แน่ใจว่าทุกองค์ประกอบถูกแบนเป็นภาพเดียว, ตั้งค่าตัวเลือก `flatten` เป็น `true`:

```java
saveOptions.setFlatten(true);
```

การตั้งค่า `flatten` เป็น `true` รับประกันว่าภาพ PNG ที่ได้จะคงรูปแบบที่ตรงกับสมุดบันทึกของคุณ

## ขั้นตอนที่ 5: บันทึกภาพ

สุดท้าย, บันทึกสมุดบันทึกเป็นภาพแบน:

```java
notebook.save(dataDir + "ExportImageasFlattenedNotebook_out.png", saveOptions);
```

แทนที่ `"ExportImageasFlattenedNotebook_out.png"` ด้วยชื่อไฟล์ผลลัพธ์ที่คุณต้องการ ขั้นตอนนี้ **บันทึก OneNote เป็นภาพ** ที่คุณสามารถแชร์หรือฝังได้ทุกที่

## กรณีการใช้งานทั่วไป

- **เอกสาร:** ฝังภาพสมุดบันทึกในคู่มือเทคนิคหรือคู่มือผู้ใช้  
- **การนำเสนอ:** ใช้สไลด์ PNG ความละเอียดสูงใน PowerPoint โดยไม่ต้องกังวลเรื่องฟอนต์หรือความเข้ากันของวัตถุ  
- **การเก็บถาวร:** เก็บภาพสแนปช็อตแบบอ่านอย่างเดียวของสมุดบันทึกสำหรับการตรวจสอบตามข้อกำหนด  

## เคล็ดลับการแก้ไขปัญหา

- **ไฟล์ไม่พบ:** ตรวจสอบพาธ `dataDir` อีกครั้งและยืนยันว่าไฟล์ `.onetoc2` มีอยู่  
- **ภาพคุณภาพต่ำ:** เพิ่ม DPI โดยเปลี่ยนเป็น `documentSaveOptions.setResolution(600);`  
- **องค์ประกอบหาย:** ตรวจสอบว่าได้เปิดใช้งาน `saveOptions.setFlatten(true);` หากไม่เปิดใช้งานบางเลเยอร์อาจยังคงแยกกันอยู่  

## คำถามที่พบบ่อย

**Q1: ฉันสามารถปรับความละเอียดของภาพผลลัพธ์ได้หรือไม่?**  
A1: ได้, คุณสามารถปรับความละเอียดตามความต้องการของคุณโดยแก้ไขพารามิเตอร์ของเมธอด `setResolution`

**Q2: Aspose.Note รองรับรูปแบบภาพอื่นสำหรับการส่งออกหรือไม่?**  
A2: ได้, Aspose.Note รองรับรูปแบบภาพหลายประเภท เช่น PNG, JPEG, BMP ฯลฯ สำหรับการส่งออกสมุดบันทึก

**Q3: ฉันสามารถปรับแต่งภาพผลลัพธ์เพิ่มเติมได้หรือไม่?**  
A3: ได้, Aspose.Note มีตัวเลือกหลากหลายสำหรับการปรับแต่งภาพผลลัพธ์ รวมถึงขนาดหน้า, การวางแนว, และการตั้งค่าคุณภาพ

**Q4: มีเวอร์ชันทดลองสำหรับ Aspose.Note สำหรับ Java หรือไม่?**  
A4: มี, คุณสามารถรับเวอร์ชันทดลองฟรีได้จาก [ที่นี่](https://releases.aspose.com/).

**Q5: ฉันจะหาแหล่งสนับสนุนสำหรับ Aspose.Note สำหรับ Java ได้จากที่ไหน?**  
A5: คุณสามารถหาแหล่งสนับสนุนและทรัพยากรได้ในฟอรั่ม Aspose.Note [ที่นี่](https://forum.aspose.com/c/note/28).
