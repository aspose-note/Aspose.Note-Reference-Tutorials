## 常见问题

**Q: 我可以在商业项目中使用 Aspose.Note for Java 吗？**  
A: 是的，生产环境需要商业许可证，但可使用免费试用进行评估。

**Q: 支持哪些 Java 版本？**  
A: 该库支持 Java 8 及更高版本。

**Q: 在检索附件时如何处理大型 OneNote 文件？**  
A: 可以以流模式加载文档，或逐个处理章节/页面，以降低内存消耗。

**Q: 能否为附加的文件设置自定义图标？**  
A: 当然可以——“Attach File and Set Icon” 教程展示了如何以编程方式指定图标。

**Q: 使用 API 是否需要在服务器上安装 OneNote？**  
A: 不需要，Aspose.Note 可独立于 OneNote 应用运行，直接读取和写入 OneNote 文件格式。
