## 常见问题

**Q: 能否直接将 OneNote 笔记本导出为 PDF 而无需中间格式？**  
A: 可以。Aspose.Note 提供一步完成的 `save` 方法，可将笔记本写入 PDF 文件，保持布局和图像。

**Q: 如何仅提取 OneNote 页面中的图像？**  
A: 使用 `Document.getImages()` API 枚举图像对象，并将每个图像保存为所需的格式（PNG、JPEG 等）。

**Q: 有办法提升大型笔记本的导出速度吗？**  
A: 在保存前调用 `Document.optimizeResources()` 方法，并考虑在并行线程中处理各章节。

**Q: 服务器上需要安装额外的软件吗？**  
A: 不需要。Aspose.Note 是纯 Java 库，只需将 JAR 包加入项目的 classpath 即可。

**Q: 生产使用有哪些授权选项？**  
A: Aspose 提供永久授权和订阅模式。可免费获取试用版进行评估。
