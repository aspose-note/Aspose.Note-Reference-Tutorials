## 常见问题

**Q: 我可以在商业项目中使用 Aspose.Note for Java 吗？**  
A: 可以。生产使用需要有效的商业许可证，但可提供免费试用进行评估。

**Q: 支持哪些 Java 版本？**  
A: 该库支持 Java 8、11 以及更新的 LTS 版本。

**Q: 如何向 OneNote 页面添加超链接？**  
A: 使用 Aspose.Note 提供的 `Hyperlink` 类定义 URL 并将其附加到 `RichText` 节点。

**Q: 能否为图像设置可访问性的替代文本？**  
A: 完全可以。`Image` 对象具有 `AltText` 属性，您可以通过编程方式设置它。

**Q: 大型笔记本的性能技巧有哪些？**  
A: 启用计量许可证，尽可能复用 `Document` 实例，并对大型资源使用流式处理，而不是完全加载到内存中。
