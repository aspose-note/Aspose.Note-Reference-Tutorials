## 什么是程序化创建 OneNote 表格？
`Programmatically create OneNote table` 指使用代码——具体而言是 Aspose.Note for Java API——在 OneNote 页面内部生成 Table 对象，而无需手动用户交互。这种方式实现文档生成自动化，确保一致性，并能在大规模工作负载下扩展。它使开发者能够直接从 Java 应用程序构建表格，节省时间并降低错误率。

## 如何程序化创建 OneNote 表格？

`Document` 表示已加载到内存中的 OneNote 笔记本文件。  
`Page.getTables().add()` 在页面上创建一个具有指定行列数的新 Table。

加载一个 `Document` 实例，添加一个 `Page`，然后调用 `Page.getTables().add()` 并传入所需的行列数。创建后，您可以设置单元格值、应用边框并设置表格样式——全部通过流畅的 Java 调用完成。这种两步模式（创建 → 配置）让您能够瞬间构建表格，即使是多页笔记本也不例外。
