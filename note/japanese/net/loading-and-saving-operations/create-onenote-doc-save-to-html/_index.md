# Aspose.Note を使用して OneNote を HTML として保存する

## はじめに

Web 公開、レポート作成、またはアーカイブのために **OneNote を HTML として保存** したい場合、Aspose.Note for .NET は Microsoft Office を必要とせず、Windows、Linux、macOS で動作するクリーンなコードベースのソリューションを提供します。このチュートリアルでは、ゼロから OneNote ドキュメントを作成し、ページを操作し、さまざまなオプションを使用してノートブック全体または選択したページを HTML にエクスポートする方法を学びます。ガイドはすべての手順を順に説明し、各設定が重要な理由を解説し、HTML エクスポートが活躍する実践シナリオ（例: ナレッジベースポータルへの OneNote コンテンツ埋め込みや静的ドキュメントの生成）を示します。

## クイック回答
- **エクスポートの主な方法は？** `Aspose.Note.Document` インスタンスで `document.Save("output.html", SaveFormat.Html)` を呼び出します。  
- **単一ページだけをエクスポートできますか？** はい—`document.Save(pageRange, "page.html", SaveFormat.Html)` を使用します。  
- **リソース（画像、CSS）は自動的に埋め込まれますか？** デフォルトでは別ファイルとして保存されますが、`HtmlSaveOptions` でメモリストリームに埋め込むことが可能です。  
- **.NET Core はサポートされていますか？** 完全にサポートしています—.NET 5、.NET 6、.NET Core 3.1+ で動作します。  
- **開発用にライセンスは必要ですか？** 無料の 30 日間トライアルでテスト可能です。ライセンスを取得すると評価用の透かしが除去されます。

## “save onenote as html” とは？

**“Save OneNote as HTML”** は、専用のバイナリ形式で保存された OneNote ノートブックを、任意のウェブブラウザで表示できる標準的な HTML ファイルに変換するプロセスを指します。変換はノートブックの階層構造、テキスト書式、画像、表、その他のリソースを保持しつつ、OneNote 固有の要素を同等の HTML タグと CSS スタイルに置き換えます。これにより、開発者は OneNote コンテンツをウェブサイトに表示したり、CMS に統合したり、クライアントなしで汎用的にアクセス可能な形式でアーカイブしたりできます。

## なぜ OneNote を HTML に変換するのか？

Aspose.Note は **50 以上の出力形式**（PDF、DOCX、PNG、HTML など）にエクスポートでき、**最大 500 ページ** のノートブックをメモリ全体にロードせずにストリーミング アーキテクチャで処理します。HTML への変換により次のことが可能になります。

1. **Web 上でコンテンツを公開** でき、OneNote クライアントは不要です。  
2. **HTML のみを受け付けるコンテンツ管理システムと統合** できます。  
3. **検索エンジン用にテキストを抽出** してインデックス化できます。  
4. **ページ単位のリソース**（画像、フォント）を他の Web プロジェクトで再利用できます。

## OneNote を HTML として保存する手順 – ステップバイステップ

OneNote を HTML に保存するには、ノートブックの読み込みまたは作成、エクスポートオプションの設定、適切な `Save` メソッドの呼び出しという 3 つのコア アクションが必要です。まず `Document` オブジェクトを作成またはロードし、次に `HtmlSaveOptions` でリソース処理や CSS 生成、エンコーディングを調整します。最後にファイル パスまたはストリームを指定して `document.Save` を呼び出し、必要に応じて `PageRange` でエクスポート範囲を限定します。この体系的なアプローチにより、環境間で一貫した結果が得られ、プロジェクトの要件に合わせて出力をカスタマイズできます。

### ステップ 1: デフォルト オプションで OneNote ドキュメントを作成

最初の例では新しい `Document` をインスタンス化し、タイトル付きの単一ページを追加し、デフォルトの `HtmlSaveOptions` でノートブック全体を HTML に保存します。追加パラメータは不要で、**OneNote を HTML として保存** する最も手軽な方法です。

```csharp
using System;
using System.Drawing;
using System.Globalization;
using System.IO;

using Aspose.Note.Saving;
using Aspose.Note.Saving.Html;

```

### ステップ 2: 特定のページ範囲を HTML にエクスポート

会議ノートの章など、ページのサブセットだけが必要な場合があります。`Save` に `PageRange` オブジェクトを渡すことで、ページ 1‑3 のみを対象にエクスポートし、他のページはそのままにできます。

```csharp
public static void CreateAndSaveToHTMLUsingDefaultOptions()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateOneNoteDocAndSaveToHTML_out.html");
    doc.Save(outputPath);

    Console.WriteLine("\nOneNote document created successfully.\nFile saved at " + outputPath);
}
```

### ステップ 3: 埋め込みリソース付きでメモリ ストリームに HTML を保存

API 応答など、単一の自己完結型 HTML ペイロードが必要なときは、CSS、フォント、画像を直接 `MemoryStream` に埋め込みます。`HtmlSaveOptions` の `EmbedResources` を `true` に設定すれば、コンパクトな HTML 文字列が生成されます。

```csharp
public static void CreateAndSavePageRange()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateAndSavePageRange_out.html");
    doc.Save(outputPath, new HtmlSaveOptions { PageCount = 1, PageIndex = 0 });

    Console.WriteLine("\nOneNote document created successfully and saved as page range.\nFile saved at " + outputPath);
}
```

### ステップ 4: リソースを別ファイルに分割して HTML を保存

従来のウェブフォルダー構成（HTML ファイルとアセット フォルダー）を好む場合は、埋め込みを無効にし、出力ディレクトリを指定します。この方法はソース ファイルが見やすく、画像やスタイルシートのキャッシュが容易になります。

```csharp
public static void SaveAsHTMLToMemoryStreamWithEmbeddedResources()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportEmbedded,
        ExportFonts = ResourceExportType.ExportEmbedded,
        ExportImages = ResourceExportType.ExportEmbedded,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to a memory stream
    var memoryStream = new MemoryStream();
    document.Save(memoryStream, options);
}
```

### ステップ 5: コールバックでリソース管理をカスタマイズして HTML を保存

画像をデータベースやクラウド バケットに保存するなど、高度なシナリオではカスタム処理が必要です。`ResourceSavingCallback` デリゲートを提供すれば、各リソースの保存方法を完全に制御でき、API は正しい HTML 参照を生成し続けます。

```csharp
public static void SaveAsHTMLToFileWithResourcesInSeparateFiles()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportAsStream,
        ExportFonts = ResourceExportType.ExportAsStream,
        ExportImages = ResourceExportType.ExportAsStream,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to HTML file with resources stored in separate files
    document.Save(Path.Combine(dataDir, "document_out.html"), options);
}
```

## よくある問題と解決策

| 問題 | 発生理由 | 対策 |
|-------|----------------|-----|
| **画像が表示されない** | リソースが HTML ファイルとは別のフォルダーに保存された。 | `HtmlSaveOptions.ResourcesFolder` を参照先フォルダーに合わせるか、`EmbedResources` を有効にする。 |
| **CSS スタイルが欠落している** | デフォルトの CSS 生成が無効化されている。 | `HtmlSaveOptions.GenerateCss = true`（デフォルト）に設定するか、手動で CSS を埋め込む。 |
| **大規模ノートブックで OutOfMemory が発生** | 非ストリーミングの `Document` コンストラクタで全ファイルを読み込んでいる。 | `Document.Load(stream, LoadOptions)` と `LoadOptions.LoadFromMemory = true` を使用してページをストリーミング処理する。 |
| **非 ASCII 文字の HTML エンコーディング問題** | `HtmlSaveOptions` の `Encoding` が誤っている。 | `HtmlSaveOptions.Encoding = Encoding.UTF8` を設定する。 |
| **Linux で保存が失敗** | 画像処理用のネイティブ依存関係が欠如している。 | Linux ホストに `libgdiplus` をインストールする。 |

**Q: Aspose.Note は HTML 以外の形式への変換もサポートしていますか？**  
A: もちろんです。PDF、DOCX、PNG、JPEG など、30 以上の出力形式に変換可能です。

**Q: Aspose.Note は .NET Core アプリケーションと互換性がありますか？**  
A: はい—.NET Core 3.1、.NET 5、.NET 6 以降をフルサポートしています。

**Q: Aspose.Note を使って OneNote ドキュメントからテキストや画像を抽出できますか？**  
A: `Page.GetText()` でプレーンテキストを取得でき、`Page.GetResources(ResourceType.Image)` で画像を取得できます。

```csharp
public static void SaveAsHTMLToMemoryStreamWithCallBacksToSaveResources()
{
    // Specify the saving callbacks configuration
    var savingCallbacks = new UserSavingCallbacks()
    {
        RootFolder = "documentFolder",
        CssFolder = "css",
        KeepCssStreamOpened = true,
        ImagesFolder = "images",
        FontsFolder = "fonts"
    };

    // Specify HTML saving options
    var options = new HtmlSaveOptions
    {
        FontFaceTypes = FontFaceType.Ttf,
        CssSavingCallback = savingCallbacks,
        FontSavingCallback = savingCallbacks,
        ImageSavingCallback = savingCallbacks
    };

    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Save the document to HTML format with resources managed by user-defined callbacks
    using (var stream = File.Create(Path.Combine(savingCallbacks.RootFolder, "document.html")))
    {
        document.Save(stream, options);
    }

    // Manually append data to the CSS stream
    using (var writer = new StreamWriter(savingCallbacks.CssStream))
    {
        writer.WriteLine();
        writer.WriteLine("/* This line is appended to stream manually by user */");
    }
}
```
