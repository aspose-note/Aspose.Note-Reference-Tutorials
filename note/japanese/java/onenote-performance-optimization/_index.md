## よくある質問

**Q: OneNote ノートブックを中間フォーマットなしで直接 PDF にエクスポートできますか？**  
A: はい。Aspose.Note は、ノートブックを PDF ファイルに書き込むワンステップの `save` メソッドを提供し、レイアウトと画像を保持します。

**Q: OneNote ページから画像だけを抽出するにはどうすればよいですか？**  
A: `Document.getImages()` API を使用して画像オブジェクトを列挙し、各画像を目的のフォーマット（PNG、JPEG など）で保存します。

**Q: 大規模なノートブックのエクスポート速度を向上させる方法はありますか？**  
A: `Document.optimizeResources()` メソッドを保存前に有効にし、セクションを並列スレッドで処理することを検討してください。

**Q: サーバーに追加のソフトウェアをインストールする必要がありますか？**  
A: いいえ。Aspose.Note は純粋な Java ライブラリであり、プロジェクトのクラスパスに JAR を追加するだけです。

**Q: 本番利用向けのライセンスオプションは何がありますか？**  
A: Aspose は永続ライセンスとサブスクリプションモデルを提供しています。評価用の無料トライアルも利用可能です。
