# Aspose Note Java – Manipulation de documents OneNote

## Introduction à Aspose Note Java

Aspose.Note for Java (souvent référencé sous le nom **aspose note java**) offre aux développeurs une API puissante et entièrement gérée pour créer, modifier et automatiser les fichiers OneNote sans jamais avoir besoin de l'application de bureau. Que vous ayez besoin de générer des comptes‑rendus de réunion, de créer des manuels de formation ou de traiter par lots des blocs‑notes existants, cette bibliothèque vous permet de tout faire de manière programmatique depuis Java.

Dans ce hub, vous trouverez trois tutoriels ciblés qui vous guident à travers les scénarios les plus courants :

- **How to use visitor** – parcourir un fichier OneNote et mettre à jour son contenu.  
- **How to manipulate OneNote** – créer des pages en texte enrichi, ajouter des images et définir les propriétés de la page.  
- **How to generate simple rich text** – la façon la plus rapide d’obtenir une page OneNote propre à partir du code.  

Chaque guide est autonome, inclut du code d'exemple prêt à l'exécution, et explique le « pourquoi » de chaque étape afin que vous puissiez adapter les solutions à vos propres projets.
