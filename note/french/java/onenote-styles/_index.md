**Frequently Asked Questions**

**Q:** *Can I apply the same style to multiple paragraphs at once?*  
**A:** Yes, you can loop through the paragraph collection and assign a shared `ParagraphStyle` object.  
**Q:** *Puis‑je appliquer le même style à plusieurs paragraphes à la fois ?*  
**A:** Oui, vous pouvez parcourir la collection de paragraphes et attribuer un objet `ParagraphStyle` partagé.  

**Q:** *Is it possible to change the default font for the entire notebook?*  
**A:** Absolutely. Set the default paragraph style at the notebook level and all new pages inherit it.  
**Q:** *Est‑il possible de changer la police par défaut pour l’ensemble du carnet ?*  
**A:** Absolument. Définissez le style de paragraphe par défaut au niveau du carnet et toutes les nouvelles pages en hériteront.  

**Q:** *Do these APIs work with encrypted OneNote files?*  
**A:** The API can open password‑protected files when you provide the decryption password during loading.  
**Q:** *Ces API fonctionnent‑elles avec des fichiers OneNote cryptés ?*  
**A:** L’API peut ouvrir les fichiers protégés par mot de passe lorsque vous fournissez le mot de passe de déchiffrement lors du chargement.  

**Q:** *What happens if I set a font size that exceeds OneNote’s limits?*  
**A:** Aspose.Note will clamp the size to the nearest supported value, ensuring the document remains valid.  
**Q:** *Que se passe‑t‑il si je définis une taille de police qui dépasse les limites de OneNote ?*  
**A:** Aspose.Note limitera la taille à la valeur prise en charge la plus proche, garantissant que le document reste valide.  

**Q:** *Can I preview the styled note before saving?*  
**A:** Yes, you can render the document to HTML or PDF to see the visual result instantly.  
**Q:** *Puis‑je prévisualiser la note stylisée avant de l’enregistrer ?*  
**A:** Oui, vous pouvez rendre le document en HTML ou PDF pour voir immédiatement le résultat visuel.
