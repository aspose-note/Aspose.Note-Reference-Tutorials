## Réponses rapides
- **Que signifie « add hyperlink to image » ?** Il intègre une URL cliquable dans un objet image à l’intérieur d’une page OneNote.  
- **Quelle bibliothèque prend‑en charge cela ?** Aspose.Note for Java fournit une API fluide pour l’hyperlien d’image.  
- **Ai‑je besoin d’une licence ?** Un essai gratuit suffit pour l’évaluation ; une licence commerciale est requise pour la production.  
- **Puis‑je utiliser des flux au lieu de fichiers ?** Oui — Aspose.Note vous permet de charger et d’enregistrer des images via `InputStream`.

## Questions fréquentes

**Q : Puis‑je ajouter un hyperlien à n’importe quel format d’image (PNG, JPEG, GIF) ?**  
R : Oui. Aspose.Note prend en charge tous les formats d’image standards ; l’hyperlien est attaché à l’objet image quel que soit son type.

**Q : Dois‑je ouvrir le fichier OneNote en mode édition pour ajouter l’hyperlien ?**  
R : Non. Vous pouvez créer ou modifier un document entièrement en mémoire, puis l’enregistrer dans un fichier `.one`.

**Q : L’hyperlien fonctionnera‑t‑il sur les applications OneNote mobiles ?**  
R : Absolument. L’hyperlien généré est stocké dans le format de fichier OneNote, reconnu par les clients de bureau, web et mobiles.

**Q : Comment vérifier que l’hyperlien a été ajouté correctement ?**  
R : Après l’enregistrement, ouvrez le fichier dans OneNote et cliquez sur l’image ; l’URL liée devrait s’ouvrir dans le navigateur par défaut.

**Q : Existe‑t‑il un moyen d’ajouter plusieurs hyperliens à une même image ?**  
R : Une image ne peut contenir qu’un seul hyperlien. Pour fournir plusieurs liens, envisagez d’utiliser une image composite avec des zones cliquables distinctes ou d’ajouter des objets image séparés.
