# Enregistrer OneNote au format HTML avec Aspose.Note

## Introduction

Si vous devez **enregistrer OneNote au format HTML** pour la publication web, les rapports ou l'archivage, Aspose.Note pour .NET vous offre une solution propre, uniquement en code, qui fonctionne sous Windows, Linux et macOS sans nécessiter Microsoft Office. Dans ce tutoriel, vous apprendrez à créer un document OneNote à partir de zéro, à manipuler ses pages et à exporter l’ensemble du bloc‑notes — ou des pages sélectionnées — vers HTML en utilisant diverses options. Le guide vous accompagne à chaque étape, explique pourquoi chaque paramètre est important et montre des scénarios réels où l’exportation HTML brille, comme l’intégration de contenu OneNote dans un portail de base de connaissances ou la génération de documentation statique.

## Réponses rapides
- **Quelle est la méthode principale pour exporter ?** Call `document.Save("output.html", SaveFormat.Html)` on an `Aspose.Note.Document` instance.  
- **Puis-je exporter une seule page ?** Yes—use `document.Save(pageRange, "page.html", SaveFormat.Html)`.  
- **Les ressources (images, CSS) sont‑elles intégrées automatiquement ?** By default they are saved as separate files; you can embed them into a memory stream with `HtmlSaveOptions`.  
- **.NET Core est‑il pris en charge ?** Absolutely—Aspose.Note works with .NET 5, .NET 6, and .NET Core 3.1+.  
- **Ai‑je besoin d’une licence pour le développement ?** A free 30‑day trial works for testing; a license removes evaluation watermarks.

## Qu’est‑ce que « enregistrer OneNote au format HTML » ?

**« Enregistrer OneNote au format HTML »** désigne le processus de conversion d’un bloc‑notes OneNote, stocké dans un format binaire propriétaire, en fichiers HTML standard pouvant être affichés par n’importe quel navigateur web. La conversion préserve la structure hiérarchique du bloc‑notes, le formatage du texte, les images, les tableaux et autres ressources, tout en traduisant les éléments spécifiques à OneNote en balises HTML équivalentes et styles CSS. Cela permet aux développeurs d’afficher le contenu OneNote sur des sites web, de l’intégrer dans des systèmes de gestion de contenu, ou de l’archiver dans un format universellement accessible sans nécessiter le client OneNote.

## Pourquoi convertir OneNote en HTML ?

Aspose.Note peut exporter vers **plus de 50 formats de sortie** (y compris PDF, DOCX, PNG et HTML) et traite les blocs‑notes contenant **jusqu’à 500 pages** sans charger le fichier complet en mémoire, grâce à son architecture de streaming. Convertir en HTML vous permet de :
1. **Publier du contenu sur le web** sans nécessiter le client OneNote.  
2. **Intégrer avec des systèmes de gestion de contenu** qui n’acceptent que le HTML.  
3. **Extraire le texte indexable** pour les moteurs d’indexation.  
4. **Réutiliser les ressources au niveau des pages** (images, polices) dans d’autres projets web.

- Familiarité avec C# et Visual Studio (toute édition).

- Compréhension de base de la structure hiérarchique de OneNote (Notebook → Section → Page).

## Comment enregistrer onenote au format html – Étape par étape

Enregistrer OneNote au format HTML implique trois actions principales : charger ou créer un bloc‑notes, configurer les options d’exportation souhaitées, et appeler la méthode `Save` appropriée. D’abord, vous créez ou chargez un objet `Document`, puis vous pouvez ajuster `HtmlSaveOptions` pour contrôler la gestion des ressources, la génération du CSS et l’encodage. Enfin, vous appelez `document.Save` avec un chemin de fichier ou un flux, en spécifiant éventuellement un `PageRange` pour limiter l’exportation. Cette approche systématique garantit des résultats cohérents sur différents environnements et vous permet d’adapter la sortie aux besoins de votre projet.

### Étape 1 : Créer un document OneNote avec les options par défaut

Dans ce premier exemple, nous instancions un nouveau `Document`, ajoutons une seule page avec un titre, et enregistrons l’ensemble du bloc‑notes au format HTML en utilisant les `HtmlSaveOptions` par défaut. Aucun paramètre supplémentaire n’est requis, ce qui en fait la façon la plus rapide d’**enregistrer OneNote au format HTML**.

```csharp
using System;
using System.Drawing;
using System.Globalization;
using System.IO;

using Aspose.Note.Saving;
using Aspose.Note.Saving.Html;

```

### Étape 2 : Exporter une plage de pages spécifique vers HTML

Souvent, vous n’avez besoin que d’un sous‑ensemble de pages — par exemple, exporter un chapitre de notes de réunion. En passant un objet `PageRange` à `Save`, vous pouvez cibler les pages 1‑3 tout en laissant les autres intactes.

```csharp
public static void CreateAndSaveToHTMLUsingDefaultOptions()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateOneNoteDocAndSaveToHTML_out.html");
    doc.Save(outputPath);

    Console.WriteLine("\nOneNote document created successfully.\nFile saved at " + outputPath);
}
```

### Étape 3 : Enregistrer en HTML dans un MemoryStream avec les ressources intégrées

Lorsque vous souhaitez un seul payload HTML autonome (par ex., pour des réponses d’API), intégrez le CSS, les polices et les images directement dans un `MemoryStream`. `HtmlSaveOptions` vous permet de définir `EmbedResources` à `true`, produisant une chaîne HTML compacte.

```csharp
public static void CreateAndSavePageRange()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateAndSavePageRange_out.html");
    doc.Save(outputPath, new HtmlSaveOptions { PageCount = 1, PageIndex = 0 });

    Console.WriteLine("\nOneNote document created successfully and saved as page range.\nFile saved at " + outputPath);
}
```

### Étape 4 : Enregistrer en HTML dans des fichiers avec les ressources dans des fichiers séparés

Si vous préférez la sortie classique de dossiers web — fichier HTML plus un dossier d’actifs — désactivez l’intégration et spécifiez un répertoire de sortie. Cette approche produit des fichiers sources plus propres et facilite la mise en cache des images ou des feuilles de style.

```csharp
public static void SaveAsHTMLToMemoryStreamWithEmbeddedResources()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportEmbedded,
        ExportFonts = ResourceExportType.ExportEmbedded,
        ExportImages = ResourceExportType.ExportEmbedded,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to a memory stream
    var memoryStream = new MemoryStream();
    document.Save(memoryStream, options);
}
```

### Étape 5 : Enregistrer en HTML en utilisant des callbacks pour gérer les ressources

Les scénarios avancés, comme le stockage d’images dans une base de données ou un bucket cloud, nécessitent une gestion personnalisée. En fournissant un délégué `ResourceSavingCallback`, vous contrôlez exactement comment chaque ressource est persistée, tandis que l’API continue de générer les références HTML correctes.

```csharp
public static void SaveAsHTMLToFileWithResourcesInSeparateFiles()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportAsStream,
        ExportFonts = ResourceExportType.ExportAsStream,
        ExportImages = ResourceExportType.ExportAsStream,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to HTML file with resources stored in separate files
    document.Save(Path.Combine(dataDir, "document_out.html"), options);
}
```

## Problèmes courants et solutions

| Problème | Pourquoi cela se produit | Solution |
|----------|--------------------------|----------|
| **Images apparaissent cassées** | Les ressources ont été enregistrées dans un dossier différent de celui du fichier HTML. | Assurez‑vous que `HtmlSaveOptions.ResourcesFolder` correspond au dossier que vous référencez, ou activez `EmbedResources`. |
| **Les styles CSS sont manquants** | La génération CSS par défaut a été désactivée. | Définissez `HtmlSaveOptions.GenerateCss = true` (par défaut) ou intégrez le CSS manuellement. |
| **Les gros blocs‑notes provoquent OutOfMemory** | L’utilisation du constructeur `Document` non‑streaming charge tout le fichier. | Utilisez `Document.Load(stream, LoadOptions)` avec `LoadOptions.LoadFromMemory = true` pour diffuser les pages. |
| **Problèmes d’encodage HTML pour les caractères non‑ASCII** | Un mauvais `Encoding` est défini sur `HtmlSaveOptions`. | Définissez `HtmlSaveOptions.Encoding = Encoding.UTF8`. |
| **Échec de l’enregistrement sous Linux** | Dépendances natives manquantes pour le traitement d’images. | Installez `libgdiplus` sur l’hôte Linux. |

```csharp
public static void SaveAsHTMLToMemoryStreamWithCallBacksToSaveResources()
{
    // Specify the saving callbacks configuration
    var savingCallbacks = new UserSavingCallbacks()
    {
        RootFolder = "documentFolder",
        CssFolder = "css",
        KeepCssStreamOpened = true,
        ImagesFolder = "images",
        FontsFolder = "fonts"
    };

    // Specify HTML saving options
    var options = new HtmlSaveOptions
    {
        FontFaceTypes = FontFaceType.Ttf,
        CssSavingCallback = savingCallbacks,
        FontSavingCallback = savingCallbacks,
        ImageSavingCallback = savingCallbacks
    };

    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Save the document to HTML format with resources managed by user-defined callbacks
    using (var stream = File.Create(Path.Combine(savingCallbacks.RootFolder, "document.html")))
    {
        document.Save(stream, options);
    }

    // Manually append data to the CSS stream
    using (var writer = new StreamWriter(savingCallbacks.CssStream))
    {
        writer.WriteLine();
        writer.WriteLine("/* This line is appended to stream manually by user */");
    }
}
```
