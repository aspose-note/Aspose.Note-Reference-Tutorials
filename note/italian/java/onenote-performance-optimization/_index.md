## Domande frequenti

**Q: Posso esportare un notebook OneNote direttamente in PDF senza formati intermedi?**  
A: Sì. Aspose.Note fornisce un metodo `save` a un solo passo che scrive il notebook in un file PDF, preservando layout e immagini.

**Q: Come posso estrarre solo le immagini da una pagina OneNote?**  
A: Usa l'API `Document.getImages()` per enumerare gli oggetti immagine e salvare ciascuno nel formato desiderato (PNG, JPEG, ecc.).

**Q: Esiste un modo per migliorare la velocità di esportazione per notebook di grandi dimensioni?**  
A: Abilita il metodo `Document.optimizeResources()` prima di salvare e considera l'elaborazione delle sezioni in thread paralleli.

**Q: È necessario installare software aggiuntivo sul server?**  
A: No. Aspose.Note è una libreria Java pura; basta includere i JAR nel classpath del tuo progetto.

**Q: Quali opzioni di licenza sono disponibili per l'uso in produzione?**  
A: Aspose offre licenze perpetue e modelli di abbonamento. È disponibile una prova gratuita per la valutazione.
