## Come creare programmaticamente una tabella OneNote?

`Document` rappresenta un file di blocco appunti OneNote caricato in memoria.  
`Page.getTables().add()` crea una nuova Table nella pagina con il numero specificato di righe e colonne.

Carica un'istanza di `Document`, aggiungi una `Page`, quindi chiama `Page.getTables().add()` con il numero desiderato di righe e colonne. Dopo la creazione puoi impostare i valori delle celle, applicare bordi e stilizzare la tabella—tutto tramite chiamate Java fluide. Questo modello a due passaggi (crea → configura) ti consente di costruire tabelle istantaneamente, anche per blocchi appunti a più pagine.

## Perché usare la manipolazione delle tabelle di Aspose.Note per Java?

L'API funziona **al 100 %** su Java puro, quindi non è necessaria alcuna installazione nativa di OneNote, garantendo un'automazione affidabile su qualsiasi server.
