# Salva OneNote come HTML usando Aspose.Note

## Introduzione

Se hai bisogno di **salvare OneNote come HTML** per la pubblicazione web, report o archiviazione, Aspose.Note per .NET ti offre una soluzione pulita, solo codice, che funziona su Windows, Linux e macOS senza richiedere Microsoft Office. In questo tutorial imparerai a creare un documento OneNote da zero, manipolare le sue pagine ed esportare l’intero blocco note — o pagine selezionate — in HTML usando una varietà di opzioni. La guida ti accompagna passo passo, spiega perché ogni impostazione è importante e mostra scenari reali in cui l’esportazione HTML brilla, come l’inserimento di contenuti OneNote in un portale di knowledge‑base o la generazione di documentazione statica.

## Risposte Rapide
- **Qual è il metodo principale per esportare?** Chiama `document.Save("output.html", SaveFormat.Html)` su un'istanza di `Aspose.Note.Document`.  
- **Posso esportare una singola pagina?** Sì — usa `document.Save(pageRange, "page.html", SaveFormat.Html)`.  
- **Le risorse (immagini, CSS) vengono incorporate automaticamente?** Per impostazione predefinita vengono salvate come file separati; puoi incorporarle in un flusso di memoria con `HtmlSaveOptions`.  
- **È supportato .NET Core?** Assolutamente — Aspose.Note funziona con .NET 5, .NET 6 e .NET Core 3.1+.  
- **È necessaria una licenza per lo sviluppo?** Una prova gratuita di 30 giorni è sufficiente per i test; una licenza rimuove le filigrane di valutazione.

## Cos'è “salvare OneNote come HTML”?

**“Salvare OneNote come HTML”** indica il processo di conversione di un blocco note OneNote, memorizzato in un formato binario proprietario, in file HTML standard che possono essere visualizzati da qualsiasi browser web. La conversione preserva la struttura gerarchica del blocco note, la formattazione del testo, le immagini, le tabelle e altre risorse, traducendo gli elementi specifici di OneNote in tag HTML equivalenti e stili CSS. Questo consente agli sviluppatori di mostrare contenuti OneNote su siti web, integrarli in sistemi di gestione dei contenuti o archiviarli in un formato universalmente accessibile senza richiedere il client OneNote.

## Perché convertire OneNote in HTML?

Aspose.Note può esportare in **oltre 50 formati di output** (inclusi PDF, DOCX, PNG e HTML) e gestisce blocchi note con **fino a 500 pagine** senza caricare l’intero file in memoria, grazie alla sua architettura di streaming. Convertire in HTML ti permette di:

1. **Pubblicare contenuti sul web** senza richiedere il client OneNote.  
2. **Integrare con sistemi di gestione dei contenuti** che accettano solo HTML.  
3. **Estrarre testo ricercabile** per i motori di indicizzazione.  
4. **Riutilizzare risorse a livello di pagina** (immagini, font) in altri progetti web.

## Come salvare onenote come html – Passo per Passo

Salvare OneNote come HTML comporta tre azioni fondamentali: caricare o creare un blocco note, configurare le opzioni di esportazione desiderate e invocare il metodo `Save` appropriato. Prima crei o carichi un oggetto `Document`, poi puoi regolare `HtmlSaveOptions` per controllare la gestione delle risorse, la generazione di CSS e la codifica. Infine, chiami `document.Save` con un percorso file o uno stream, opzionalmente specificando un `PageRange` per limitare l’esportazione. Questo approccio sistematico garantisce risultati coerenti su diversi ambienti e ti consente di personalizzare l’output in base alle esigenze del progetto.

### Passo 1: Crea un documento OneNote con le opzioni predefinite

In questo primo esempio istanziamo un nuovo `Document`, aggiungiamo una singola pagina con un titolo e salviamo l’intero blocco note in HTML usando le `HtmlSaveOptions` predefinite. Nessun parametro aggiuntivo è richiesto, rendendolo il modo più rapido per **salvare OneNote come HTML**.

```csharp
using System;
using System.Drawing;
using System.Globalization;
using System.IO;

using Aspose.Note.Saving;
using Aspose.Note.Saving.Html;

```

### Passo 2: Esporta un intervallo di pagine specifico in HTML

Spesso ti serve solo un sottoinsieme di pagine — ad esempio, esportare un capitolo di appunti di riunione. Passando un oggetto `PageRange` a `Save`, puoi mirare alle pagine 1‑3 lasciando intatto il resto.

```csharp
public static void CreateAndSaveToHTMLUsingDefaultOptions()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateOneNoteDocAndSaveToHTML_out.html");
    doc.Save(outputPath);

    Console.WriteLine("\nOneNote document created successfully.\nFile saved at " + outputPath);
}
```

### Passo 3: Salva come HTML in un Memory Stream con risorse incorporate

Quando desideri un unico payload HTML autonomo (ad es., per risposte API), incorpora CSS, font e immagini direttamente in un `MemoryStream`. `HtmlSaveOptions` ti permette di impostare `EmbedResources` a `true`, producendo una stringa HTML compatta.

```csharp
public static void CreateAndSavePageRange()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateAndSavePageRange_out.html");
    doc.Save(outputPath, new HtmlSaveOptions { PageCount = 1, PageIndex = 0 });

    Console.WriteLine("\nOneNote document created successfully and saved as page range.\nFile saved at " + outputPath);
}
```

### Passo 4: Salva come HTML in file con risorse in file separati

Se preferisci l’output classico di una cartella web — file HTML più una cartella di asset — disabilita l’incorporamento e specifica una directory di output. Questo approccio genera file sorgente più puliti e una più facile cache di immagini o fogli di stile.

```csharp
public static void SaveAsHTMLToMemoryStreamWithEmbeddedResources()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportEmbedded,
        ExportFonts = ResourceExportType.ExportEmbedded,
        ExportImages = ResourceExportType.ExportEmbedded,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to a memory stream
    var memoryStream = new MemoryStream();
    document.Save(memoryStream, options);
}
```

### Passo 5: Salva come HTML usando callback per gestire le risorse

Scenari avanzati, come la memorizzazione di immagini in un database o in un bucket cloud, richiedono una gestione personalizzata. Fornendo un delegato `ResourceSavingCallback`, controlli esattamente come ogni risorsa viene persa, mentre l’API continua a generare riferimenti HTML corretti.

```csharp
public static void SaveAsHTMLToFileWithResourcesInSeparateFiles()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportAsStream,
        ExportFonts = ResourceExportType.ExportAsStream,
        ExportImages = ResourceExportType.ExportAsStream,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to HTML file with resources stored in separate files
    document.Save(Path.Combine(dataDir, "document_out.html"), options);
}
```

## Problemi comuni e soluzioni

| Problema | Perché accade | Soluzione |
|----------|----------------|-----------|
| **Le immagini appaiono rotte** | Le risorse sono state salvate in una cartella diversa dal file HTML. | Assicurati che `HtmlSaveOptions.ResourcesFolder` corrisponda alla cartella di riferimento, o abilita `EmbedResources`. |
| **Gli stili CSS sono mancanti** | La generazione CSS predefinita è stata disabilitata. | Imposta `HtmlSaveOptions.GenerateCss = true` (predefinito) o incorpora manualmente il CSS. |
| **Quaderni di grandi dimensioni causano OutOfMemory** | L'uso del costruttore `Document` non in streaming carica l'intero file. | Usa `Document.Load(stream, LoadOptions)` con `LoadOptions.LoadFromMemory = true` per streammare le pagine. |
| **Problemi di codifica HTML per caratteri non ASCII** | `Encoding` errato impostato su `HtmlSaveOptions`. | Imposta `HtmlSaveOptions.Encoding = Encoding.UTF8`. |
| **Salvataggio fallito su Linux** | Mancano dipendenze native per l'elaborazione delle immagini. | Installa `libgdiplus` sull'host Linux. |

```csharp
public static void SaveAsHTMLToMemoryStreamWithCallBacksToSaveResources()
{
    // Specify the saving callbacks configuration
    var savingCallbacks = new UserSavingCallbacks()
    {
        RootFolder = "documentFolder",
        CssFolder = "css",
        KeepCssStreamOpened = true,
        ImagesFolder = "images",
        FontsFolder = "fonts"
    };

    // Specify HTML saving options
    var options = new HtmlSaveOptions
    {
        FontFaceTypes = FontFaceType.Ttf,
        CssSavingCallback = savingCallbacks,
        FontSavingCallback = savingCallbacks,
        ImageSavingCallback = savingCallbacks
    };

    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Save the document to HTML format with resources managed by user-defined callbacks
    using (var stream = File.Create(Path.Combine(savingCallbacks.RootFolder, "document.html")))
    {
        document.Save(stream, options);
    }

    // Manually append data to the CSS stream
    using (var writer = new StreamWriter(savingCallbacks.CssStream))
    {
        writer.WriteLine();
        writer.WriteLine("/* This line is appended to stream manually by user */");
    }
}
```
