## Что такое программное создание таблицы OneNote?

Такой подход автоматизирует создание документов, обеспечивает согласованность и масштабируемость при больших нагрузках. Он позволяет разработчикам строить таблицы напрямую из Java‑приложений, экономя время и снижая количество ошибок.

## Как программно создать таблицу OneNote?

`Document` представляет файл блокнота OneNote, загруженный в память.  
`Page.getTables().add()` создаёт новую Table на странице с указанным количеством строк и столбцов.

Загрузите экземпляр `Document`, добавьте `Page`, затем вызовите `Page.getTables().add()` с нужным числом строк и столбцов. После создания вы можете задавать значения ячеек, применять границы и стилизовать таблицу — всё через цепочку вызовов Java. Этот двухшаговый шаблон (создать → настроить) позволяет мгновенно строить таблицы, даже для многостраничных блокнотов.
