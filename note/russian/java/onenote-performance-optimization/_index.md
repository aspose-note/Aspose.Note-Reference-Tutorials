## Часто задаваемые вопросы

**Q: Могу ли я экспортировать блокнот OneNote напрямую в PDF без промежуточных форматов?**  
A: Да. Aspose.Note предоставляет одношаговый метод `save`, который записывает блокнот в файл PDF, сохраняя макет и изображения.

**Q: Как извлечь только изображения со страницы OneNote?**  
A: Используйте API `Document.getImages()`, чтобы перечислить объекты изображений и сохранить каждый в нужный формат (PNG, JPEG и т.д.).

**Q: Есть ли способ ускорить экспорт больших блокнотов?**  
A: Включите метод `Document.optimizeResources()` перед сохранением и рассмотрите обработку разделов в параллельных потоках.

**Q: Нужно ли устанавливать дополнительное программное обеспечение на сервер?**  
A: Нет. Aspose.Note — это чистая Java‑библиотека; достаточно добавить JAR‑файлы в classpath вашего проекта.

**Q: Какие варианты лицензирования доступны для продакшн‑использования?**  
A: Aspose предлагает бессрочные лицензии и модели подписки. Доступна бесплатная пробная версия для оценки.
