- **Какая это библиотека поддержки?** Aspose.Note для Java предоставляет удобный API для гиперссылок изображений.
- **Нужна ли лицензия?** Бесплатная пробная версия подходит для оценки; для производства требуется коммерческая лицензия.
- **Можно ли использовать потоки вместо файлов?** Да — Aspose.Note Позволяет загружать и сохранять изображение через `InputStream`.
- **Совместимо ли это с OneNote 2016 и OneNote для Windows10?** Сгенерированный файл `.one` работает на всех современных клиентах OneNote.

## Часто задаваемые вопросы

**В: Можно ли добавить гиперссылку к изображению любого формата (PNG, JPEG, GIF)?**
А: Да. Aspose.Note поддерживает все стандартные форматы изображений; гиперссылка привязывается к объекту изображения независимо от его типа.

**В: Нужно ли открывать файл OneNote в режиме редактирования, чтобы добавить гиперссылку?**
А: Нет. Вы можете полностью создать или изменить документ в памяти, а затем сохранить его в файле `.one`.

**В: Будет ли гиперссылка работать в мобильных приложениях OneNote?**
А: Абсолютно. Сгенерированная гиперссылка приводится в формат файла OneNote, которая распознает клиентов на настольных, веб- и мобильных платформах.

**В: Как проверить, что гиперссылка добавлена ​​корректно?**
О: После сохранения файла в OneNote и щёлкните по изображению; связанный URL-адрес должен открыться в браузере по умолчанию.

**В: Можно ли добавить несколько гиперссылок к одному изображению?**
О: Одно изображение может сохранить только одну гиперссылку. Чтобы указать несколько ссылок, рассмотрите возможность использования составного изображения с выделением кликабельных областей или адаптеров объектов изображений.
