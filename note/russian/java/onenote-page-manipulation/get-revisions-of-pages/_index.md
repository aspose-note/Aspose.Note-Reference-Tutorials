# Получение ревизий страниц в OneNote – Aspose.Note

## Введение

В этом руководстве вы **получите время последнего изменения** страниц внутри документа OneNote и узнаете, как извлечь полную историю ревизий с помощью Aspose.Note для Java. Независимо от того, создаёте ли вы систему управления документами, проводите аудит изменений или просто хотите отобразить, когда страница была отредактирована в последний раз, это руководство покажет, как программно извлечь эту информацию.

## Быстрые ответы
- **Что возвращает “get last modified time”?** Метка времени последнего редактирования страницы OneNote.  
- **Какой класс предоставляет историю ревизий?** `PageHistory` через `Document.getPageHistory(Page)`.  
- **Нужна ли лицензия для этой функции?** Да, для использования в продакшене требуется действующая лицензия Aspose.Note.  
- **Какая версия Java поддерживается?** Java 8 или выше (JDK 8+).  
- **Можно ли фильтровать ревизии по автору?** Вы можете читать свойство `Author` каждого объекта `Page` и применять собственный фильтр.

## Что такое “get last modified time” в OneNote?

**Время последнего изменения** — это поле метаданных, хранящееся у каждой страницы и фиксирующее момент её последнего изменения. Aspose.Note предоставляет это значение через метод `Page.getLastModifiedTime()`, что упрощает отображение или запись дат изменений.

## Зачем получать ревизии страниц?

- **Аудит:** Храните запись о том, кто и когда что изменил.  
- **Сравнение версий:** Реализуйте функции сравнения двух ревизий рядом.  
- **Анализ совместной работы:** Понимайте паттерны редактирования в общих блокнотах.

## Как получить время последнего изменения страницы OneNote

### Шаг 1: Установите каталог документа
Определите каталог, в котором находится ваш документ OneNote.

```java
String dataDir = "Your Document Directory";
```

### Шаг 2: Загрузите документ
Загрузите документ OneNote в Aspose.Note.

```java
Document doc = new Document(dataDir + "Sample1.one");
```

### Шаг 3: Получите первую страницу
Извлеките первую страницу из документа.

```java
Page firstPage = doc.getFirstChild();
```

### Шаг 4: Получите ревизии страницы
Получите историю ревизий страницы.

```java
PageHistory revisions = doc.getPageHistory(firstPage);
```

### Шаг 5: Обойдите ревизии страницы
Итерируйте список ревизий страницы и извлекайте нужную информацию, включая **время последнего изменения**.

```java
for (Page pageRevision : revisions) {
    System.out.println("LastModifiedTime: " + pageRevision.getLastModifiedTime());
    System.out.println("CreationTime: " + pageRevision.getCreationTime());
    System.out.println("Title: " + pageRevision.getTitle());
    System.out.println("Level: " + pageRevision.getLevel());
    System.out.println("Author: " + pageRevision.getAuthor());
    System.out.println();
}
```

## Распространённые проблемы и решения
- **`PageHistory` равно null:** Убедитесь, что документ действительно содержит ревизии; иначе `getPageHistory` может вернуть `null`.  
- **Разница часовых поясов:** `getLastModifiedTime()` возвращает `java.util.Date` в часовом поясе системы по умолчанию. При необходимости преобразуйте в UTC.  
- **Лицензия не загружена:** Без действующей лицензии Aspose.Note работает в режиме оценки, ограничивая количество обрабатываемых страниц.
