## Часто задаваемые вопросы

**В: Могу ли я использовать Aspose.Note for Java в коммерческом проекте?**  
A: Да, для использования в продакшн‑среде требуется коммерческая лицензия, но доступна бесплатная пробная версия для оценки.

**В: Какие версии Java поддерживаются?**  
A: Библиотека поддерживает Java 8 и более поздние версии.

**В: Как обрабатывать большие файлы OneNote при получении вложений?**  
A: Загружайте документ в режиме потоковой передачи или обрабатывайте разделы/страницы по отдельности, чтобы снизить потребление памяти.

**В: Можно ли установить пользовательский значок для прикреплённого файла?**  
A: Конечно — руководство «Прикрепление файла и установка значка» показывает, как программно указать значок.

**В: Нужно ли устанавливать OneNote на сервере для использования API?**  
A: Нет, Aspose.Note работает независимо от приложения OneNote; он читает и записывает форматы файлов OneNote напрямую.
