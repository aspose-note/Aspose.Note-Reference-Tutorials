## Что такое «конвертировать OneNote в обычный текст»?

## Как конвертировать OneNote в обычный текст с помощью Aspose.Note
Ниже представлена полная пошаговая инструкция. Каждый шаг объясняется простым языком перед фрагментом кода, чтобы вы всегда знали *почему* пишем конкретную строку.

### Java extract text from one file
Сначала включите необходимые импорты, чтобы компилятор знал, какие классы мы используем.

```java
import java.util.List;
import java.util.stream.Collectors;
import com.aspose.note.Document;
import com.aspose.note.RichText;
```

### Шаг 1: Установить путь к каталогу документов
Определите путь к папке, содержащей ваши файлы `.one`. Хранение пути в переменной упрощает повторное использование.

```java
String dataDir = "Your Document Directory";
```

### Шаг 2: Загрузить документ OneNote
Создайте объект `Document`, указав путь к файлу OneNote, который нужно обработать.

```java
Document oneFile = new Document(dataDir + "Sample1.one");
```

### Шаг 3: Получить текстовые узлы
OneNote хранит содержимое в виде иерархии узлов. Мы запрашиваем у документа все узлы типа `RichText`, которые представляют фрагменты обычного текста.

```java
List<RichText> textNodes = (List<RichText>) oneFile.getChildNodes(RichText.class);
```

### Шаг 4: Извлечь текст
Итерируйте каждый узел `RichText`, извлекайте его строковое значение и добавляйте в `StringBuilder`. Это формирует один непрерывный блок обычного текста.

```java
StringBuilder text = new StringBuilder();
for (RichText richText : textNodes) {
    text = text.append(richText.getText().toString());
}
```

### Шаг 5: Вывести текст
Наконец, выведите объединённую строку в консоль. В реальном сценарии вы можете записать её в файл `.txt`.

```java
System.out.println(text)
```

Повторите эти шаги для любого документа OneNote, и Aspose.Note для Java эффективно **конвертирует OneNote в обычный текст**.
