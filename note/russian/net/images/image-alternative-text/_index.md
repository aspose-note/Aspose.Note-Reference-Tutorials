# Как добавить альтернативный текст к изображениям в Aspose.Note

## Введение

Если вам нужно **how to add alt text** для изображений в ваших OneNote‑подобных документах, вы попали по адресу. Этот учебник пошагово покажет, как добавить альтернативный текст (и заголовок, и описание) к изображению с помощью Aspose.Note для .NET. Добавление alt‑текста не только повышает доступность для пользователей скрин‑ридеров, но и улучшает SEO для любого веб‑контента, в котором позже будут использоваться эти изображения.

## Быстрые ответы
- **Что означает “alt text”?** Текстовое описание, которое представляет изображение, когда оно не может быть отображено.  
- **Зачем использовать Aspose.Note для alt text?** Он предоставляет простой API для программного задания как заголовка, так и описания.  
- **Какие требования?** Среда разработки .NET, Visual Studio и установленный Aspose.Note для .NET.  
- **Можно ли добавить alt text к существующим изображениям?** Да — можно загрузить объект изображения, установить его свойства и сохранить документ.  
- **Где сохраняется вывод?** По пути, который вы укажете в `document.Save(...)`.

## Что означает “how to add alt text” в Aspose.Note?

Добавление alt‑текста означает присвоение свойств `AlternativeTextTitle` и `AlternativeTextDescription` объекту `Image`. Эти свойства читаются скрин‑ридерами и другими вспомогательными технологиями для передачи смысла картинки.

## Почему добавлять альтернативный текст к изображению в документе?

- **Соответствие требованиям доступности** – соответствует рекомендациям WCAG и Section 508.  
- **Улучшенный SEO** – поисковые системы индексируют описательный текст.  
- **Лучший пользовательский опыт** – пользователи с отключёнными изображениями всё равно понимают содержание.

## Требования

Прежде чем начать, убедитесь, что у вас есть:

- Базовые знания C# и разработки на .NET.  
- Установленный Visual Studio.  
- Aspose.Note для .NET, скачанный и подключённый к вашему проекту – его можно получить [здесь](https://releases.aspose.com/note/net/).  
- Файл изображения (например, `image.jpg`), который вы хотите встроить.

## Импорт пространств имён

Сначала подключите пространства имён, необходимые для работы с файлами и объектами Aspose.Note.

```csharp
using System.IO;
using Aspose.Note;
using System.Collections.Generic;
using System.Drawing;
using System;
```

## Шаг 1: Инициализация документа и страницы

Создайте новый экземпляр `Document` и добавьте `Page`, на которой будет находиться изображение.

```csharp
var document = new Document();
var page = new Page(document);
```

## Шаг 2: Загрузка изображения

Укажите папку, содержащую вашу картинку, и создайте объект `Image`.

```csharp
string dataDir = "Your Document Directory";
var image = new Image(document, dataDir + "image.jpg");
```

## Шаг 3: Установка альтернативного текста

Здесь мы **add alternative text image** заполняя поля заголовка и описания.

```csharp
image.AlternativeTextTitle = "This is an image's title!";
image.AlternativeTextDescription = "And this is an image's description!";
```

## Шаг 4: Добавление изображения на страницу

Теперь мы **append image to page** с помощью метода `AppendChildLast`.

```csharp
page.AppendChildLast(image);
```

## Шаг 5: Сохранение документа

Укажите имя выходного файла и сохраните документ OneNote.

```csharp
dataDir = dataDir + "ImageAlternativeText_out.one";
document.Save(dataDir);
```

## Шаг 6: Отображение сообщения об успехе

Простое сообщение в консоли подтверждает, что операция завершилась успешно.

```csharp
Console.WriteLine("\nImage alternative text setup successfully.\nFile saved at " + dataDir); 
```

## Распространённые проблемы и решения

| Проблема | Причина | Решение |
|----------|---------|---------|
| **Alt text не отображается** | `AlternativeTextTitle` или `AlternativeTextDescription` оставлены пустыми | Убедитесь, что оба свойства заданы перед сохранением. |
| **Файл не найден** | Неправильный путь `dataDir` | Используйте абсолютный путь или проверьте, что относительная папка существует. |
| **Исключение при сохранении** | Недостаточно прав на запись | Запустите Visual Studio от имени администратора или выберите папку с правом записи. |
