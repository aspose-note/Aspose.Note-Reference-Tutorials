# Сохранить OneNote в формате HTML с помощью Aspose.Note

## Введение

Если вам необходимо **сохранить OneNote в формате HTML** для публикации в вебе, создания отчетов или архивирования, Aspose.Note для .NET предоставляет чистое решение, основанное только на коде, которое работает в Windows, Linux и macOS без необходимости установки Microsoft Office. В этом руководстве вы узнаете, как создать документ OneNote с нуля, управлять его страницами и экспортировать всю записную книжку — или выбранные страницы — в HTML, используя различные параметры. Руководство пошагово объясняет каждый шаг, почему важна каждая настройка, и демонстрирует реальные сценарии, где экспорт в HTML особенно полезен, например, встраивание содержимого OneNote в портал базы знаний или генерация статической документации.

## Быстрые ответы
- **Какой основной метод экспорта?** Вызовите `document.Save("output.html", SaveFormat.Html)` у экземпляра `Aspose.Note.Document`.  
- **Можно ли экспортировать отдельную страницу?** Да — используйте `document.Save(pageRange, "page.html", SaveFormat.Html)`.  
- **Встраиваются ли ресурсы (изображения, CSS) автоматически?** По умолчанию они сохраняются как отдельные файлы; их можно встроить в поток памяти с помощью `HtmlSaveOptions`.  
- **Поддерживается ли .NET Core?** Абсолютно — Aspose.Note работает с .NET 5, .NET 6 и .NET Core 3.1+.  
- **Нужна ли лицензия для разработки?** Бесплатная 30‑дневная trial‑версия подходит для тестирования; лицензия убирает водяные знаки оценки.

## Что такое «save onenote as html»?

**«Save OneNote as HTML»** — это процесс преобразования записной книжки OneNote, хранящейся в проприетарном бинарном формате, в стандартные HTML‑файлы, которые могут отображаться любым веб‑браузером. Конверсия сохраняет иерархическую структуру записной книжки, форматирование текста, изображения, таблицы и другие ресурсы, переводя специфические элементы OneNote в эквивалентные HTML‑теги и CSS‑стили. Это позволяет разработчикам отображать содержимое OneNote на веб‑сайтах, интегрировать его в системы управления контентом или архивировать в универсальном формате без необходимости установки клиента OneNote.

## Почему стоит конвертировать OneNote в HTML?

Aspose.Note может экспортировать в **более чем 50 форматов** (включая PDF, DOCX, PNG и HTML) и обрабатывает записные книжки с **до 500 страниц** без загрузки всего файла в память благодаря потоковой архитектуре. Конверсия в HTML позволяет:

1. **Публиковать контент в вебе** без необходимости клиента OneNote.  
2. **Интегрировать с системами управления контентом**, принимающими только HTML.  
3. **Извлекать индексируемый текст** для поисковых движков.  
4. **Повторно использовать ресурсы уровня страниц** (изображения, шрифты) в других веб‑проектах.

## Как сохранить onenote как html – пошагово

Сохранение OneNote в HTML включает три основных действия: загрузка или создание записной книжки, настройка желаемых параметров экспорта и вызов соответствующего метода `Save`. Сначала вы создаёте или загружаете объект `Document`, затем при необходимости настраиваете `HtmlSaveOptions` для управления ресурсами, генерацией CSS и кодировкой. Наконец, вызываете `document.Save`, указывая путь к файлу или поток, при желании ограничивая экспорт диапазоном страниц через `PageRange`. Такой системный подход обеспечивает стабильные результаты в разных средах и позволяет адаптировать вывод под нужды вашего проекта.

### Шаг 1: Создать документ OneNote с параметрами по умолчанию

В этом первом примере мы создаём новый `Document`, добавляем одну страницу с заголовком и сохраняем всю записную книжку в HTML, используя параметры `HtmlSaveOptions` по умолчанию. Дополнительные параметры не требуются, что делает этот способ самым быстрым для **сохранения OneNote в HTML**.

```csharp
using System;
using System.Drawing;
using System.Globalization;
using System.IO;

using Aspose.Note.Saving;
using Aspose.Note.Saving.Html;

```

### Шаг 2: Экспортировать конкретный диапазон страниц в HTML

Часто требуется экспортировать лишь часть страниц — например, главу с заметками встречи. Передав объект `PageRange` в `Save`, вы можете выбрать страницы 1‑3, оставив остальные нетронутыми.

```csharp
public static void CreateAndSaveToHTMLUsingDefaultOptions()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateOneNoteDocAndSaveToHTML_out.html");
    doc.Save(outputPath);

    Console.WriteLine("\nOneNote document created successfully.\nFile saved at " + outputPath);
}
```

### Шаг 3: Сохранить HTML в поток памяти с встроенными ресурсами

Когда нужен единый самодостаточный HTML‑payload (например, для ответов API), встраиваем CSS, шрифты и изображения непосредственно в `MemoryStream`. `HtmlSaveOptions` позволяет установить `EmbedResources` в `true`, получая компактную HTML‑строку.

```csharp
public static void CreateAndSavePageRange()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateAndSavePageRange_out.html");
    doc.Save(outputPath, new HtmlSaveOptions { PageCount = 1, PageIndex = 0 });

    Console.WriteLine("\nOneNote document created successfully and saved as page range.\nFile saved at " + outputPath);
}
```

### Шаг 4: Сохранить HTML в файлы с ресурсами в отдельных папках

Если предпочтительнее классический вывод в веб‑папку — HTML‑файл плюс папка с ассетами — отключите встраивание и укажите каталог вывода. Такой подход даёт более чистые исходные файлы и упрощает кэширование изображений или таблиц стилей.

```csharp
public static void SaveAsHTMLToMemoryStreamWithEmbeddedResources()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportEmbedded,
        ExportFonts = ResourceExportType.ExportEmbedded,
        ExportImages = ResourceExportType.ExportEmbedded,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to a memory stream
    var memoryStream = new MemoryStream();
    document.Save(memoryStream, options);
}
```

### Шаг 5: Сохранить HTML с использованием обратных вызовов для управления ресурсами

Для продвинутых сценариев, например, сохранения изображений в базе данных или облачном бакете, требуется кастомная обработка. Предоставив делегат `ResourceSavingCallback`, вы полностью контролируете, как каждый ресурс сохраняется, при этом API автоматически генерирует корректные ссылки в HTML.

```csharp
public static void SaveAsHTMLToFileWithResourcesInSeparateFiles()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportAsStream,
        ExportFonts = ResourceExportType.ExportAsStream,
        ExportImages = ResourceExportType.ExportAsStream,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to HTML file with resources stored in separate files
    document.Save(Path.Combine(dataDir, "document_out.html"), options);
}
```

## Распространённые проблемы и решения

| Проблема | Почему происходит | Решение |
|----------|-------------------|---------|
| **Изображения отображаются с ошибкой** | Ресурсы сохранены в другой папке, чем HTML‑файл. | Убедитесь, что `HtmlSaveOptions.ResourcesFolder` указывает на нужную папку, либо включите `EmbedResources`. |
| **Отсутствуют стили CSS** | Отключена генерация CSS по умолчанию. | Установите `HtmlSaveOptions.GenerateCss = true` (по умолчанию) или вручную внедрите CSS. |
| **Большие записные книжки вызывают OutOfMemory** | Конструктор `Document` без потоковой загрузки загружает весь файл. | Используйте `Document.Load(stream, LoadOptions)` с `LoadOptions.LoadFromMemory = true` для потоковой обработки страниц. |
| **Проблемы кодировки HTML для не‑ASCII символов** | Неправильная `Encoding` в `HtmlSaveOptions`. | Установите `HtmlSaveOptions.Encoding = Encoding.UTF8`. |
| **Сохранение не работает в Linux** | Отсутствуют нативные зависимости для обработки изображений. | Установите `libgdiplus` на Linux‑хосте. |

```csharp
public static void SaveAsHTMLToMemoryStreamWithCallBacksToSaveResources()
{
    // Specify the saving callbacks configuration
    var savingCallbacks = new UserSavingCallbacks()
    {
        RootFolder = "documentFolder",
        CssFolder = "css",
        KeepCssStreamOpened = true,
        ImagesFolder = "images",
        FontsFolder = "fonts"
    };

    // Specify HTML saving options
    var options = new HtmlSaveOptions
    {
        FontFaceTypes = FontFaceType.Ttf,
        CssSavingCallback = savingCallbacks,
        FontSavingCallback = savingCallbacks,
        ImageSavingCallback = savingCallbacks
    };

    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Save the document to HTML format with resources managed by user-defined callbacks
    using (var stream = File.Create(Path.Combine(savingCallbacks.RootFolder, "document.html")))
    {
        document.Save(stream, options);
    }

    // Manually append data to the CSS stream
    using (var writer = new StreamWriter(savingCallbacks.CssStream))
    {
        writer.WriteLine();
        writer.WriteLine("/* This line is appended to stream manually by user */");
    }
}
```
