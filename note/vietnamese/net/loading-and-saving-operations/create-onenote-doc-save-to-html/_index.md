# Lưu OneNote dưới dạng HTML bằng Aspose.Note

## Giới thiệu

Nếu bạn cần **lưu OneNote dưới dạng HTML** để xuất bản web, báo cáo hoặc lưu trữ, Aspose.Note cho .NET cung cấp cho bạn một giải pháp sạch, chỉ mã, hoạt động trên Windows, Linux và macOS mà không cần Microsoft Office. Trong hướng dẫn này, bạn sẽ học cách tạo tài liệu OneNote từ đầu, thao tác các trang của nó, và xuất toàn bộ sổ ghi—hoặc các trang đã chọn—ra HTML bằng nhiều tùy chọn khác nhau. Hướng dẫn sẽ đưa bạn qua từng bước, giải thích lý do mỗi cài đặt quan trọng, và trình bày các kịch bản thực tế nơi xuất HTML tỏa sáng, chẳng hạn như nhúng nội dung OneNote vào cổng kiến thức hoặc tạo tài liệu tĩnh.

## Câu trả lời nhanh
- **Phương pháp chính để xuất là gì?** Gọi `document.Save("output.html", SaveFormat.Html)` trên một thể hiện `Aspose.Note.Document`.  
- **Tôi có thể xuất một trang duy nhất không?** Có—sử dụng `document.Save(pageRange, "page.html", SaveFormat.Html)`.  
- **Các tài nguyên (hình ảnh, CSS) có được nhúng tự động không?** Mặc định chúng được lưu dưới dạng các tệp riêng; bạn có thể nhúng chúng vào một memory stream bằng `HtmlSaveOptions`.  
- **.NET Core có được hỗ trợ không?** Chắc chắn—Aspose.Note hoạt động với .NET 5, .NET 6 và .NET Core 3.1+.  
- **Tôi có cần giấy phép để phát triển không?** Bản dùng thử miễn phí 30 ngày đủ cho việc thử nghiệm; giấy phép sẽ loại bỏ watermark đánh giá.

## “Lưu OneNote dưới dạng HTML” là gì?

**“Lưu OneNote dưới dạng HTML”** đề cập đến quá trình chuyển đổi một sổ ghi OneNote, được lưu ở định dạng nhị phân độc quyền, thành các tệp HTML tiêu chuẩn có thể được bất kỳ trình duyệt web nào hiển thị. Quá trình chuyển đổi giữ nguyên cấu trúc phân cấp, định dạng văn bản, hình ảnh, bảng và các tài nguyên khác của sổ ghi, đồng thời dịch các yếu tố đặc thù của OneNote sang các thẻ HTML và kiểu CSS tương đương. Điều này cho phép các nhà phát triển hiển thị nội dung OneNote trên website, tích hợp vào hệ thống quản lý nội dung, hoặc lưu trữ dưới dạng dễ truy cập mà không cần client OneNote.

## Tại sao chuyển đổi OneNote sang HTML?

Aspose.Note có thể xuất ra **hơn 50 định dạng** (bao gồm PDF, DOCX, PNG và HTML) và xử lý sổ ghi **lên tới 500 trang** mà không cần tải toàn bộ tệp vào bộ nhớ, nhờ kiến trúc streaming. Chuyển đổi sang HTML cho phép bạn:

1. **Xuất bản nội dung trên web** mà không cần client OneNote.  
2. **Tích hợp với các hệ thống quản lý nội dung** chỉ chấp nhận HTML.  
3. **Trích xuất văn bản có thể tìm kiếm** cho các công cụ lập chỉ mục.  
4. **Tái sử dụng tài nguyên ở mức trang** (hình ảnh, phông chữ) trong các dự án web khác.

## Cách lưu OneNote dưới dạng HTML – Bước từng bước

Lưu OneNote dưới dạng HTML bao gồm ba hành động chính: tải hoặc tạo một sổ ghi, cấu hình các tùy chọn xuất mong muốn, và gọi phương thức `Save` thích hợp. Đầu tiên, bạn tạo hoặc tải một đối tượng `Document`, sau đó có thể điều chỉnh `HtmlSaveOptions` để kiểm soát việc xử lý tài nguyên, tạo CSS và mã hoá. Cuối cùng, bạn gọi `document.Save` với đường dẫn tệp hoặc stream, tùy chọn chỉ định `PageRange` để giới hạn phạm vi xuất. Cách tiếp cận có hệ thống này đảm bảo kết quả nhất quán trên các môi trường khác nhau và cho phép bạn tùy chỉnh đầu ra phù hợp với nhu cầu dự án.

### Bước 1: Tạo tài liệu OneNote với tùy chọn mặc định

Trong ví dụ đầu tiên này chúng ta khởi tạo một `Document` mới, thêm một trang duy nhất với tiêu đề, và lưu toàn bộ sổ ghi ra HTML bằng `HtmlSaveOptions` mặc định. Không cần tham số bổ sung, đây là cách nhanh nhất để **lưu OneNote dưới dạng HTML**.

```csharp
using System;
using System.Drawing;
using System.Globalization;
using System.IO;

using Aspose.Note.Saving;
using Aspose.Note.Saving.Html;

```

### Bước 2: Xuất một phạm vi trang cụ thể sang HTML

Thường bạn chỉ cần một phần của sổ ghi—ví dụ, xuất một chương ghi chú họp. Bằng cách truyền một đối tượng `PageRange` vào `Save`, bạn có thể chỉ định các trang 1‑3 trong khi để lại các trang còn lại nguyên vẹn.

```csharp
public static void CreateAndSaveToHTMLUsingDefaultOptions()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateOneNoteDocAndSaveToHTML_out.html");
    doc.Save(outputPath);

    Console.WriteLine("\nOneNote document created successfully.\nFile saved at " + outputPath);
}
```

### Bước 3: Lưu dưới dạng HTML vào Memory Stream với tài nguyên nhúng

Khi bạn muốn một payload HTML tự chứa duy nhất (ví dụ, cho phản hồi API), nhúng CSS, phông chữ và hình ảnh trực tiếp vào một `MemoryStream`. `HtmlSaveOptions` cho phép bạn bật `EmbedResources` thành `true`, tạo ra một chuỗi HTML gọn gàng.

```csharp
public static void CreateAndSavePageRange()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateAndSavePageRange_out.html");
    doc.Save(outputPath, new HtmlSaveOptions { PageCount = 1, PageIndex = 0 });

    Console.WriteLine("\nOneNote document created successfully and saved as page range.\nFile saved at " + outputPath);
}
```

### Bước 4: Lưu dưới dạng HTML vào các tệp với tài nguyên trong các tệp riêng biệt

Nếu bạn thích đầu ra dạng thư mục web truyền thống—tệp HTML cộng với một thư mục tài nguyên—vô hiệu hoá việc nhúng và chỉ định thư mục đầu ra. Cách này cho các tệp nguồn sạch hơn và dễ dàng cache hình ảnh hoặc stylesheet.

```csharp
public static void SaveAsHTMLToMemoryStreamWithEmbeddedResources()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportEmbedded,
        ExportFonts = ResourceExportType.ExportEmbedded,
        ExportImages = ResourceExportType.ExportEmbedded,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to a memory stream
    var memoryStream = new MemoryStream();
    document.Save(memoryStream, options);
}
```

### Bước 5: Lưu dưới dạng HTML sử dụng Callbacks để quản lý tài nguyên

Các kịch bản nâng cao, chẳng hạn lưu hình ảnh vào cơ sở dữ liệu hoặc bucket đám mây, đòi hỏi xử lý tùy chỉnh. Bằng cách cung cấp một delegate `ResourceSavingCallback`, bạn kiểm soát cách mỗi tài nguyên được lưu, trong khi API vẫn tạo ra các tham chiếu HTML chính xác.

```csharp
public static void SaveAsHTMLToFileWithResourcesInSeparateFiles()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportAsStream,
        ExportFonts = ResourceExportType.ExportAsStream,
        ExportImages = ResourceExportType.ExportAsStream,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to HTML file with resources stored in separate files
    document.Save(Path.Combine(dataDir, "document_out.html"), options);
}
```

| Vấn đề | Nguyên nhân | Cách khắc phục |

| **Hình ảnh bị hỏng** | Các tài nguyên đã được lưu vào thư mục khác so với tệp HTML. | Đảm bảo `HtmlSaveOptions.ResourcesFolder` khớp với thư mục bạn tham chiếu, hoặc bật `EmbedResources`. |

| **Vấn đề mã hoá HTML cho ký tự không phải ASCII** | `Encoding` sai được đặt trên `HtmlSaveOptions`. | Đặt `HtmlSaveOptions.Encoding = Encoding.UTF8`. |

```csharp
public static void SaveAsHTMLToMemoryStreamWithCallBacksToSaveResources()
{
    // Specify the saving callbacks configuration
    var savingCallbacks = new UserSavingCallbacks()
    {
        RootFolder = "documentFolder",
        CssFolder = "css",
        KeepCssStreamOpened = true,
        ImagesFolder = "images",
        FontsFolder = "fonts"
    };

    // Specify HTML saving options
    var options = new HtmlSaveOptions
    {
        FontFaceTypes = FontFaceType.Ttf,
        CssSavingCallback = savingCallbacks,
        FontSavingCallback = savingCallbacks,
        ImageSavingCallback = savingCallbacks
    };

    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Save the document to HTML format with resources managed by user-defined callbacks
    using (var stream = File.Create(Path.Combine(savingCallbacks.RootFolder, "document.html")))
    {
        document.Save(stream, options);
    }

    // Manually append data to the CSS stream
    using (var writer = new StreamWriter(savingCallbacks.CssStream))
    {
        writer.WriteLine();
        writer.WriteLine("/* This line is appended to stream manually by user */");
    }
}
```
