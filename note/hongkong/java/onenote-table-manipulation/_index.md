## 如何程式化建立 OneNote 表格？

`Document` 代表載入記憶體中的 OneNote 筆記本檔案。  
`Page.getTables().add()` 會在頁面上建立具有指定列數與欄數的新 Table。

先載入 `Document` 實例，新增一個 `Page`，然後使用 `Page.getTables().add()` 並傳入所需的列與欄數。建立後即可設定儲存格值、套用邊框與樣式——全部透過流暢的 Java 呼叫完成。這個兩步驟模式（建立 → 設定）讓您即時建構表格，即使是多頁筆記本亦可。
