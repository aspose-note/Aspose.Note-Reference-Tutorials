## 常見問題

**Q: 可以在商業專案中使用 Aspose.Note for Java 嗎？**  
A: 可以，正式上線需購買商業授權，但可使用免費試用版進行評估。

**Q: 支援哪些 Java 版本？**  
A: 此程式庫支援 Java 8 以及之後的版本。

**Q: 取得附件時，若 OneNote 檔案很大該怎麼處理？**  
A: 可使用串流模式載入文件，或逐段（分區/頁面）處理，以降低記憶體使用量。

**Q: 能否為附加的檔案設定自訂圖示？**  
A: 當然可以 – 「使用 Java 在 OneNote 附加檔案並設定圖示」教學會示範如何以程式指定圖示。

**Q: 使用 API 前需要在伺服器上安裝 OneNote 嗎？**  
A: 不需要，Aspose.Note 可獨立於 OneNote 應用程式運作，直接讀寫 OneNote 檔案格式。
