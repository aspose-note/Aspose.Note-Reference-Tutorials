## 常見問題

**Q: 我可以直接將 OneNote 筆記本匯出為 PDF 而不經過中間格式嗎？**  
A: 可以。Aspose.Note 提供一鍵式的 `save` 方法，將筆記本寫入 PDF 檔案，保留版面配置與影像。

**Q: 我該如何只從 OneNote 頁面擷取影像？**  
A: 使用 `Document.getImages()` API 列舉影像物件，並將每個影像儲存為所需的格式（PNG、JPEG 等）。

**Q: 有沒有方法可以提升大型筆記本的匯出速度？**  
A: 在儲存前啟用 `Document.optimizeResources()` 方法，並考慮以平行執行緒處理各區段。

**Q: 我需要在伺服器上安裝其他軟體嗎？**  
A: 不需要。Aspose.Note 是純 Java 函式庫，只需將 JAR 檔案加入專案的 classpath 即可。

**Q: 生產環境使用有哪些授權方案？**  
A: Aspose 提供永久授權與訂閱模式，亦提供免費試用供評估使用。
