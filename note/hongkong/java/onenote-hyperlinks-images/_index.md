## 常見問題

**Q: 我可以為任何圖片格式（PNG、JPEG、GIF）加入超連結嗎？**  
A: 可以。Aspose.Note 支援所有標準圖片格式；超連結會附加於圖片物件，不論其類型為何。

**Q: 我需要在編輯模式下開啟 OneNote 檔案才能加入超連結嗎？**  
A: 不需要。您可以在記憶體中完整建立或修改文件，然後儲存為 `.one` 檔案。

**Q: 超連結在行動版 OneNote 應用程式上會正常運作嗎？**  
A: 當然會。產生的超連結儲存在 OneNote 檔案格式中，桌面、網頁與行動客戶端皆能辨識。

**Q: 我要如何驗證超連結已正確加入？**  
A: 儲存後，在 OneNote 中開啟檔案並點擊圖片；連結的 URL 應會在預設瀏覽器中開啟。

**Q: 是否可以為單一圖片加入多個超連結？**  
A: 每張圖片只能容納一個超連結。若需多個連結，可考慮使用合成圖片，將不同可點擊區域分開，或是加入多個圖片物件。
