## 常見問題

**Q: 我可以在商業專案中使用 Aspose.Note for Java 嗎？**  
A: 可以。正式上線需具備有效的商業授權，但可使用免費試用版進行評估。

**Q: 支援哪些 Java 版本？**  
A: 此函式庫支援 Java 8、11 以及更新的 LTS 版本。

**Q: 如何在 OneNote 頁面加入超連結？**  
A: 使用 Aspose.Note 提供的 `Hyperlink` 類別定義 URL，並將其附加至 `RichText` 節點。

**Q: 是否可以為影像設定替代文字以提升可及性？**  
A: 當然可以。`Image` 物件具備 `AltText` 屬性，您可以以程式方式設定。

**Q: 大型筆記本的效能優化建議是什麼？**  
A: 啟用計量授權、盡可能重複使用 `Document` 實例，並以串流方式處理大型資源，避免完整載入記憶體。
