# 儲存 OneNote 為 HTML 使用 Aspose.Note

## 介紹

如果您需要 **將 OneNote 儲存為 HTML** 以進行網站發佈、報告或存檔，Aspose.Note for .NET 為您提供一個乾淨、僅程式碼的解決方案，能在 Windows、Linux 與 macOS 上運作，且不需要 Microsoft Office。在本教學中，您將學習如何從頭建立 OneNote 文件、操作其頁面，並使用各種選項將整個筆記本或選取的頁面匯出為 HTML。指南會逐步說明每個步驟、解釋各設定的意義，並展示 HTML 匯出在實務情境中的優勢，例如在知識庫入口網站嵌入 OneNote 內容或產生靜態文件。

## 快速解答
- **主要的匯出方法是什麼？** Call `document.Save("output.html", SaveFormat.Html)` on an `Aspose.Note.Document` instance.  
- **我可以匯出單一頁面嗎？** Yes—use `document.Save(pageRange, "page.html", SaveFormat.Html)`.  
- **資源（圖片、CSS）會自動嵌入嗎？** By default they are saved as separate files; you can embed them into a memory stream with `HtmlSaveOptions`.  
- **支援 .NET Core 嗎？** Absolutely—Aspose.Note works with .NET 5, .NET 6, and .NET Core 3.1+.  
- **開發時需要授權嗎？** A free 30‑day trial works for testing; a license removes evaluation watermarks.

## 何謂「將 OneNote 儲存為 HTML」？
**「將 OneNote 儲存為 HTML」** 指的是將 OneNote 筆記本（以專有二進位格式儲存）轉換成標準的 HTML 檔案，這些檔案可在任何網頁瀏覽器中呈現。轉換過程會保留筆記本的層級結構、文字格式、圖片、表格及其他資源，同時將 OneNote 專屬的元素轉換為等效的 HTML 標籤與 CSS 樣式。這讓開發人員能在網站上顯示 OneNote 內容、整合至內容管理系統，或以通用可存取的格式進行存檔，而無需 OneNote 客戶端。

## 為何將 OneNote 轉換為 HTML？
Aspose.Note 能匯出至 **50+ 輸出格式**（包括 PDF、DOCX、PNG 與 HTML），且可在不將整個檔案載入記憶體的情況下處理 **最多 500 頁** 的筆記本，這歸功於其串流架構。將筆記本轉換為 HTML 可讓您：

1. **在網路上發佈內容**，無需 OneNote 客戶端。  
2. **與僅接受 HTML 的內容管理系統整合**。  
3. **擷取可搜尋的文字** 供索引引擎使用。  
4. **在其他網站專案中重複使用頁面層級資源**（圖片、字型）。

## 前置條件

- 已安裝 .NET Framework 4.6+ **或** .NET Core 3.1+。  
- 已在專案中加入 Aspose.Note for .NET。可從 [here](https://releases.aspose.com/note/net/) 下載。  
- 具備 OneNote 階層結構（筆記本 → 區段 → 頁面）的基本認識。

## 如何將 OneNote 儲存為 HTML – 步驟說明

將 OneNote 儲存為 HTML 包含三個核心動作：載入或建立筆記本、設定所需的匯出選項，並呼叫相應的 `Save` 方法。首先，建立或載入 `Document` 物件，接著可調整 `HtmlSaveOptions` 以控制資源處理、CSS 產生與編碼。最後，使用 `document.Save` 並提供檔案路徑或串流，必要時指定 `PageRange` 以限制匯出範圍。此系統化方法確保在不同環境下產出一致的結果，並讓您依專案需求客製化輸出。

### 步驟 1：使用預設選項建立 OneNote 文件
在此第一個範例中，我們建立一個全新的 `Document`，加入一個帶標題的單一頁面，並使用預設的 `HtmlSaveOptions` 將整個筆記本儲存為 HTML。無需額外參數，這是最快速的 **將 OneNote 儲存為 HTML** 方式。

```csharp
using System;
using System.Drawing;
using System.Globalization;
using System.IO;

using Aspose.Note.Saving;
using Aspose.Note.Saving.Html;

```

### 步驟 2：匯出特定頁面範圍為 HTML
通常您只需要部分頁面，例如匯出會議記錄的某一章節。將 `PageRange` 物件傳遞給 `Save` 後，即可針對第 1‑3 頁進行匯出，而其餘頁面保持不變。

```csharp
public static void CreateAndSaveToHTMLUsingDefaultOptions()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateOneNoteDocAndSaveToHTML_out.html");
    doc.Save(outputPath);

    Console.WriteLine("\nOneNote document created successfully.\nFile saved at " + outputPath);
}
```

### 步驟 3：將 HTML 儲存至 Memory Stream 並嵌入資源
當您需要單一自包含的 HTML 負載（例如 API 回應）時，可將 CSS、字型與圖片直接嵌入 `MemoryStream`。`HtmlSaveOptions` 允許您將 `EmbedResources` 設為 `true`，產生緊湊的 HTML 字串。

```csharp
public static void CreateAndSavePageRange()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateAndSavePageRange_out.html");
    doc.Save(outputPath, new HtmlSaveOptions { PageCount = 1, PageIndex = 0 });

    Console.WriteLine("\nOneNote document created successfully and saved as page range.\nFile saved at " + outputPath);
}
```

### 步驟 4：將 HTML 儲存為檔案，資源分離為獨立檔案
如果您偏好傳統的網站資料夾輸出——HTML 檔案加上資產資料夾——可停用嵌入並指定輸出目錄。此方式可產生較為整潔的原始檔案，且更方便對圖片或樣式表進行快取。

```csharp
public static void SaveAsHTMLToMemoryStreamWithEmbeddedResources()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportEmbedded,
        ExportFonts = ResourceExportType.ExportEmbedded,
        ExportImages = ResourceExportType.ExportEmbedded,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to a memory stream
    var memoryStream = new MemoryStream();
    document.Save(memoryStream, options);
}
```

### 步驟 5：使用回呼函式管理資源並儲存為 HTML
進階情境（例如將圖片儲存至資料庫或雲端儲存桶）需要自訂處理。提供 `ResourceSavingCallback` 委派後，您即可精確控制每個資源的保存方式，同時 API 仍會產生正確的 HTML 參考。

```csharp
public static void SaveAsHTMLToFileWithResourcesInSeparateFiles()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportAsStream,
        ExportFonts = ResourceExportType.ExportAsStream,
        ExportImages = ResourceExportType.ExportAsStream,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to HTML file with resources stored in separate files
    document.Save(Path.Combine(dataDir, "document_out.html"), options);
}
```

## 常見問題與解決方案

| 問題 | 發生原因 | 解決方法 |
|-------|----------------|-----|
| **圖片顯示損壞** | 資源被儲存至與 HTML 檔案不同的資料夾。 | 確保 `HtmlSaveOptions.ResourcesFolder` 與您參考的資料夾相同，或啟用 `EmbedResources`。 |
| **CSS 樣式遺失** | 預設的 CSS 產生被停用。 | 將 `HtmlSaveOptions.GenerateCss = true`（預設值）或手動嵌入 CSS。 |
| **大型筆記本導致記憶體不足** | 使用非串流的 `Document` 建構子會將整個檔案載入。 | 使用 `Document.Load(stream, LoadOptions)` 並將 `LoadOptions.LoadFromMemory = true` 以串流方式載入頁面。 |
| **非 ASCII 字元的 HTML 編碼問題** | `HtmlSaveOptions` 設定了錯誤的 `Encoding`。 | 將 `HtmlSaveOptions.Encoding = Encoding.UTF8` 設定為 UTF-8。 |
| **在 Linux 上儲存失敗** | 缺少影像處理的原生相依性。 | 在 Linux 主機上安裝 `libgdiplus`。 |

**Q: Aspose.Note 是否支援除 HTML 之外的其他格式轉換？**  
A: 當然可以；您可以轉換為 PDF、DOCX、PNG、JPEG 等等，支援超過 30 種輸出格式。

**Q: Aspose.Note 是否相容於 .NET Core 應用程式？**  
A: 是的——完整支援 .NET Core 3.1、.NET 5、.NET 6 以及更高版本。

**Q: 我可以使用 Aspose.Note 從 OneNote 文件中擷取文字與圖片嗎？**  
A: 您可以使用 `Page.GetText()` 取得純文字，並透過 `Page.GetResources(ResourceType.Image)` 取得圖片。

public static void SaveAsHTMLToMemoryStreamWithCallBacksToSaveResources()
{
    // Specify the saving callbacks configuration
    var savingCallbacks = new UserSavingCallbacks()
    {
        RootFolder = "documentFolder",
        CssFolder = "css",
        KeepCssStreamOpened = true,
        ImagesFolder = "images",
        FontsFolder = "fonts"
    };

    // Specify HTML saving options
    var options = new HtmlSaveOptions
    {
        FontFaceTypes = FontFaceType.Ttf,
        CssSavingCallback = savingCallbacks,
        FontSavingCallback = savingCallbacks,
        ImageSavingCallback = savingCallbacks
    };

    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Save the document to HTML format with resources managed by user-defined callbacks
    using (var stream = File.Create(Path.Combine(savingCallbacks.RootFolder, "document.html")))
    {
        document.Save(stream, options);
    }

    // Manually append data to the CSS stream
    using (var writer = new StreamWriter(savingCallbacks.CssStream))
    {
        writer.WriteLine();
        writer.WriteLine("/* This line is appended to stream manually by user */");
    }
}
