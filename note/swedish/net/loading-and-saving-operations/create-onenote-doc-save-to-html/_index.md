# Spara OneNote som HTML med Aspose.Note

## Introduktion

Om du behöver **spara OneNote som HTML** för webbpublicering, rapportering eller arkivering, ger Aspose.Note för .NET dig en ren, kod‑endast lösning som fungerar på Windows, Linux och macOS utan att kräva Microsoft Office. I den här handledningen kommer du att lära dig hur du skapar ett OneNote‑dokument från grunden, manipulerar dess sidor och exporterar hela anteckningsboken—eller utvalda sidor—till HTML med en mängd olika alternativ. Guiden går igenom varje steg, förklarar varför varje inställning är viktig och visar verkliga scenarier där HTML‑exporten glänser, såsom att bädda in OneNote‑innehåll i en kunskapsbasportal eller generera statisk dokumentation.

## Snabba svar
- **Vad är den primära metoden för export?** Anropa `document.Save("output.html", SaveFormat.Html)` på en `Aspose.Note.Document`‑instans.  
- **Kan jag exportera en enskild sida?** Ja—använd `document.Save(pageRange, "page.html", SaveFormat.Html)`.  
- **Bäddar resurser (bilder, CSS) in automatiskt?** Som standard sparas de som separata filer; du kan bädda in dem i ett minnesström med `HtmlSaveOptions`.  
- **Stöds .NET Core?** Absolut—Aspose.Note fungerar med .NET 5, .NET 6 och .NET Core 3.1+.  
- **Behöver jag en licens för utveckling?** En gratis 30‑dagars provversion fungerar för testning; en licens tar bort utvärderingsvattenstämplar.

## Vad är “spara OneNote som HTML”?

**“Spara OneNote som HTML”** avser processen att konvertera en OneNote‑anteckningsbok, som lagras i ett proprietärt binärt format, till standard‑HTML‑filer som kan renderas av vilken webbläsare som helst. Konverteringen bevarar anteckningsbokens hierarkiska struktur, textformatering, bilder, tabeller och andra resurser, samtidigt som OneNote‑specifika element översätts till motsvarande HTML‑taggar och CSS‑stilar. Detta möjliggör för utvecklare att visa OneNote‑innehåll på webbplatser, integrera det i innehållshanteringssystem eller arkivera det i ett universellt åtkomligt format utan att kräva OneNote‑klienten.

## Varför konvertera OneNote till HTML?

Aspose.Note kan exportera till **50+ utdataformat** (inklusive PDF, DOCX, PNG och HTML) och bearbetar anteckningsböcker med **upp till 500 sidor** utan att ladda hela filen i minnet, tack vare sin streaming‑arkitektur. Att konvertera till HTML låter dig:

1. **Publicera innehåll på webben** utan att kräva OneNote‑klienten.  
2. **Integrera med innehållshanteringssystem** som endast accepterar HTML.  
3. **Extrahera sökbar text** för indexeringsmotorer.  
4. **Återanvänd resurser på sidnivå** (bilder, typsnitt) i andra webbprojekt.

## Så sparar du OneNote som HTML – Steg för steg

Att spara OneNote som HTML innebär tre grundläggande åtgärder: ladda eller skapa en anteckningsbok, konfigurera önskade exportalternativ och anropa den lämpliga `Save`‑metoden. Först skapar eller laddar du ett `Document`‑objekt, sedan kan du justera `HtmlSaveOptions` för att styra resurshantering, CSS‑generering och kodning. Slutligen anropar du `document.Save` med antingen en filsökväg eller ett strömobjekt, eventuellt med en `PageRange` för att begränsa exporten. Detta systematiska tillvägagångssätt säkerställer konsekventa resultat i olika miljöer och låter dig anpassa utdata efter ditt projekts behov.

### Steg 1: Skapa ett OneNote‑dokument med standardalternativ

I det här första exemplet instansierar vi ett nytt `Document`, lägger till en enda sida med en titel och sparar hela anteckningsboken till HTML med standard‑`HtmlSaveOptions`. Inga extra parametrar krävs, vilket gör det till det snabbaste sättet att **spara OneNote som HTML**.

```csharp
using System;
using System.Drawing;
using System.Globalization;
using System.IO;

using Aspose.Note.Saving;
using Aspose.Note.Saving.Html;

```

### Steg 2: Exportera ett specifikt sidintervall till HTML

Ofta behöver du bara ett delmängd av sidor—till exempel exportera ett kapitel med mötesanteckningar. Genom att skicka ett `PageRange`‑objekt till `Save` kan du rikta in dig på sidor 1‑3 medan resten lämnas orörda.

```csharp
public static void CreateAndSaveToHTMLUsingDefaultOptions()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateOneNoteDocAndSaveToHTML_out.html");
    doc.Save(outputPath);

    Console.WriteLine("\nOneNote document created successfully.\nFile saved at " + outputPath);
}
```

### Steg 3: Spara som HTML till ett minnesström med inbäddade resurser

När du vill ha en enda självständig HTML‑payload (t.ex. för API‑svar), bädda in CSS, typsnitt och bilder direkt i ett `MemoryStream`. `HtmlSaveOptions` låter dig sätta `EmbedResources` till `true`, vilket ger en kompakt HTML‑sträng.

```csharp
public static void CreateAndSavePageRange()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateAndSavePageRange_out.html");
    doc.Save(outputPath, new HtmlSaveOptions { PageCount = 1, PageIndex = 0 });

    Console.WriteLine("\nOneNote document created successfully and saved as page range.\nFile saved at " + outputPath);
}
```

### Steg 4: Spara som HTML till filer med resurser i separata filer

Om du föredrar klassisk webb‑mapp‑utgång—HTML‑fil plus en mapp med tillgångar—inaktivera inbäddning och ange en utdatamapp. Detta tillvägagångssätt ger renare källfiler och enklare cachning av bilder eller stilmallar.

```csharp
public static void SaveAsHTMLToMemoryStreamWithEmbeddedResources()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportEmbedded,
        ExportFonts = ResourceExportType.ExportEmbedded,
        ExportImages = ResourceExportType.ExportEmbedded,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to a memory stream
    var memoryStream = new MemoryStream();
    document.Save(memoryStream, options);
}
```

### Steg 5: Spara som HTML med återanrop för att hantera resurser

Avancerade scenarier, såsom att lagra bilder i en databas eller en molnbucket, kräver anpassad hantering. Genom att tillhandahålla en `ResourceSavingCallback`‑delegat styr du exakt hur varje resurs sparas, medan API:t fortsätter att generera korrekta HTML‑referenser.

```csharp
public static void SaveAsHTMLToFileWithResourcesInSeparateFiles()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportAsStream,
        ExportFonts = ResourceExportType.ExportAsStream,
        ExportImages = ResourceExportType.ExportAsStream,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to HTML file with resources stored in separate files
    document.Save(Path.Combine(dataDir, "document_out.html"), options);
}
```

## Vanliga problem och lösningar

| Problem | Varför det händer | Lösning |
|---------|-------------------|---------|
| **Bilder visas trasiga** | Resurser sparades i en annan mapp än HTML‑filen. | Se till att `HtmlSaveOptions.ResourcesFolder` matchar den mapp du refererar till, eller aktivera `EmbedResources`. |
| **CSS‑stilar saknas** | Standard‑CSS‑generering var inaktiverad. | Sätt `HtmlSaveOptions.GenerateCss = true` (standard) eller bädda in CSS manuellt. |
| **Stora anteckningsböcker orsakar OutOfMemory** | Användning av den icke‑streamande `Document`‑konstruktorn laddar hela filen. | Använd `Document.Load(stream, LoadOptions)` med `LoadOptions.LoadFromMemory = true` för att streama sidor. |
| **HTML‑kodningsproblem för icke‑ASCII‑tecken** | Fel `Encoding` satt på `HtmlSaveOptions`. | Sätt `HtmlSaveOptions.Encoding = Encoding.UTF8`. |
| **Sparande misslyckas på Linux** | Saknade inhemska beroenden för bildbehandling. | Installera `libgdiplus` på Linux‑värden. |

```csharp
public static void SaveAsHTMLToMemoryStreamWithCallBacksToSaveResources()
{
    // Specify the saving callbacks configuration
    var savingCallbacks = new UserSavingCallbacks()
    {
        RootFolder = "documentFolder",
        CssFolder = "css",
        KeepCssStreamOpened = true,
        ImagesFolder = "images",
        FontsFolder = "fonts"
    };

    // Specify HTML saving options
    var options = new HtmlSaveOptions
    {
        FontFaceTypes = FontFaceType.Ttf,
        CssSavingCallback = savingCallbacks,
        FontSavingCallback = savingCallbacks,
        ImageSavingCallback = savingCallbacks
    };

    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Save the document to HTML format with resources managed by user-defined callbacks
    using (var stream = File.Create(Path.Combine(savingCallbacks.RootFolder, "document.html")))
    {
        document.Save(stream, options);
    }

    // Manually append data to the CSS stream
    using (var writer = new StreamWriter(savingCallbacks.CssStream))
    {
        writer.WriteLine();
        writer.WriteLine("/* This line is appended to stream manually by user */");
    }
}
```
