## Hur man programmerat skapar OneNote‑tabell?
`Document` representerar en OneNote‑anteckningsbokfil som laddats in i minnet.  
`Page.getTables().add()` skapar en ny Table på sidan med de angivna raderna och kolumnerna.

Läs in en `Document`‑instans, lägg till en `Page` och anropa sedan `Page.getTables().add()` med önskat antal rader och kolumner. Efter skapandet kan du sätta cellvärden, applicera kanter och formatera tabellen — allt via flytande Java‑anrop. Detta tvåstegsmönster (skapa → konfigurera) låter dig bygga tabeller omedelbart, även för flersidiga anteckningsböcker.
