## Vanliga frågor

**Q: Kan jag lägga till en hyperlänk till vilket bildformat som helst (PNG, JPEG, GIF)?**  
**A:** Ja. Aspose.Note stödjer alla standardbildformat; hyperlänken fästs på bildobjektet oavsett typ.

**Q: Behöver jag öppna OneNote‑filen i redigeringsläge för att lägga till hyperlänken?**  
**A:** Nej. Du kan skapa eller ändra ett dokument helt i minnet och sedan spara det till en `.one`‑fil.

**Q: Fungerar hyperlänken i mobila OneNote‑appar?**  
**A:** Absolut. Den genererade hyperlänken lagras i OneNote‑filformatet, vilket känns igen av skrivbords-, webb- och mobilklienter.

**Q: Hur kan jag verifiera att hyperlänken lades till korrekt?**  
**A:** Efter att du sparat, öppna filen i OneNote och klicka på bilden; den länkade URL‑en bör öppnas i standardwebbläsaren.

**Q: Finns det ett sätt att lägga till flera hyperlänkar på en enda bild?**  
**A:** En bild kan bara ha en hyperlänk. För att erbjuda flera länkar, överväg att använda en sammansatt bild med separata klickbara regioner eller lägga till separata bildobjekt.
