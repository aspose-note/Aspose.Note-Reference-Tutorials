# OneNote'u Aspose.Note ile HTML Olarak Kaydet

## Giriş

Web yayıncılığı, raporlama veya arşivleme için **OneNote'u HTML olarak kaydetmeniz** gerekiyorsa, Aspose.Note for .NET, Microsoft Office gerektirmeden Windows, Linux ve macOS'ta çalışan temiz, yalnızca kod tabanlı bir çözüm sunar. Bu öğreticide, sıfırdan bir OneNote belgesi oluşturmayı, sayfalarını manipüle etmeyi ve çeşitli seçenekler kullanarak tüm not defterini—veya seçili sayfaları—HTML'e dışa aktarmayı öğreneceksiniz. Kılavuz, her adımı size gösterir, her ayarın neden önemli olduğunu açıklar ve HTML dışa aktarmanın parladığı gerçek dünya senaryolarını sunar; örneğin OneNote içeriğini bir bilgi tabanı portalına gömmek veya statik dokümantasyon üretmek gibi.

## Hızlı Yanıtlar
- **Dışa aktarmanın temel yöntemi nedir?** `Aspose.Note.Document` örneği üzerinde `document.Save("output.html", SaveFormat.Html)` çağrısı yapın.  
- **Tek bir sayfayı dışa aktarabilir miyim?** Evet—`document.Save(pageRange, "page.html", SaveFormat.Html)` kullanın.  
- **Kaynaklar (görseller, CSS) otomatik olarak gömülür mü?** Varsayılan olarak ayrı dosyalar olarak kaydedilir; `HtmlSaveOptions` ile bunları bir bellek akışına gömebilirsiniz.  
- **.NET Core destekleniyor mu?** Kesinlikle—Aspose.Note, .NET 5, .NET 6 ve .NET Core 3.1+ ile çalışır.  
- **Geliştirme için lisansa ihtiyacım var mı?** Ücretsiz 30‑günlük deneme sürümü test için yeterlidir; bir lisans değerlendirme filigranlarını kaldırır.

## “save onenote as html” nedir?

**“OneNote'u HTML olarak kaydetmek”**, özel ikili formatta depolanan bir OneNote not defterini, herhangi bir web tarayıcısı tarafından render edilebilen standart HTML dosyalarına dönüştürme sürecini ifade eder. Dönüşüm, not defterinin hiyerarşik yapısını, metin biçimlendirmesini, görselleri, tabloları ve diğer kaynakları korurken, OneNote‑özel öğeleri eşdeğer HTML etiketleri ve CSS stilleriyle değiştirir. Bu sayede geliştiriciler, OneNote içeriğini web sitelerinde gösterebilir, içerik‑yönetim sistemlerine entegre edebilir veya OneNote istemcisine ihtiyaç duymadan evrensel bir formatta arşivleyebilir.

## Neden OneNote'u HTML'e Dönüştürmeliyiz?

Aspose.Note, **50+ çıktı formatı** (PDF, DOCX, PNG ve HTML dahil) sunar ve akış mimarisi sayesinde **500 sayfaya kadar** not defterlerini tüm dosyayı belleğe yüklemeden işler. HTML'e dönüştürmek şunları sağlar:

1. **İçeriği webde yayınlamak** ve OneNote istemcisine ihtiyaç duymamak.  
2. **HTML kabul eden içerik‑yönetim sistemleri**yle entegrasyon.  
3. **Arama motorları için metin çıkarma** ve indeksleme.  
4. **Sayfa‑düzeyindeki kaynakları** (görseller, fontlar) diğer web projelerinde yeniden kullanma.

## OneNote'u HTML olarak kaydetme – Adım Adım

OneNote'u HTML olarak kaydetmek üç temel eylemi içerir: bir not defteri yüklemek veya oluşturmak, istenen dışa aktarma seçeneklerini yapılandırmak ve uygun `Save` metodunu çağırmak. İlk olarak bir `Document` nesnesi oluşturur veya yüklersiniz, ardından `HtmlSaveOptions` ile kaynak yönetimi, CSS üretimi ve kodlamayı ayarlarsınız. Son olarak, dosya yolu ya da akış belirterek `document.Save` çağrısı yaparsınız; isteğe bağlı olarak `PageRange` belirterek dışa aktarmayı sınırlayabilirsiniz. Bu sistematik yaklaşım, farklı ortamlar arasında tutarlı sonuçlar verir ve çıktıyı projenizin ihtiyaçlarına göre özelleştirmenizi sağlar.

### Adım 1: Varsayılan Seçeneklerle Bir OneNote Belgesi Oluşturma

Bu ilk örnekte yeni bir `Document` oluşturur, tek bir sayfa başlığı ekler ve varsayılan `HtmlSaveOptions` ile tüm not defterini HTML olarak kaydederiz. Ek parametre gerekmez; bu, **OneNote'u HTML olarak kaydetmenin** en hızlı yoludur.

```csharp
using System;
using System.Drawing;
using System.Globalization;
using System.IO;

using Aspose.Note.Saving;
using Aspose.Note.Saving.Html;

```

### Adım 2: Belirli Bir Sayfa Aralığını HTML'e Dışa Aktarma

Genellikle sadece bir sayfa alt kümesi gerekir—örneğin bir toplantı notları bölümü. `Save` metoduna bir `PageRange` nesnesi geçirerek 1‑3. sayfaları hedefleyebilir, geri kalanını dokunmadan bırakabilirsiniz.

```csharp
public static void CreateAndSaveToHTMLUsingDefaultOptions()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateOneNoteDocAndSaveToHTML_out.html");
    doc.Save(outputPath);

    Console.WriteLine("\nOneNote document created successfully.\nFile saved at " + outputPath);
}
```

### Adım 3: Gömülü Kaynaklarla Bellek Akışına HTML Kaydetme

Tek bir bütünleşik HTML yükü (ör. API yanıtları) istediğinizde, CSS, font ve görselleri doğrudan bir `MemoryStream` içine gömebilirsiniz. `HtmlSaveOptions` ile `EmbedResources` özelliğini `true` yaparak kompakt bir HTML dizesi elde edersiniz.

```csharp
public static void CreateAndSavePageRange()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateAndSavePageRange_out.html");
    doc.Save(outputPath, new HtmlSaveOptions { PageCount = 1, PageIndex = 0 });

    Console.WriteLine("\nOneNote document created successfully and saved as page range.\nFile saved at " + outputPath);
}
```

### Adım 4: Kaynakları Ayrı Dosyalarda Tutacak Şekilde HTML Kaydetme

Klasik web‑klasör çıktısını—HTML dosyası ve bir varlık klasörü—tercih ediyorsanız gömmeyi devre dışı bırakın ve bir çıktı dizini belirtin. Bu yöntem, kaynak dosyalarının daha temiz olmasını ve görsellerin ya da stil sayfalarının daha kolay önbelleğe alınmasını sağlar.

```csharp
public static void SaveAsHTMLToMemoryStreamWithEmbeddedResources()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportEmbedded,
        ExportFonts = ResourceExportType.ExportEmbedded,
        ExportImages = ResourceExportType.ExportEmbedded,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to a memory stream
    var memoryStream = new MemoryStream();
    document.Save(memoryStream, options);
}
```

### Adım 5: Kaynakları Yönetmek İçin Geri Çağrılar Kullanarak HTML Kaydetme

Görselleri bir veritabanına ya da bulut deposuna kaydetmek gibi gelişmiş senaryolar, özel işleme gerektirir. Bir `ResourceSavingCallback` temsilcisi sağlayarak her kaynağın nasıl saklanacağını tam kontrol edebilir, API'nin doğru HTML referansları üretmesini sağlayabilirsiniz.

```csharp
public static void SaveAsHTMLToFileWithResourcesInSeparateFiles()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportAsStream,
        ExportFonts = ResourceExportType.ExportAsStream,
        ExportImages = ResourceExportType.ExportAsStream,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to HTML file with resources stored in separate files
    document.Save(Path.Combine(dataDir, "document_out.html"), options);
}
```

## Yaygın Sorunlar ve Çözümler

| Sorun | Neden Oluşur | Çözüm |
|-------|--------------|------|
| **Görseller bozuk görünüyor** | Kaynaklar HTML dosyasından farklı bir klasöre kaydedildi. | `HtmlSaveOptions.ResourcesFolder` değerinin referans verdiğiniz klasörle eşleştiğinden emin olun veya `EmbedResources` özelliğini etkinleştirin. |
| **CSS stilleri eksik** | Varsayılan CSS üretimi devre dışı bırakıldı. | `HtmlSaveOptions.GenerateCss = true` (varsayılan) ayarlayın veya CSS'i manuel olarak gömün. |
| **Büyük not defterlerinde OutOfMemory** | Akış‑dışı `Document` yapıcı kullanılarak tüm dosya belleğe yüklendi. | `Document.Load(stream, LoadOptions)` ve `LoadOptions.LoadFromMemory = true` kullanarak sayfaları akış olarak yükleyin. |
| **HTML kodlama sorunları (ASCII dışı karakterler)** | `HtmlSaveOptions` üzerinde yanlış `Encoding` ayarlandı. | `HtmlSaveOptions.Encoding = Encoding.UTF8` olarak ayarlayın. |
| **Linux'ta kaydetme başarısız** | Görsel işleme için yerel bağımlılıklar eksik. | Linux hostunda `libgdiplus` paketini kurun. |

public static void SaveAsHTMLToMemoryStreamWithCallBacksToSaveResources()
{
    // Specify the saving callbacks configuration
    var savingCallbacks = new UserSavingCallbacks()
    {
        RootFolder = "documentFolder",
        CssFolder = "css",
        KeepCssStreamOpened = true,
        ImagesFolder = "images",
        FontsFolder = "fonts"
    };

    // Specify HTML saving options
    var options = new HtmlSaveOptions
    {
        FontFaceTypes = FontFaceType.Ttf,
        CssSavingCallback = savingCallbacks,
        FontSavingCallback = savingCallbacks,
        ImageSavingCallback = savingCallbacks
    };

    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Save the document to HTML format with resources managed by user-defined callbacks
    using (var stream = File.Create(Path.Combine(savingCallbacks.RootFolder, "document.html")))
    {
        document.Save(stream, options);
    }

    // Manually append data to the CSS stream
    using (var writer = new StreamWriter(savingCallbacks.CssStream))
    {
        writer.WriteLine();
        writer.WriteLine("/* This line is appended to stream manually by user */");
    }
}
