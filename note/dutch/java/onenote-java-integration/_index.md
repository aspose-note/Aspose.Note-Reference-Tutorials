## Veelgestelde Vragen

**Q: Kan ik Aspose.Note for Java gebruiken in een commercieel project?**  
A: Ja, een commerciële licentie is vereist voor productiegebruik, maar een gratis proefversie is beschikbaar voor evaluatie.

**Q: Welke Java‑versies worden ondersteund?**  
A: De bibliotheek ondersteunt Java 8 en latere versies.

**Q: Hoe ga ik om met grote OneNote‑bestanden bij het ophalen van bijlagen?**  
A: Laad het document in een streaming‑modus of verwerk secties/pagina's afzonderlijk om het geheugenverbruik te verminderen.

**Q: Is het mogelijk om een aangepast pictogram in te stellen voor een bijgevoegd bestand?**  
A: Absoluut – de tutorial “Attach File and Set Icon” laat zien hoe je een pictogram programmatisch kunt specificeren.

**Q: Moet ik OneNote op de server installeren om de API te gebruiken?**  
A: Nee, Aspose.Note werkt onafhankelijk van de OneNote‑applicatie; het leest en schrijft OneNote‑bestandsformaten direct.
