# OneNote opslaan als HTML met Aspose.Note

## Introductie

Als u **OneNote als HTML wilt opslaan** voor webpublicatie, rapportage of archivering, biedt Aspose.Note voor .NET een schone, alleen‑code oplossing die werkt op Windows, Linux en macOS zonder dat Microsoft Office vereist is. In deze tutorial leert u hoe u een OneNote‑document vanaf nul maakt, de pagina’s bewerkt en het volledige notitieboek – of geselecteerde pagina’s – exporteert naar HTML met verschillende opties. De gids leidt u stap voor stap, legt uit waarom elke instelling belangrijk is en toont praktijkvoorbeelden waarin HTML‑export uitblinkt, zoals het insluiten van OneNote‑inhoud in een kennis‑base portal of het genereren van statische documentatie.

## Snelle antwoorden
- **Wat is de primaire methode om te exporteren?** Roep `document.Save("output.html", SaveFormat.Html)` aan op een `Aspose.Note.Document`‑instantie.  
- **Kan ik een enkele pagina exporteren?** Ja – gebruik `document.Save(pageRange, "page.html", SaveFormat.Html)`.  
- **Worden bronnen (afbeeldingen, CSS) automatisch ingesloten?** Standaard worden ze opgeslagen als afzonderlijke bestanden; u kunt ze in een geheugen‑stream insluiten met `HtmlSaveOptions`.  
- **Wordt .NET Core ondersteund?** Absoluut – Aspose.Note werkt met .NET 5, .NET 6 en .NET Core 3.1+.  
- **Heb ik een licentie nodig voor ontwikkeling?** Een gratis proefversie van 30 dagen werkt voor testen; een licentie verwijdert evaluatiewatermerken.

## Wat betekent “save onenote as html”?

**“Save OneNote as HTML”** verwijst naar het proces waarbij een OneNote‑notitieboek, dat is opgeslagen in een propriëtair binair formaat, wordt omgezet naar standaard‑HTML‑bestanden die door elke webbrowser kunnen worden weergegeven. De conversie behoudt de hiërarchische structuur van het notitieboek, tekstopmaak, afbeeldingen, tabellen en andere bronnen, terwijl OneNote‑specifieke elementen worden vertaald naar equivalente HTML‑tags en CSS‑stijlen. Dit stelt ontwikkelaars in staat OneNote‑inhoud op websites te tonen, te integreren in content‑management‑systemen, of het te archiveren in een universeel toegankelijk formaat zonder de OneNote‑client te vereisen.

## Waarom OneNote naar HTML converteren?

Aspose.Note kan exporteren naar **meer dan 50 outputformaten** (inclusief PDF, DOCX, PNG en HTML) en verwerkt notitieboeken met **tot 500 pagina’s** zonder het volledige bestand in het geheugen te laden, dankzij de streaming‑architectuur. Converteren naar HTML stelt u in staat om:

1. **Inhoud op het web publiceren** zonder de OneNote‑client te vereisen.  
2. **Integreren met content‑management‑systemen** die alleen HTML accepteren.  
3. **Zoekbare tekst extraheren** voor indexeringsmachines.  
4. **Pagina‑specifieke bronnen** (afbeeldingen, lettertypen) hergebruiken in andere webprojecten.

## Vereisten

- Basisbegrip van de hiërarchische structuur van OneNote (Notebook → Section → Page).

## Hoe OneNote opslaan als HTML – stap voor stap

OneNote opslaan als HTML omvat drie kernacties: een notitieboek laden of maken, eventuele gewenste exportopties configureren, en de juiste `Save`‑methode aanroepen. Eerst maakt u of laadt u een `Document`‑object, vervolgens kunt u `HtmlSaveOptions` aanpassen om de bronafhandeling, CSS‑generatie en codering te regelen. Ten slotte roept u `document.Save` aan met een bestandspad of een stream, eventueel met een `PageRange` om de export te beperken. Deze systematische aanpak zorgt voor consistente resultaten in verschillende omgevingen en stelt u in staat de output af te stemmen op de behoeften van uw project.

### Stap 1: Een OneNote‑document maken met standaardopties

In dit eerste voorbeeld instantieren we een nieuw `Document`, voegen een enkele pagina met een titel toe, en slaan het volledige notitieboek op als HTML met de standaard `HtmlSaveOptions`. Er zijn geen extra parameters nodig, waardoor dit de snelste manier is om **OneNote als HTML op te slaan**.

```csharp
using System;
using System.Drawing;
using System.Globalization;
using System.IO;

using Aspose.Note.Saving;
using Aspose.Note.Saving.Html;

```

### Stap 2: Een specifiek paginabereik exporteren naar HTML

Vaak heeft u slechts een deel van de pagina’s nodig – bijvoorbeeld het exporteren van een hoofdstuk met vergadernotities. Door een `PageRange`‑object aan `Save` door te geven, kunt u pagina’s 1‑3 targeten terwijl de rest onaangeroerd blijft.

```csharp
public static void CreateAndSaveToHTMLUsingDefaultOptions()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateOneNoteDocAndSaveToHTML_out.html");
    doc.Save(outputPath);

    Console.WriteLine("\nOneNote document created successfully.\nFile saved at " + outputPath);
}
```

### Stap 3: Opslaan als HTML naar een geheugen‑stream met ingesloten bronnen

Wanneer u een enkele zelf‑behorende HTML‑payload wilt (bijv. voor API‑reacties), kunt u CSS, lettertypen en afbeeldingen direct in een `MemoryStream` insluiten. `HtmlSaveOptions` laat u `EmbedResources` op `true` zetten, waardoor een compacte HTML‑string ontstaat.

```csharp
public static void CreateAndSavePageRange()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateAndSavePageRange_out.html");
    doc.Save(outputPath, new HtmlSaveOptions { PageCount = 1, PageIndex = 0 });

    Console.WriteLine("\nOneNote document created successfully and saved as page range.\nFile saved at " + outputPath);
}
```

### Stap 4: Opslaan als HTML naar bestanden met bronnen in afzonderlijke bestanden

Als u de klassieke web‑mapoutput verkiest – HTML‑bestand plus een map met assets – schakelt u insluiten uit en specificeert u een uitvoermap. Deze aanpak levert schonere bronbestanden op en maakt caching van afbeeldingen of stylesheets eenvoudiger.

```csharp
public static void SaveAsHTMLToMemoryStreamWithEmbeddedResources()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportEmbedded,
        ExportFonts = ResourceExportType.ExportEmbedded,
        ExportImages = ResourceExportType.ExportEmbedded,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to a memory stream
    var memoryStream = new MemoryStream();
    document.Save(memoryStream, options);
}
```

### Stap 5: Opslaan als HTML met callbacks om bronnen te beheren

Geavanceerde scenario’s, zoals het opslaan van afbeeldingen in een database of een cloud‑bucket, vereisen aangepaste afhandeling. Door een `ResourceSavingCallback`‑delegate te leveren, bepaalt u precies hoe elke bron wordt bewaard, terwijl de API correcte HTML‑referenties blijft genereren.

```csharp
public static void SaveAsHTMLToFileWithResourcesInSeparateFiles()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportAsStream,
        ExportFonts = ResourceExportType.ExportAsStream,
        ExportImages = ResourceExportType.ExportAsStream,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to HTML file with resources stored in separate files
    document.Save(Path.Combine(dataDir, "document_out.html"), options);
}
```

## Veelvoorkomende problemen en oplossingen

| Probleem | Waarom het gebeurt | Oplossing |
|----------|--------------------|-----------|
| **Afbeeldingen verschijnen kapot** | Bronnen werden opgeslagen in een andere map dan het HTML‑bestand. | Zorg dat `HtmlSaveOptions.ResourcesFolder` overeenkomt met de map die u referentieert, of schakel `EmbedResources` in. |
| **CSS‑stijlen ontbreken** | Standaard CSS‑generatie was uitgeschakeld. | Stel `HtmlSaveOptions.GenerateCss = true` (standaard) in of voeg CSS handmatig in. |
| **Grote notitieboeken veroorzaken OutOfMemory** | De niet‑streaming `Document`‑constructor laadt het hele bestand. | Gebruik `Document.Load(stream, LoadOptions)` met `LoadOptions.LoadFromMemory = true` om pagina’s te streamen. |
| **HTML‑coderingproblemen voor niet‑ASCII tekens** | Verkeerde `Encoding` ingesteld op `HtmlSaveOptions`. | Stel `HtmlSaveOptions.Encoding = Encoding.UTF8` in. |
| **Opslaan mislukt op Linux** | Ontbrekende native afhankelijkheden voor beeldverwerking. | Installeer `libgdiplus` op de Linux‑host. |

```csharp
public static void SaveAsHTMLToMemoryStreamWithCallBacksToSaveResources()
{
    // Specify the saving callbacks configuration
    var savingCallbacks = new UserSavingCallbacks()
    {
        RootFolder = "documentFolder",
        CssFolder = "css",
        KeepCssStreamOpened = true,
        ImagesFolder = "images",
        FontsFolder = "fonts"
    };

    // Specify HTML saving options
    var options = new HtmlSaveOptions
    {
        FontFaceTypes = FontFaceType.Ttf,
        CssSavingCallback = savingCallbacks,
        FontSavingCallback = savingCallbacks,
        ImageSavingCallback = savingCallbacks
    };

    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Save the document to HTML format with resources managed by user-defined callbacks
    using (var stream = File.Create(Path.Combine(savingCallbacks.RootFolder, "document.html")))
    {
        document.Save(stream, options);
    }

    // Manually append data to the CSS stream
    using (var writer = new StreamWriter(savingCallbacks.CssStream))
    {
        writer.WriteLine();
        writer.WriteLine("/* This line is appended to stream manually by user */");
    }
}
```
