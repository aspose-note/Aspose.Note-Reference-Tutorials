# Salvar OneNote como HTML usando Aspose.Note

## Introdução

Se você precisa **salvar OneNote como HTML** para publicação na web, relatórios ou arquivamento, o Aspose.Note para .NET oferece uma solução limpa, apenas código, que funciona no Windows, Linux e macOS sem exigir o Microsoft Office. Neste tutorial você aprenderá como criar um documento OneNote do zero, manipular suas páginas e exportar todo o bloco de notas — ou páginas selecionadas — para HTML usando diversas opções. O guia acompanha você em cada passo, explica por que cada configuração é importante e mostra cenários reais onde a exportação para HTML se destaca, como incorporar conteúdo OneNote em um portal de base de conhecimento ou gerar documentação estática.

## Respostas Rápidas
- **Qual é o método principal para exportar?** Chame `document.Save("output.html", SaveFormat.Html)` em uma instância `Aspose.Note.Document`.  
- **Posso exportar uma única página?** Sim—use `document.Save(pageRange, "page.html", SaveFormat.Html)`.  
- **Os recursos (imagens, CSS) são incorporados automaticamente?** Por padrão eles são salvos como arquivos separados; você pode incorporá‑los em um stream de memória com `HtmlSaveOptions`.  
- **O .NET Core é suportado?** Absolutamente—Aspose.Note funciona com .NET 5, .NET 6 e .NET Core 3.1+.  
- **Preciso de uma licença para desenvolvimento?** Um teste gratuito de 30 dias funciona para testes; uma licença remove as marcas d'água de avaliação.

## O que é “salvar onenote como html”?

**“Salvar OneNote como HTML”** refere‑se ao processo de converter um bloco de notas OneNote, que está armazenado em um formato binário proprietário, em arquivos HTML padrão que podem ser renderizados por qualquer navegador web. A conversão preserva a estrutura hierárquica do bloco de notas, formatação de texto, imagens, tabelas e outros recursos, enquanto traduz os elementos específicos do OneNote em tags HTML equivalentes e estilos CSS. Isso permite que desenvolvedores exibam conteúdo OneNote em sites, integrem‑no a sistemas de gerenciamento de conteúdo ou arquivem‑o em um formato universalmente acessível sem exigir o cliente OneNote.

## Por que converter OneNote para HTML?

Aspose.Note pode exportar para **mais de 50 formatos de saída** (incluindo PDF, DOCX, PNG e HTML) e processa blocos de notas com **até 500 páginas** sem carregar o arquivo inteiro na memória, graças à sua arquitetura de streaming. Converter para HTML permite que você:

1. **Publique conteúdo na web** sem exigir o cliente OneNote.  
2. **Integre com sistemas de gerenciamento de conteúdo** que aceitam apenas HTML.  
3. **Extraia texto pesquisável** para mecanismos de indexação.  
4. **Reutilize recursos em nível de página** (imagens, fontes) em outros projetos web.

## Pré‑requisitos

- Familiaridade com C# e Visual Studio (qualquer edição).  
- .NET Framework 4.6+ **ou** .NET Core 3.1+ instalado.  
- Aspose.Note para .NET adicionado ao seu projeto. Baixe‑o de [aqui](https://releases.aspose.com/note/net/).  
- Compreensão básica da estrutura hierárquica do OneNote (Notebook → Section → Page).

## Como salvar onenote como html – Passo a Passo

Salvar OneNote como HTML envolve três ações principais: carregar ou criar um bloco de notas, configurar as opções de exportação desejadas e invocar o método `Save` apropriado. Primeiro, você cria ou carrega um objeto `Document`, depois pode ajustar `HtmlSaveOptions` para controlar o tratamento de recursos, geração de CSS e codificação. Finalmente, você chama `document.Save` com um caminho de arquivo ou um stream, opcionalmente especificando um `PageRange` para limitar a exportação. Essa abordagem sistemática garante resultados consistentes em diferentes ambientes e permite que você ajuste a saída às necessidades do seu projeto.

### Passo 1: Criar um Documento OneNote com Opções Padrão

Neste primeiro exemplo, instanciamos um novo `Document`, adicionamos uma única página com um título e salvamos todo o bloco de notas em HTML usando as `HtmlSaveOptions` padrão. Nenhum parâmetro extra é necessário, tornando‑a a maneira mais rápida de **salvar OneNote como HTML**.

```csharp
using System;
using System.Drawing;
using System.Globalization;
using System.IO;

using Aspose.Note.Saving;
using Aspose.Note.Saving.Html;

```

### Passo 2: Exportar um Intervalo de Páginas Específico para HTML

Frequentemente você precisa apenas de um subconjunto de páginas — por exemplo, exportar um capítulo de notas de reunião. Ao passar um objeto `PageRange` para `Save`, você pode direcionar as páginas 1‑3 enquanto deixa o restante intacto.

```csharp
public static void CreateAndSaveToHTMLUsingDefaultOptions()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateOneNoteDocAndSaveToHTML_out.html");
    doc.Save(outputPath);

    Console.WriteLine("\nOneNote document created successfully.\nFile saved at " + outputPath);
}
```

### Passo 3: Salvar como HTML em um Memory Stream com Recursos Incorporados

Quando você deseja um único payload HTML autocontido (por exemplo, para respostas de API), incorpore CSS, fontes e imagens diretamente em um `MemoryStream`. `HtmlSaveOptions` permite alternar `EmbedResources` para `true`, produzindo uma string HTML compacta.

```csharp
public static void CreateAndSavePageRange()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateAndSavePageRange_out.html");
    doc.Save(outputPath, new HtmlSaveOptions { PageCount = 1, PageIndex = 0 });

    Console.WriteLine("\nOneNote document created successfully and saved as page range.\nFile saved at " + outputPath);
}
```

### Passo 4: Salvar como HTML em Arquivos com Recursos em Arquivos Separados

Se você prefere a saída clássica de pasta web — arquivo HTML mais uma pasta de ativos — desative a incorporação e especifique um diretório de saída. Essa abordagem gera arquivos‑fonte mais limpos e facilita o cache de imagens ou folhas de estilo.

```csharp
public static void SaveAsHTMLToMemoryStreamWithEmbeddedResources()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportEmbedded,
        ExportFonts = ResourceExportType.ExportEmbedded,
        ExportImages = ResourceExportType.ExportEmbedded,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to a memory stream
    var memoryStream = new MemoryStream();
    document.Save(memoryStream, options);
}
```

### Passo 5: Salvar como HTML Usando Callbacks para Gerenciar Recursos

Cenários avançados, como armazenar imagens em um banco de dados ou em um bucket na nuvem, exigem tratamento personalizado. Ao fornecer um delegate `ResourceSavingCallback`, você controla exatamente como cada recurso é persistido, enquanto a API continua a gerar referências HTML corretas.

```csharp
public static void SaveAsHTMLToFileWithResourcesInSeparateFiles()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportAsStream,
        ExportFonts = ResourceExportType.ExportAsStream,
        ExportImages = ResourceExportType.ExportAsStream,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to HTML file with resources stored in separate files
    document.Save(Path.Combine(dataDir, "document_out.html"), options);
}
```

## Problemas Comuns e Soluções

| Problema | Por que acontece | Correção |
|----------|------------------|----------|
| **Imagens aparecem quebradas** | Os recursos foram salvos em uma pasta diferente da do arquivo HTML. | Certifique‑se de que `HtmlSaveOptions.ResourcesFolder` corresponda à pasta que você referencia, ou habilite `EmbedResources`. |
| **Estilos CSS ausentes** | A geração padrão de CSS foi desativada. | Defina `HtmlSaveOptions.GenerateCss = true` (padrão) ou incorpore CSS manualmente. |
| **Blocos de notas grandes causam OutOfMemory** | Usando o construtor `Document` não‑streaming carrega todo o arquivo. | Use `Document.Load(stream, LoadOptions)` com `LoadOptions.LoadFromMemory = true` para transmitir páginas. |
| **Problemas de codificação HTML para caracteres não‑ASCII** | Codificação errada definida em `HtmlSaveOptions`. | Defina `HtmlSaveOptions.Encoding = Encoding.UTF8`. |
| **Falha ao salvar no Linux** | Dependências nativas ausentes para processamento de imagens. | Instale `libgdiplus` no host Linux. |

```csharp
public static void SaveAsHTMLToMemoryStreamWithCallBacksToSaveResources()
{
    // Specify the saving callbacks configuration
    var savingCallbacks = new UserSavingCallbacks()
    {
        RootFolder = "documentFolder",
        CssFolder = "css",
        KeepCssStreamOpened = true,
        ImagesFolder = "images",
        FontsFolder = "fonts"
    };

    // Specify HTML saving options
    var options = new HtmlSaveOptions
    {
        FontFaceTypes = FontFaceType.Ttf,
        CssSavingCallback = savingCallbacks,
        FontSavingCallback = savingCallbacks,
        ImageSavingCallback = savingCallbacks
    };

    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Save the document to HTML format with resources managed by user-defined callbacks
    using (var stream = File.Create(Path.Combine(savingCallbacks.RootFolder, "document.html")))
    {
        document.Save(stream, options);
    }

    // Manually append data to the CSS stream
    using (var writer = new StreamWriter(savingCallbacks.CssStream))
    {
        writer.WriteLine();
        writer.WriteLine("/* This line is appended to stream manually by user */");
    }
}
```
