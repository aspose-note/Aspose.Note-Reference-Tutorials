## Perguntas Frequentes

**Q: Posso exportar um caderno OneNote diretamente para PDF sem formatos intermediários?**  
A: Sim. Aspose.Note fornece um método `save` de um passo que grava o caderno em um arquivo PDF, preservando o layout e as imagens.

**Q: Como extraio apenas as imagens de uma página OneNote?**  
A: Use a API `Document.getImages()` para enumerar os objetos de imagem e salvar cada um no formato de imagem desejado (PNG, JPEG, etc.).

**Q: Existe uma maneira de melhorar a velocidade de exportação para cadernos grandes?**  
A: Ative o método `Document.optimizeResources()` antes de salvar e considere processar seções em threads paralelas.

**Q: Preciso instalar algum software adicional no servidor?**  
A: Não. Aspose.Note é uma biblioteca Java pura; basta incluir os JARs no classpath do seu projeto.
