## Perguntas Frequentes

**P: Posso adicionar um hyperlink a qualquer formato de imagem (PNG, JPEG, GIF)?**  
R: Sim. Aspose.Note suporta todos os formatos de imagem padrão; o hyperlink é anexado ao objeto de imagem independentemente do tipo.

**P: Preciso abrir o arquivo OneNote no modo de edição para adicionar o hyperlink?**  
R: Não. Você pode criar ou modificar um documento totalmente na memória e então salvá-lo em um arquivo `.one`.

**P: O hyperlink funcionará nos aplicativos móveis do OneNote?**  
R: Absolutamente. O hyperlink gerado é armazenado no formato de arquivo do OneNote, que é reconhecido pelos clientes desktop, web e móveis.

**P: Como posso verificar se o hyperlink foi adicionado corretamente?**  
R: Após salvar, abra o arquivo no OneNote e clique na imagem; a URL vinculada deve abrir no navegador padrão.

**P: Existe uma maneira de adicionar múltiplos hyperlinks a uma única imagem?**  
R: Uma imagem pode conter apenas um hyperlink. Para fornecer múltiplos links, considere usar uma imagem composta com regiões clicáveis separadas ou adicionar objetos de imagem separados.
