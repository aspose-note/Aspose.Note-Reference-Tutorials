## O que é criar tabela OneNote programaticamente?
`Programmatically create OneNote table` refere‑se ao uso de código — especificamente a API Aspose.Note for Java — para gerar um objeto Table dentro de uma página OneNote sem interação manual do usuário. Essa abordagem automatiza a geração de documentos, garante consistência e escala em grandes cargas de trabalho. Permite que desenvolvedores criem tabelas diretamente a partir de aplicações Java, economizando tempo e reduzindo erros.

## Como criar tabela OneNote programaticamente?

`Document` representa um arquivo de bloco de notas OneNote carregado na memória.  
`Page.getTables().add()` cria uma nova Table na página com o número especificado de linhas e colunas.

Carregue uma instância de `Document`, adicione uma `Page` e então chame `Page.getTables().add()` com a quantidade desejada de linhas e colunas. Após a criação, você pode definir valores das células, aplicar bordas e estilizar a tabela — tudo através de chamadas fluentes em Java. Esse padrão de duas etapas (criar → configurar) permite construir tabelas instantaneamente, mesmo para blocos de notas com várias páginas.
