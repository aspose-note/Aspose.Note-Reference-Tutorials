# OneNote mentése HTML-ként az Aspose.Note segítségével

## Bevezetés

Ha **save OneNote as HTML**‑t szeretne webes közzétételhez, jelentéskészítéshez vagy archiváláshoz, az Aspose.Note for .NET tiszta, csak kódból álló megoldást nyújt, amely Windows, Linux és macOS rendszereken működik, anélkül, hogy a Microsoft Office-ra szükség lenne. Ebben az útmutatóban megtanulja, hogyan hozhat létre egy OneNote dokumentumot a semmiből, hogyan kezelheti az oldalait, és hogyan exportálhatja az egész jegyzetfüzetet – vagy a kiválasztott oldalakat – HTML-be különféle beállítások használatával. Az útmutató minden lépésen végigvezet, elmagyarázza, miért fontos minden beállítás, és valós példákat mutat, ahol a HTML export kiemelkedik, például OneNote tartalom beágyazása egy tudásbázis portálba vagy statikus dokumentáció generálása.

## Gyors válaszok
- **Mi a fő módszer az exportáláshoz?** Hívja meg a `document.Save("output.html", SaveFormat.Html)` metódust egy `Aspose.Note.Document` példányon.  
- **Exportálhatok egyetlen oldalt?** Igen—használja a `document.Save(pageRange, "page.html", SaveFormat.Html)`-t.  
- **A források (képek, CSS) automatikusan beágyazódnak?** Alapértelmezés szerint külön fájlokként mentődnek; beágyazhatja őket egy memóriafolyamba a `HtmlSaveOptions` segítségével.  
- **.NET Core támogatott?** Teljesen – az Aspose.Note működik .NET 5, .NET 6 és .NET Core 3.1+ verziókkal.  
- **Fejlesztéshez szükség van licencre?** Egy ingyenes 30‑napos próba a teszteléshez elegendő; egy licenc eltávolítja a kiértékelési vízjeleket.

## Mi az a „save onenote as html”?

**“Save OneNote as HTML”** a folyamatra utal, amely során egy OneNote jegyzetfüzetet, amely egy saját tulajdonú bináris formátumban van tárolva, szabványos HTML fájlokká alakítanak, amelyeket bármely webböngésző megjeleníthet. Az átalakítás megőrzi a jegyzetfüzet hierarchikus felépítését, a szövegformázást, képeket, táblázatokat és egyéb erőforrásokat, miközben a OneNote‑specifikus elemeket ekvivalens HTML címkékké és CSS stílusokká konvertálja. Ez lehetővé teszi a fejlesztők számára, hogy OneNote tartalmat jelenítsenek meg weboldalakon, integrálják tartalomkezelő rendszerekbe, vagy univerzálisan hozzáférhető formátumban archiválják anélkül, hogy a OneNote kliensre szükség lenne.

## Miért konvertáljuk a OneNote-ot HTML-re?

Az Aspose.Note **50+ kimeneti formátumba** tud exportálni (beleértve a PDF, DOCX, PNG és HTML formátumokat is), és akár **500 oldalig** képes feldolgozni a jegyzetfüzeteket anélkül, hogy az egész fájlt a memóriába töltené, köszönhetően a streaming architektúrájának. A HTML-re konvertálás lehetővé teszi, hogy:

1. **Tartalmat tegyen közzé a weben** a OneNote kliens nélkül.  
2. **Integrálja tartalomkezelő rendszerekkel**, amelyek csak HTML-t fogadnak el.  
3. **Kinyerje a kereshető szöveget** indexelő motorok számára.  
4. **Újra felhasználja az oldal szintű erőforrásokat** (képek, betűkészletek) más webprojektekben.

## Hogyan mentse a onenote-ot html‑ként – Lépésről lépésre

A OneNote HTML‑ként történő mentése három fő lépést foglal magában: egy jegyzetfüzet betöltése vagy létrehozása, a kívánt exportálási beállítások konfigurálása, és a megfelelő `Save` metódus meghívása. Először létrehozza vagy betölti a `Document` objektumot, majd módosíthatja a `HtmlSaveOptions`‑t az erőforráskezelés, a CSS generálás és a kódolás szabályozásához. Végül meghívja a `document.Save`‑t fájlúttal vagy stream‑mel, opcionálisan megadva egy `PageRange`‑t az export korlátozásához. Ez a rendszerszerű megközelítés biztosítja a konzisztens eredményeket különböző környezetekben, és lehetővé teszi a kimenet testreszabását a projekt igényei szerint.

### 1. lépés: OneNote dokumentum létrehozása alapértelmezett beállításokkal

Ebben az első példában egy új `Document` példányt hozunk létre, hozzáadunk egyetlen oldalt címmel, és a teljes jegyzetfüzetet HTML‑ként mentjük az alapértelmezett `HtmlSaveOptions` használatával. Nincs szükség extra paraméterekre, így ez a leggyorsabb módja a **save OneNote as HTML**‑nek.

```csharp
using System;
using System.Drawing;
using System.Globalization;
using System.IO;

using Aspose.Note.Saving;
using Aspose.Note.Saving.Html;

```

### 2. lépés: Egy adott oldaltartomány exportálása HTML‑be

Gyakran csak egy oldalcsoport szükséges – például egy megbeszélés jegyzeteinek egy fejezetének exportálása. A `PageRange` objektum `Save`‑nek átadásával a 1‑3. oldalakat célozhatja meg, miközben a többit érintetlenül hagyja.

```csharp
public static void CreateAndSaveToHTMLUsingDefaultOptions()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateOneNoteDocAndSaveToHTML_out.html");
    doc.Save(outputPath);

    Console.WriteLine("\nOneNote document created successfully.\nFile saved at " + outputPath);
}
```

### 3. lépés: HTML mentése memóriafolyamra beágyazott erőforrásokkal

Ha egy önálló HTML payload‑ra van szükség (pl. API válaszokhoz), beágyazhatja a CSS‑t, betűkészleteket és képeket közvetlenül egy `MemoryStream`‑be. A `HtmlSaveOptions` lehetővé teszi az `EmbedResources` `true`‑ra állítását, így egy kompakt HTML karakterláncot kap.

```csharp
public static void CreateAndSavePageRange()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateAndSavePageRange_out.html");
    doc.Save(outputPath, new HtmlSaveOptions { PageCount = 1, PageIndex = 0 });

    Console.WriteLine("\nOneNote document created successfully and saved as page range.\nFile saved at " + outputPath);
}
```

### 4. lépés: HTML mentése fájlokba, az erőforrások külön fájlokban

Ha a klasszikus web‑mappa kimenetet részesíti előnyben – HTML fájl plusz egy eszközöket tartalmazó mappa –, tiltsa le a beágyazást, és adja meg a kimeneti könyvtárat. Ez a megközelítés tisztább forrásfájlokat és egyszerűbb képek vagy stíluslapok gyorsítótárazását eredményezi.

```csharp
public static void SaveAsHTMLToMemoryStreamWithEmbeddedResources()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportEmbedded,
        ExportFonts = ResourceExportType.ExportEmbedded,
        ExportImages = ResourceExportType.ExportEmbedded,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to a memory stream
    var memoryStream = new MemoryStream();
    document.Save(memoryStream, options);
}
```

### 5. lépés: HTML mentése visszahívásokkal az erőforrások kezelése érdekében

Haladó esetekben, például képek adatbázisba vagy felhő tárolóba mentésekor, egyedi kezelést igényelnek. Egy `ResourceSavingCallback` delegált biztosításával pontosan szabályozhatja, hogyan kerülnek az egyes erőforrások tárolásra, miközben az API továbbra is helyes HTML hivatkozásokat generál.

```csharp
public static void SaveAsHTMLToFileWithResourcesInSeparateFiles()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportAsStream,
        ExportFonts = ResourceExportType.ExportAsStream,
        ExportImages = ResourceExportType.ExportAsStream,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to HTML file with resources stored in separate files
    document.Save(Path.Combine(dataDir, "document_out.html"), options);
}
```

## Gyakori problémák és megoldások

| Issue | Why It Happens | Fix |
|-------|----------------|-----|
| **A képek hibásan jelennek meg** | Az erőforrások a HTML fájltól eltérő mappába lettek mentve. | Győződjön meg arról, hogy a `HtmlSaveOptions.ResourcesFolder` megegyezik a hivatkozott mappával, vagy engedélyezze az `EmbedResources`‑t. |
| **A CSS stílusok hiányoznak** | Az alapértelmezett CSS generálás le lett tiltva. | Állítsa be a `HtmlSaveOptions.GenerateCss = true`‑t (alapértelmezett) vagy ágyazza be manuálisan a CSS‑t. |
| **Nagy jegyzetfüzetek OutOfMemory hibát okoznak** | A nem streaming `Document` konstruktor használata betölti az egész fájlt. | Használja a `Document.Load(stream, LoadOptions)`‑t a `LoadOptions.LoadFromMemory = true` beállítással az oldalak streamingjéhez. |
| **HTML kódolási problémák nem ASCII karakterekhez** | Helytelen `Encoding` van beállítva a `HtmlSaveOptions`‑ban. | Állítsa be a `HtmlSaveOptions.Encoding = Encoding.UTF8`‑t. |
| **Mentés Linuxon sikertelen** | Hiányoznak a natív függőségek a képfeldolgozáshoz. | Telepítse a `libgdiplus`‑t a Linux gépen. |

```csharp
public static void SaveAsHTMLToMemoryStreamWithCallBacksToSaveResources()
{
    // Specify the saving callbacks configuration
    var savingCallbacks = new UserSavingCallbacks()
    {
        RootFolder = "documentFolder",
        CssFolder = "css",
        KeepCssStreamOpened = true,
        ImagesFolder = "images",
        FontsFolder = "fonts"
    };

    // Specify HTML saving options
    var options = new HtmlSaveOptions
    {
        FontFaceTypes = FontFaceType.Ttf,
        CssSavingCallback = savingCallbacks,
        FontSavingCallback = savingCallbacks,
        ImageSavingCallback = savingCallbacks
    };

    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Save the document to HTML format with resources managed by user-defined callbacks
    using (var stream = File.Create(Path.Combine(savingCallbacks.RootFolder, "document.html")))
    {
        document.Save(stream, options);
    }

    // Manually append data to the CSS stream
    using (var writer = new StreamWriter(savingCallbacks.CssStream))
    {
        writer.WriteLine();
        writer.WriteLine("/* This line is appended to stream manually by user */");
    }
}
```
