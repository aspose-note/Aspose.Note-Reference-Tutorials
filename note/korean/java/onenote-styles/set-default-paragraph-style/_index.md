# OneNote 기본 글꼴 크기 설정 – 기본 단락 스타일 설정

## 소개

프로그래밍 방식으로 **default font size onenote**를 설정하면 각 페이지마다 일관된 모양을 유지할 수 있으며, 각 단락을 수동으로 편집할 필요가 없습니다. 이 튜토리얼에서는 Aspose.Note for Java를 사용하여 원하는 글꼴 크기, 글꼴 이름 및 기타 서식 옵션을 포함하는 기본 단락 스타일을 정의하는 방법을 배웁니다. 마지막에는 OneNote 파일을 다루는 모든 Java 프로젝트에 삽입할 수 있는 재사용 가능한 코드 조각을 얻게 됩니다.

## 빠른 답변
- **“default font size onenote”가 무엇을 제어하나요?** 새 단락마다 적용되는 기본 글꼴 크기를 정의하며, 별도로 지정하지 않는 한 적용됩니다.  
- **어떤 라이브러리가 이를 처리하나요?** Aspose.Note for Java는 단락 기본값을 설정하기 위한 유창한 API를 제공합니다.  
- **라이선스가 필요합니까?** 무료 체험판은 개발에 사용할 수 있으며, 프로덕션에서는 상용 라이선스가 필요합니다.  
- **다른 스타일 속성도 동시에 변경할 수 있나요?** 예 – 글꼴 이름, 색상, 정렬 및 줄 간격 모두 구성할 수 있습니다.  
- **이것이 모든 OneNote 버전과 호환되나요?** Aspose.Note는 2007년 이후의 OneNote 파일을 지원하며, 활성 노트북의 95 % 이상을 포괄합니다.

## default font size onenote란 무엇인가요?
**default font size onenote**는 명시적인 크기가 설정되지 않은 경우 OneNote 문서의 새 단락에 적용되는 기본 텍스트 크기입니다. 한 번 정의하면 반복적인 서식을 피하고 노트북 전체의 시각적 일관성을 보장합니다.

## OneNote에서 기본 단락 스타일을 설정하는 이유는?
Aspose.Note는 **50개 이상의 입력 및 출력 형식**을 지원하며 전체 파일을 메모리에 로드하지 않고도 **2 GB**까지의 콘텐츠가 포함된 노트북을 처리할 수 있습니다. 기본 단락 스타일을 설정하면 API 호출 수를 최대 **40 %**까지 줄이고, 문서 생성 속도를 높이며, 모든 단락이 기업 브랜드 가이드라인을 준수하도록 보장합니다.

## 사전 요구 사항

1. **Java Development Kit (JDK)** – JDK 11 이상을 권장합니다.  
2. **Aspose.Note for Java** – [다운로드 페이지](https://releases.aspose.com/note/java/)에서 다운로드하십시오.  
3. **An IDE** – Eclipse, IntelliJ IDEA, 또는 Java 지원이 포함된 VS Code와 같은 IDE를 사용합니다.

## OneNote 문서, 페이지 및 개요를 초기화하려면 어떻게 해야 하나요?

Document는 전체 OneNote 노트북을 나타내며 내용 조작을 위한 메서드를 제공합니다.  
Page는 노트북 내의 단일 페이지에 해당하며, 개요 및 기타 요소를 포함합니다.  
Outline은 페이지에서 관련된 리치 텍스트 요소를 그룹화하는 컨테이너입니다.

OneNote 파일을 나타내는 핵심 객체와 해당 파일 내의 페이지, 그리고 리치 텍스트를 포함하는 개요 컨테이너를 생성합니다. 이러한 객체는 추가 스타일링의 기반이 되며, 콘텐츠를 추가하기 전에 인스턴스화되어야 합니다.

```java
String dataDir = "Your Document Directory";
Document document = new Document();
Page page = new Page();
Outline outline = new Outline();
```

## 단락을 호스트할 개요 요소를 만들려면 어떻게 하나요?

OutlineElement는 Outline의 자식으로, 개별 단락을 나타내는 여러 RichText 객체를 포함할 수 있습니다.

개요 요소는 여러 단락 객체를 담는 컨테이너 역할을 하여 관련 텍스트 블록을 그룹화할 수 있게 합니다. 생성 후에는 페이지에 첨부하여 스타일이 적용된 텍스트가 노트북 내 원하는 위치에 표시되도록 합니다.

```java
OutlineElement outlineElem = new OutlineElement();
```

## 글꼴 크기를 포함한 기본 단락 스타일을 정의하려면 어떻게 하나요?

ParagraphStyle는 단락에 적용되는 글꼴 이름, 크기, 색상 및 정렬과 같은 서식 속성을 정의합니다.

ParagraphStyle 인스턴스를 정의하고, fontSize를 원하는 값(예: 12 pt)으로 설정하며, 필요에 따라 fontName, color 또는 alignment를 지정합니다. 이 스타일을 문서의 기본값으로 할당하면 새 단락이 추가 코드 없이 자동으로 이러한 설정을 상속합니다.

```java
ParagraphStyle defaultStyle = new ParagraphStyle()
										.setFontName("Courier New")
										.setFontSize(20);
```

## 기본 스타일을 자동으로 사용하는 리치 텍스트를 만들려면 어떻게 하나요?

RichText는 개별 서식이 적용된 여러 런을 포함할 수 있는 텍스트 블록을 나타냅니다.

명시적인 글꼴 크기를 지정하지 않고 RichText 객체를 인스턴스화하여 이전에 설정한 기본 스타일을 사용합니다. 객체는 정의된 글꼴 크기와 구성한 다른 스타일 속성을 사용해 렌더링되며, 모든 단락에서 일관된 외관을 보장합니다.

```java
RichText text = new RichText()
					.append("DefaultParagraphFontAndSize")
					.append(System.lineSeparator())
					.append("OnlyDefaultParagraphFont", new TextStyle().setFontSize(14))
					.append(System.lineSeparator())
					.append("OnlyDefaultParagraphFontSize", new TextStyle().setFontName("Verdana"));
text.setParagraphStyle(defaultStyle);
```

## 리치 텍스트를 개요 요소에 추가하려면 어떻게 하나요?

appendChildLast는 컨테이너 컬렉션의 끝에 자식 노드를 추가합니다.

appendChildLast 메서드를 사용하여 이전에 만든 개요 요소에 RichText 인스턴스를 추가합니다. 이 작업은 스타일이 적용된 콘텐츠를 개요에 연결하여 계층 구조를 유지하고 텍스트가 페이지에 올바른 순서대로 표시되도록 합니다.

```java
outlineElem.appendChildLast(text);
```

## 개요 요소를 페이지에 첨부하려면 어떻게 하나요?

Page.appendChildLast는 Outline과 같은 자식 요소를 페이지 컬렉션에 추가합니다.

appendChildLast 메서드를 사용하여 개요를 페이지의 개요 컬렉션에 추가합니다. 이 단계는 스타일이 적용된 콘텐츠를 페이지 구조에 통합하여 문서를 OneNote에서 열 때 표시되게 합니다.

```java
outline.appendChildLast(outlineElem);
```

## 페이지를 OneNote 문서에 추가하려면 어떻게 하나요?

Document.appendChildLast는 페이지 또는 기타 요소를 노트북 계층 구조에 추가합니다.

appendChildLast 메서드를 사용하여 완전히 준비된 페이지를 Document 객체에 삽입합니다. 이제 문서는 전체에 기본 단락 스타일이 적용된 페이지를 포함하고 있어 저장 준비가 완료되었습니다.

```java
page.appendChildLast(outline);
```

## OneNote 문서를 디스크에 저장하려면 어떻게 하나요?

Document.save는 노트북을 지정된 형식의 파일로 기록합니다.

save 메서드를 사용하여 Document를 .one 파일로 저장하고, 파일이 기록될 전체 경로를 지정합니다. 결과 파일은 모든 단락에 기본 글꼴 크기가 이미 적용된 상태로 OneNote에서 열립니다.

```java
document.appendChildLast(page);
```

## 저장된 파일을 확인하는 최종 단계는 무엇인가요?

OneNote에서 파일을 열면 적용된 스타일을 시각적으로 확인할 수 있습니다.

생성된 .one 파일을 Microsoft OneNote 또는 호환 가능한 뷰어에서 엽니다. 지정한 기본 글꼴 크기로 모든 단락이 렌더링되는 것을 확인할 수 있으며, 스타일이 올바르게 적용되고 문서가 오류 없이 로드되는지 확인할 수 있습니다.

```java
document.save(Paths.get(dataDir, "SetDefaultParagraphStyle.one").toString());
```

## 일반적인 문제 및 해결책

- **Paragraph style not applied:** `Document` 객체에 스타일을 `RichText` 인스턴스를 만들기 *앞에* 설정했는지 확인하십시오.  
- **Unexpected font size:** `fontSize` 속성에 포인트(pt)를 사용했는지 확인하십시오; 픽셀을 전달하면 스케일 차이가 발생할 수 있습니다.  
- **Large notebooks cause memory pressure:** 대용량 파일을 효율적으로 처리하려면 `LoadOptions.setLoadMode(LoadMode.Stream)`와 함께 `Document.load(inputStream, LoadOptions)`를 사용하십시오.
