# OneNote 하이퍼링크 및 이미지

## 소개

OneNote 기술을 내장한 Java 개발자이신가요? Aspose.Note for Java와 함께하는 종합적인 튜토리얼을 통해 OneNote 경험을 강화할 수 있는 전문 지식을 제공받을 수 있습니다. 이 가이드에서는 OneNote 문서에 **이미지에 하이퍼링크를 추가하는 방법**을 배우게 하여 메모지를 인터랙티브하고 보고적으로 매력적으로 만들 수 있습니다.

## 빠른 답변
- **“이미지에 하이퍼링크 추가”는 무엇을 의미하는지?** OneNote 페이지 내 이미지를 클릭할 수 있는 URL을 삽입합니다.
- **어떤 라이브러리가 있나요?** Aspose.Note for Java는 이미지를 대표하는 유창한 API를 제공합니다.
- **라이센스가 필요합니까?** 평가를 위해 무료로 실험을 사용할 수 있으며, 스피커에서는 회전이 필요합니다.
- **파일 대신 스트림을 사용할 수 있습니까?** 예—Aspose.Note는 `InputStream`을 통해 이미지를 로드하고 디버깅할 수 있습니다.
- **OneNote 2016 및 OneNote for Windows10과 호환되나요?** 생성된 `.one` 파일은 최신 OneNote 클라이언트에서 모두 작동합니다.

## 자주 묻는 질문

**Q: 모든 이미지 형식(PNG, JPEG, GIF)에 하이퍼링크를 추가할 수 있나요?**
답: 예. Aspose.Note는 모든 표준 이미지 형식을 지원하며, 하이퍼링크는 형식에 상관없이 이미지가 첨부됩니다.

**Q: 하이퍼링크를 추가하려면 OneNote 파일을 편집 모드로 열어야 할까요?**
답: 그렇지 않습니다. 문서를 메모리 내에서 완전히 생성하거나 수정한 뒤에 `.one` 파일로 수정할 수 있습니다.

**Q: 모바일 OneNote 앱에서도 하이퍼링크가 작동하나요?**
A: 물론입니다. 생성된 하이퍼링크는 OneNote 파일 형식에 저장되고 데스크톱, 웹 및 모바일 클라이언트에서 인식됩니다.

**Q: 하이퍼링크가 추가로 있도록 해줄 수 있나요?**
A: 저장 후 OneNote에서 파일을 사용하는 이미지를 클릭하면 연결된 URL이 기본 브라우저에서 작동됩니다.

**Q: 하나의 이미지에 여러 하이퍼링크를 추가할 수 있나요?**
A: 하나의 이미지에는 하나의 하이퍼링크만 추가할 수 있습니다. 다양한 링크를 제공하려면 별도의 링크를 선택하여 있는 건물 이미지를 사용하거나 이미지를 추가하는 것이 좋습니다.
