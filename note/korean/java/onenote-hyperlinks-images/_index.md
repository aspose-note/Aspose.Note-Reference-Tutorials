# OneNote 하이퍼링크 및 이미지

## Introduction

OneNote 기술을 향상시키고자 하는 Java 개발자이신가요? Aspose.Note for Java와 함께하는 포괄적인 튜토리얼을 통해 OneNote 경험을 강화할 수 있는 전문 지식을 제공받으세요. 이 가이드에서는 OneNote 문서에 **이미지에 하이퍼링크 추가하는 방법**을 배우게 되며, 메모를 인터랙티브하고 시각적으로 매력적으로 만들 수 있습니다.

## Quick Answers
- **“이미지에 하이퍼링크 추가”는 무엇을 의미하나요?** OneNote 페이지 내 이미지 객체에 클릭 가능한 URL을 삽입합니다.  
- **어떤 라이브러리가 이를 지원하나요?** Aspose.Note for Java는 이미지 하이퍼링크를 위한 유창한 API를 제공합니다.  
- **라이선스가 필요합니까?** 평가를 위해 무료 체험을 사용할 수 있으며, 프로덕션에서는 상업용 라이선스가 필요합니다.  
- **파일 대신 스트림을 사용할 수 있나요?** 예—Aspose.Note는 `InputStream`을 통해 이미지를 로드하고 저장할 수 있습니다.  
- **OneNote 2016 및 OneNote for Windows 10과 호환되나요?** 생성된 `.one` 파일은 최신 OneNote 클라이언트 모두에서 작동합니다.

## Frequently Asked Questions

**Q: 모든 이미지 형식(PNG, JPEG, GIF)에 하이퍼링크를 추가할 수 있나요?**  
A: 예. Aspose.Note는 모든 표준 이미지 형식을 지원하며, 하이퍼링크는 형식에 관계없이 이미지 객체에 첨부됩니다.

**Q: 하이퍼링크를 추가하려면 OneNote 파일을 편집 모드로 열어야 하나요?**  
A: 아니요. 문서를 메모리 내에서 완전히 생성하거나 수정한 뒤 `.one` 파일로 저장할 수 있습니다.

**Q: 모바일 OneNote 앱에서도 하이퍼링크가 작동하나요?**  
A: 물론입니다. 생성된 하이퍼링크는 OneNote 파일 형식에 저장되며, 데스크톱, 웹 및 모바일 클라이언트에서 인식됩니다.

**Q: 하이퍼링크가 올바르게 추가되었는지 어떻게 확인할 수 있나요?**  
A: 저장 후 OneNote에서 파일을 열고 이미지를 클릭하면 연결된 URL이 기본 브라우저에서 열립니다.

**Q: 하나의 이미지에 여러 하이퍼링크를 추가할 수 있나요?**  
A: 하나의 이미지에는 하나의 하이퍼링크만 포함될 수 있습니다. 여러 링크를 제공하려면 별도의 클릭 영역이 있는 복합 이미지를 사용하거나 별도의 이미지 객체를 추가하십시오.
