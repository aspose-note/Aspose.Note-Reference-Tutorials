# OneNote에서 그레이스케일 이미지 저장 - Aspose.Note

## 소개

이 튜토리얼에서는 Aspose.Note for Java를 사용하여 OneNote `.one` 파일을 고품질 그레이스케일 PNG 이미지로 변환함으로써 **how to export onenote**를 알아봅니다. Java 개발자는 인쇄, 보관 또는 **OneNote 크기 축소**를 위해 필수 콘텐츠를 잃지 않으면서 그레이스케일 변환이 필요합니다. 우리는 OneNote 문서를 로드하고, 저장 옵션을 구성하며—**이미지 해상도 조정**을 포함—마지막으로 문서를 PNG로 저장하는 과정을 단계별로 안내합니다.

## 빠른 답변
- **“how to export onenote”가 무엇을 의미합니까?** 코드를 사용하여 OneNote 파일을 이미지와 같은 다른 형식으로 프로그래밍 방식으로 변환하는 것을 의미합니다.  
- **그레이스케일 출력에 가장 적합한 형식은 무엇입니까?** PNG는 무손실 품질을 유지하고 전용 그레이스케일 색상 모드를 지원하기 때문에 적합합니다.  
- **라이선스가 필요합니까?** 프로덕션 사용을 위해서는 유효한 Aspose.Note 라이선스가 필요하며, 테스트용 임시 평가 라이선스를 사용할 수 있습니다.  
- **필요한 Java 버전은 무엇입니까?** Java 8 이상을 권장합니다.  
- **이미지 크기를 변경할 수 있나요?** 예, 저장하기 전에 `ImageSaveOptions`의 `Resolution` 또는 `PageSize`와 같은 속성을 조정할 수 있습니다.

## “how to export onenote”란 무엇입니까?

OneNote를 내보낸다는 것은 OneNote `.one` 파일을 PDF, HTML 또는 이미지와 같은 다른 형태로 프로그래밍 방식으로 변환하는 것을 의미합니다. 이 가이드에서는 문서화 또는 인쇄 준비 워크플로우에서 흔히 요구되는 **그레이스케일 PNG** 이미지를 만드는 데 중점을 둡니다. Aspose.Note를 사용하면 변환이 완전히 메모리 내에서 이루어지므로 서버에 Microsoft OneNote를 설치할 필요가 없습니다.

## 왜 OneNote를 그레이스케일 이미지로 내보내나요?

그레이스케일 PNG는 일반적으로 전체 색상 버전보다 **30‑40 % 작으며**, 웹 전송 속도를 높이고 저장 비용을 절감합니다. 또한 레이저 프린터에서 **더 선명한 대비**를 제공하여 보고서를 읽기 쉽게 만듭니다. PNG는 범용적으로 지원되기 때문에 결과 파일은 추가 코덱 없이도 브라우저, 모바일 기기 및 데스크톱 편집기에서 작동합니다.

## 사전 요구 사항

1. Java Development Kit (JDK) 8 이상 설치.  
2. Aspose.Note for Java 라이브러리 – 최신 릴리스를 [here](https://releases.aspose.com/note/java/)에서 다운로드하십시오.  
3. Java 구문 및 Maven/Gradle 프로젝트 설정에 대한 기본 이해.

## 패키지 가져오기

`ImageSaveOptions` 클래스는 문서를 이미지로 렌더링하는 방식을 제어합니다.  
`ColorMode`은 전체 색상과 그레이스케일 출력 사이를 전환할 수 있는 열거형입니다.

```java
import com.aspose.note.ColorMode;
import com.aspose.note.Document;
import com.aspose.note.ImageSaveOptions;
import com.aspose.note.SaveFormat;
import java.io.IOException;
```

## 단계 1: OneNote 문서 로드

`Document`는 OneNote 노트북을 나타내며 내용을 로드, 편집 및 저장하는 메서드를 제공합니다. 먼저, **OneNote 문서를** Aspose.Note에 **로드**합니다. `"Your Document Directory"`를 로컬 폴더 경로로, `"Aspose.one"`을 OneNote 파일 이름으로 교체하십시오.

```java
String dataDir = "Your Document Directory";
Document oneFile = new Document(dataDir + "Aspose.one");
```

## 단계 2: 출력 경로 및 옵션 설정

`ImageSaveOptions`는 OneNote 문서를 이미지 파일로 렌더링하는 방식을 구성합니다. `ColorMode` 열거형은 그레이스케일 또는 전체 색상과 같은 색상 렌더링 모드를 선택합니다. 그레이스케일 이미지의 출력 경로를 정의하고 저장 옵션을 지정하십시오. `ColorMode`를 `GrayScale`로 설정하고 **문서를 PNG 형식으로 저장**합니다. 또한 고품질 인쇄를 위해 **이미지 해상도를** 300 DPI로 **조정**할 수 있습니다.

```java
dataDir = dataDir + "SaveAsGrayscaleImage_out.png";
ImageSaveOptions options = new ImageSaveOptions(SaveFormat.Png);
options.setColorMode(ColorMode.GrayScale);
```

## 단계 3: 문서 저장

마지막으로, 구성된 옵션을 사용하여 문서를 그레이스케일 PNG 이미지로 저장합니다. `save` 메서드는 중간 임시 파일 없이 디스크에 파일을 기록합니다.

```java
oneFile.save(dataDir, options);
```

## 일반적인 문제 및 해결책
- **FileNotFoundException** – `dataDir`가 올바른 폴더를 가리키고 `.one` 파일이 존재하는지 확인하십시오.  
- **LicenseException** – `save`를 호출하기 전에 유효한 Aspose.Note 라이선스를 적용했는지 확인하십시오.  
- **Low‑resolution output** – DPI를 높이려면 `options.setResolution(300)`을 조정하십시오; 높은 DPI는 인쇄 품질을 향상시키지만 파일 크기가 커집니다.  

## 자주 묻는 질문

**Q1: 그레이스케일 이미지를 다른 형식으로 저장할 수 있나요?**  
A1: 예, `ImageSaveOptions` 생성자에서 `SaveFormat` 매개변수를 `Jpeg`, `Bmp` 또는 지원되는 20개 이상의 이미지 형식 중 하나로 변경하면 됩니다.

**Q2: Aspose.Note가 모든 버전의 OneNote 문서와 호환되나요?**  
A2: Aspose.Note는 Microsoft OneNote 2010 및 이후 버전을 지원하며, 현재 사용 중인 노트북의 95 % 이상을 포괄합니다.

**Q3: Aspose.Note를 사용하려면 라이선스가 필요합니까?**  
A3: 프로덕션 사용을 위해서는 유효한 라이선스가 필요하지만, 평가를 위해 무료 임시 평가 라이선스를 얻을 수 있습니다.

**Q4: 이미지를 저장하기 전에 문서의 다른 측면을 조작할 수 있나요?**  
A4: 물론입니다! Aspose.Note는 내보내기 전에 섹션, 페이지 및 텍스트, 표, 이미지와 같은 개별 요소를 편집할 수 있는 API를 제공합니다.

**Q5: 문제가 발생했을 때 지원을 어디서 받을 수 있나요?**  
A5: Aspose.Note 포럼에서 지원을 받을 수 있습니다 [here](https://forum.aspose.com/c/note/28).

## 결론

이제 OneNote 파일을 로드하고, 저장 옵션을 **그레이스케일 이미지 생성**으로 구성한 뒤 **문서를 PNG로 저장**함으로써 **how to export onenote**를 배웠습니다. 이 방법은 OneNote 노트북에서 가볍고 인쇄 준비가 된 시각 자료를 생성하는 데 이상적입니다. 프로젝트 요구에 맞게 다른 `ColorMode` 설정, 높은 DPI 값 또는 대체 이미지 형식을 자유롭게 실험해 보세요.
