# Aspose.Note를 사용하여 OneNote를 HTML로 저장하기

## 소개

웹 게시, 보고서 작성 또는 보관을 위해 **OneNote를 HTML로 저장**해야 하는 경우, Aspose.Note for .NET은 Microsoft Office 없이 Windows, Linux 및 macOS에서 작동하는 깔끔한 코드‑전용 솔루션을 제공합니다. 이 튜토리얼에서는 처음부터 OneNote 문서를 만들고, 페이지를 조작하며, 다양한 옵션을 사용해 전체 노트북 또는 선택된 페이지를 HTML로 내보내는 방법을 배웁니다. 가이드는 모든 단계를 안내하고 각 설정이 중요한 이유를 설명하며, HTML 내보내기가 빛을 발하는 실제 시나리오(예: 지식‑베이스 포털에 OneNote 콘텐츠 삽입 또는 정적 문서 생성)를 보여줍니다.

## 빠른 답변
- **내보내기의 기본 메서드는 무엇입니까?** `Aspose.Note.Document` 인스턴스에서 `document.Save("output.html", SaveFormat.Html)`를 호출합니다.  
- **단일 페이지를 내보낼 수 있나요?** 예—`document.Save(pageRange, "page.html", SaveFormat.Html)`를 사용합니다.  
- **리소스(이미지, CSS)가 자동으로 포함되나요?** 기본적으로 별도 파일로 저장되며, `HtmlSaveOptions`를 사용해 메모리 스트림에 포함시킬 수 있습니다.  
- **.NET Core가 지원되나요?** 물론—Aspose.Note는 .NET 5, .NET 6 및 .NET Core 3.1+와 호환됩니다.  
- **개발에 라이선스가 필요합니까?** 무료 30일 체험판으로 테스트가 가능하며, 라이선스를 적용하면 평가 워터마크가 제거됩니다.

## “save onenote as html”란 무엇입니까?

**“Save OneNote as HTML”**은 고유 바이너리 형식으로 저장된 OneNote 노트북을 표준 HTML 파일로 변환하는 과정을 의미합니다. 변환은 노트북의 계층 구조, 텍스트 서식, 이미지, 표 및 기타 리소스를 보존하면서 OneNote‑전용 요소를 동등한 HTML 태그와 CSS 스타일로 변환합니다. 이를 통해 개발자는 OneNote 콘텐츠를 웹사이트에 표시하거나 콘텐츠‑관리 시스템에 통합하거나 OneNote 클라이언트 없이도 보편적으로 접근 가능한 형식으로 보관할 수 있습니다.

## 왜 OneNote를 HTML로 변환합니까?

Aspose.Note는 **50개 이상의 출력 형식**(PDF, DOCX, PNG, HTML 등)을 지원하고, 스트리밍 아키텍처 덕분에 **최대 500 페이지**까지 메모리 전체를 로드하지 않고 처리할 수 있습니다. HTML로 변환하면 다음과 같은 이점이 있습니다:

1. **OneNote 클라이언트 없이 웹에 콘텐츠를 게시**할 수 있습니다.  
2. **HTML만 허용하는 콘텐츠‑관리 시스템과 통합**할 수 있습니다.  
3. **검색 엔진을 위한 텍스트 추출**이 가능합니다.  
4. **페이지‑레벨 리소스**(이미지, 폰트)를 다른 웹 프로젝트에서 재사용할 수 있습니다.

- OneNote의 계층 구조(노트북 → 섹션 → 페이지)에 대한 기본 이해.

## OneNote를 HTML로 저장하는 방법 – 단계별

OneNote를 HTML로 저장하려면 세 가지 핵심 작업을 수행합니다: 노트북을 로드하거나 생성하고, 원하는 내보내기 옵션을 구성한 다음, 적절한 `Save` 메서드를 호출합니다. 먼저 `Document` 객체를 생성 또는 로드하고, `HtmlSaveOptions`를 조정해 리소스 처리, CSS 생성 및 인코딩을 제어합니다. 마지막으로 파일 경로나 스트림을 지정해 `document.Save`를 호출하고, 필요에 따라 `PageRange`를 지정해 내보내기를 제한합니다. 이 체계적인 접근 방식은 다양한 환경에서 일관된 결과를 보장하고 프로젝트 요구에 맞게 출력을 맞춤화할 수 있게 합니다.

### 단계 1: 기본 옵션으로 OneNote 문서 만들기

이 첫 번째 예제에서는 새 `Document`를 인스턴스화하고, 제목이 있는 단일 페이지를 추가한 뒤, 기본 `HtmlSaveOptions`를 사용해 전체 노트북을 HTML로 저장합니다. 별도의 매개변수가 필요 없으므로 **OneNote를 HTML로 저장**하는 가장 빠른 방법입니다.

```csharp
using System;
using System.Drawing;
using System.Globalization;
using System.IO;

using Aspose.Note.Saving;
using Aspose.Note.Saving.Html;

```

### 단계 2: 특정 페이지 범위를 HTML로 내보내기

예를 들어 회의 노트의 한 챕터만 내보내야 할 때가 있습니다. `Save`에 `PageRange` 객체를 전달하면 1‑3 페이지만 대상이 되고 나머지는 그대로 유지됩니다.

```csharp
public static void CreateAndSaveToHTMLUsingDefaultOptions()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateOneNoteDocAndSaveToHTML_out.html");
    doc.Save(outputPath);

    Console.WriteLine("\nOneNote document created successfully.\nFile saved at " + outputPath);
}
```

### 단계 3: 리소스를 포함한 메모리 스트림으로 HTML 저장

단일 자체 포함 HTML 페이로드(API 응답 등)가 필요할 때는 CSS, 폰트 및 이미지를 `MemoryStream`에 직접 포함합니다. `HtmlSaveOptions`에서 `EmbedResources`를 `true`로 설정하면 압축된 HTML 문자열을 생성할 수 있습니다.

```csharp
public static void CreateAndSavePageRange()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateAndSavePageRange_out.html");
    doc.Save(outputPath, new HtmlSaveOptions { PageCount = 1, PageIndex = 0 });

    Console.WriteLine("\nOneNote document created successfully and saved as page range.\nFile saved at " + outputPath);
}
```

### 단계 4: 리소스를 별도 파일로 저장하여 HTML 파일로 저장

전통적인 웹 폴더 구조—HTML 파일과 자산 폴더를 원한다면 임베딩을 비활성화하고 출력 디렉터리를 지정합니다. 이 방식은 소스 파일을 깔끔하게 유지하고 이미지나 스타일시트 캐싱을 쉽게 합니다.

```csharp
public static void SaveAsHTMLToMemoryStreamWithEmbeddedResources()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportEmbedded,
        ExportFonts = ResourceExportType.ExportEmbedded,
        ExportImages = ResourceExportType.ExportEmbedded,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to a memory stream
    var memoryStream = new MemoryStream();
    document.Save(memoryStream, options);
}
```

### 단계 5: 콜백을 사용하여 리소스를 관리하며 HTML 저장

이미지를 데이터베이스나 클라우드 버킷에 저장하는 등 고급 시나리오에서는 `ResourceSavingCallback` 대리자를 제공해 각 리소스가 어떻게 저장되는지 정확히 제어하면서 API는 올바른 HTML 참조를 생성합니다.

```csharp
public static void SaveAsHTMLToFileWithResourcesInSeparateFiles()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportAsStream,
        ExportFonts = ResourceExportType.ExportAsStream,
        ExportImages = ResourceExportType.ExportAsStream,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to HTML file with resources stored in separate files
    document.Save(Path.Combine(dataDir, "document_out.html"), options);
}
```

## 일반적인 문제 및 해결책

| 문제 | 발생 원인 | 해결 방법 |
|------|----------|----------|
| **이미지가 깨짐** | 리소스가 HTML 파일과 다른 폴더에 저장되었습니다. | `HtmlSaveOptions.ResourcesFolder`가 참조하는 폴더와 일치하도록 하거나 `EmbedResources`를 활성화하십시오. |
| **CSS 스타일이 누락됨** | 기본 CSS 생성이 비활성화되었습니다. | `HtmlSaveOptions.GenerateCss = true`(기본값)로 설정하거나 CSS를 수동으로 포함하십시오. |
| **대형 노트북으로 인한 OutOfMemory** | 비스트리밍 `Document` 생성자가 전체 파일을 로드합니다. | `Document.Load(stream, LoadOptions)`와 `LoadOptions.LoadFromMemory = true`를 사용하여 페이지를 스트리밍하십시오. |
| **HTML 인코딩 문제(비ASCII 문자)** | `HtmlSaveOptions`에 잘못된 `Encoding`이 설정되었습니다. | `HtmlSaveOptions.Encoding = Encoding.UTF8`로 설정하십시오. |
| **Linux에서 저장 실패** | 이미지 처리에 필요한 네이티브 종속성이 누락되었습니다. | Linux 호스트에 `libgdiplus`를 설치하십시오. |

```csharp
public static void SaveAsHTMLToMemoryStreamWithCallBacksToSaveResources()
{
    // Specify the saving callbacks configuration
    var savingCallbacks = new UserSavingCallbacks()
    {
        RootFolder = "documentFolder",
        CssFolder = "css",
        KeepCssStreamOpened = true,
        ImagesFolder = "images",
        FontsFolder = "fonts"
    };

    // Specify HTML saving options
    var options = new HtmlSaveOptions
    {
        FontFaceTypes = FontFaceType.Ttf,
        CssSavingCallback = savingCallbacks,
        FontSavingCallback = savingCallbacks,
        ImageSavingCallback = savingCallbacks
    };

    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Save the document to HTML format with resources managed by user-defined callbacks
    using (var stream = File.Create(Path.Combine(savingCallbacks.RootFolder, "document.html")))
    {
        document.Save(stream, options);
    }

    // Manually append data to the CSS stream
    using (var writer = new StreamWriter(savingCallbacks.CssStream))
    {
        writer.WriteLine();
        writer.WriteLine("/* This line is appended to stream manually by user */");
    }
}
```
