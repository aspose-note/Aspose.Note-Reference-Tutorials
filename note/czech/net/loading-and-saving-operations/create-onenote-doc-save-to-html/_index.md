# Uložit OneNote jako HTML pomocí Aspose.Note

## Úvod

Pokud potřebujete **save OneNote as HTML** pro webové publikování, reportování nebo archivaci, Aspose.Note pro .NET vám poskytuje čisté řešení pouze pomocí kódu, které funguje na Windows, Linuxu i macOS bez nutnosti Microsoft Office. V tomto tutoriálu se naučíte, jak vytvořit dokument OneNote od nuly, manipulovat s jeho stránkami a exportovat celý sešit – nebo vybrané stránky – do HTML pomocí různých možností. Průvodce vás provede každým krokem, vysvětlí, proč je každé nastavení důležité, a ukáže reálné scénáře, kde export do HTML vyniká, například při vkládání obsahu OneNote do portálu znalostní báze nebo generování statické dokumentace.

## Rychlé odpovědi
- **Jaká je hlavní metoda pro export?** Zavolejte `document.Save("output.html", SaveFormat.Html)` na instanci `Aspose.Note.Document`.  
- **Mohu exportovat jednu stránku?** Ano—použijte `document.Save(pageRange, "page.html", SaveFormat.Html)`.  
- **Vkládají se zdroje (obrázky, CSS) automaticky?** Ve výchozím nastavení jsou uloženy jako samostatné soubory; můžete je vložit do paměťového proudu pomocí `HtmlSaveOptions`.  
- **Je .NET Core podporováno?** Ano—Aspose.Note funguje s .NET 5, .NET 6 a .NET Core 3.1+.  
- **Potřebuji licenci pro vývoj?** Bezplatná 30‑denní zkušební verze funguje pro testování; licence odstraňuje vodotisky hodnocení.

## Co je „save onenote as html“?

**“Save OneNote as HTML”** označuje proces převodu sešitu OneNote, který je uložen v proprietárním binárním formátu, do standardních HTML souborů, které může vykreslit libovolný webový prohlížeč. Konverze zachovává hierarchickou strukturu sešitu, formátování textu, obrázky, tabulky a další zdroje, přičemž OneNote‑specifické prvky převádí na ekvivalentní HTML značky a CSS styly. To umožňuje vývojářům zobrazovat obsah OneNote na webových stránkách, integrovat jej do systémů pro správu obsahu nebo archivovat v univerzálně přístupném formátu bez nutnosti klienta OneNote.

## Proč převádět OneNote do HTML?

Aspose.Note může exportovat do **50+ výstupních formátů** (včetně PDF, DOCX, PNG a HTML) a zpracovává sešity s **až 500 stránkami** bez načítání celého souboru do paměti díky své streamovací architektuře. Převod do HTML vám umožní:

1. **Publikovat obsah na webu** bez nutnosti klienta OneNote.  
2. **Integrovat s CMS**, které přijímají pouze HTML.  
3. **Extrahovat prohledávatelný text** pro indexovací nástroje.  
4. **Znovu použít zdroje na úrovni stránky** (obrázky, fonty) v jiných webových projektech.

## Jak uložit onenote jako html – krok za krokem

Ukládání OneNote jako HTML zahrnuje tři hlavní akce: načtení nebo vytvoření sešitu, nastavení požadovaných možností exportu a volání příslušné metody `Save`. Nejprve vytvoříte nebo načtete objekt `Document`, poté můžete upravit `HtmlSaveOptions` pro řízení zacházení se zdroji, generování CSS a kódování. Nakonec zavoláte `document.Save` s cestou k souboru nebo proudem, případně s `PageRange` pro omezení exportu. Tento systematický přístup zajišťuje konzistentní výsledky napříč různými prostředími a umožňuje přizpůsobit výstup potřebám vašeho projektu.

### Krok 1: Vytvořit dokument OneNote s výchozími možnostmi

V tomto prvním příkladu vytvoříme nový `Document`, přidáme jednu stránku s titulkem a uložíme celý sešit do HTML pomocí výchozích `HtmlSaveOptions`. Žádné další parametry nejsou potřeba, což je nejrychlejší způsob, jak **save OneNote as HTML**.

```csharp
using System;
using System.Drawing;
using System.Globalization;
using System.IO;

using Aspose.Note.Saving;
using Aspose.Note.Saving.Html;

```

### Krok 2: Exportovat konkrétní rozsah stránek do HTML

Často potřebujete jen podmnožinu stránek – například export kapitoly poznámek ze schůzky. Předáním objektu `PageRange` do `Save` můžete cílit na stránky 1‑3 a zbytek nechat nedotčený.

```csharp
public static void CreateAndSaveToHTMLUsingDefaultOptions()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateOneNoteDocAndSaveToHTML_out.html");
    doc.Save(outputPath);

    Console.WriteLine("\nOneNote document created successfully.\nFile saved at " + outputPath);
}
```

### Krok 3: Uložit jako HTML do paměťového proudu s vloženými zdroji

Když potřebujete jediný samostatný HTML payload (např. pro odpovědi API), vložte CSS, fonty a obrázky přímo do `MemoryStream`. `HtmlSaveOptions` vám umožní nastavit `EmbedResources` na `true`, čímž získáte kompaktní HTML řetězec.

```csharp
public static void CreateAndSavePageRange()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateAndSavePageRange_out.html");
    doc.Save(outputPath, new HtmlSaveOptions { PageCount = 1, PageIndex = 0 });

    Console.WriteLine("\nOneNote document created successfully and saved as page range.\nFile saved at " + outputPath);
}
```

### Krok 4: Uložit jako HTML do souborů se zdroji v samostatných souborech

Pokud dáváte přednost klasickému výstupu webové složky – HTML soubor plus složka s aktivy – zakážete vkládání a určíte výstupní adresář. Tento přístup poskytuje čistší zdrojové soubory a usnadňuje cachování obrázků nebo stylů.

```csharp
public static void SaveAsHTMLToMemoryStreamWithEmbeddedResources()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportEmbedded,
        ExportFonts = ResourceExportType.ExportEmbedded,
        ExportImages = ResourceExportType.ExportEmbedded,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to a memory stream
    var memoryStream = new MemoryStream();
    document.Save(memoryStream, options);
}
```

### Krok 5: Uložit jako HTML pomocí zpětných volání pro správu zdrojů

Pokročilé scénáře, jako je ukládání obrázků do databáze nebo cloudového bucketu, vyžadují vlastní zpracování. Poskytnutím delegáta `ResourceSavingCallback` určíte, jak bude každý zdroj uložen, zatímco API nadále generuje správné odkazy v HTML.

```csharp
public static void SaveAsHTMLToFileWithResourcesInSeparateFiles()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportAsStream,
        ExportFonts = ResourceExportType.ExportAsStream,
        ExportImages = ResourceExportType.ExportAsStream,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to HTML file with resources stored in separate files
    document.Save(Path.Combine(dataDir, "document_out.html"), options);
}
```

## Časté problémy a řešení

| Problém | Proč k tomu dochází | Řešení |
|-------|----------------|-----|
| **Obrázky jsou poškozené** | Zdroje byly uloženy do jiného složky než HTML soubor. | Ujistěte se, že `HtmlSaveOptions.ResourcesFolder` odpovídá složce, na kterou odkazujete, nebo povolte `EmbedResources`. |
| **Chybí CSS styly** | Generování výchozího CSS bylo zakázáno. | Nastavte `HtmlSaveOptions.GenerateCss = true` (výchozí) nebo vložte CSS ručně. |
| **Velké sešity způsobují OutOfMemory** | Použití konstruktoru `Document` bez streamování načte celý soubor. | Použijte `Document.Load(stream, LoadOptions)` s `LoadOptions.LoadFromMemory = true` pro streamování stránek. |
| **Problémy s kódováním HTML pro ne‑ASCII znaky** | Nesprávné nastavení `Encoding` v `HtmlSaveOptions`. | Nastavte `HtmlSaveOptions.Encoding = Encoding.UTF8`. |
| **Ukládání selhává na Linuxu** | Chybějící nativní závislosti pro zpracování obrázků. | Nainstalujte `libgdiplus` na Linux host. |

```csharp
public static void SaveAsHTMLToMemoryStreamWithCallBacksToSaveResources()
{
    // Specify the saving callbacks configuration
    var savingCallbacks = new UserSavingCallbacks()
    {
        RootFolder = "documentFolder",
        CssFolder = "css",
        KeepCssStreamOpened = true,
        ImagesFolder = "images",
        FontsFolder = "fonts"
    };

    // Specify HTML saving options
    var options = new HtmlSaveOptions
    {
        FontFaceTypes = FontFaceType.Ttf,
        CssSavingCallback = savingCallbacks,
        FontSavingCallback = savingCallbacks,
        ImageSavingCallback = savingCallbacks
    };

    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Save the document to HTML format with resources managed by user-defined callbacks
    using (var stream = File.Create(Path.Combine(savingCallbacks.RootFolder, "document.html")))
    {
        document.Save(stream, options);
    }

    // Manually append data to the CSS stream
    using (var writer = new StreamWriter(savingCallbacks.CssStream))
    {
        writer.WriteLine();
        writer.WriteLine("/* This line is appended to stream manually by user */");
    }
}
```
