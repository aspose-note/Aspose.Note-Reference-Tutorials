## Co je programatické vytvoření tabulky OneNote?
`Programmatically create OneNote table` odkazuje na použití kódu – konkrétně Aspose.Note for Java API – k vytvoření objektu Table uvnitř stránky OneNote bez ruční interakce uživatele. Tento přístup automatizuje generování dokumentů, zajišťuje konzistenci a škáluje při velkých objemech práce. Umožňuje vývojářům vytvářet tabulky přímo z Java aplikací, šetří čas a snižuje chyby.

## Jak programaticky vytvořit tabulku OneNote?

`Document` představuje soubor sešitů OneNote načtený do paměti.  
`Page.getTables().add()` vytvoří novou Tabulku na stránce se zadaným počtem řádků a sloupců.

Nahrajte instanci `Document`, přidejte `Page`, poté zavolejte `Page.getTables().add()` s požadovaným počtem řádků a sloupců. Po vytvoření můžete nastavit hodnoty buněk, aplikovat ohraničení a stylovat tabulku – vše pomocí plynulých volání v Javě. Tento dvoustupňový vzor (vytvořit → konfigurovat) vám umožní okamžitě vytvářet tabulky, i pro více stránkové sešity.
