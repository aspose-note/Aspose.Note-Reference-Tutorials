Zmianażej przedstawiamy podstawowe koncepcje, zanim przejdziemy do dwóch szczegółowych tutoriali.

### Podstawowe koncepcje
- **Formatowanie tekstu:** Nazwa czcionki, rozmiar, kolor i podświetlenie są kontrolowane za pomocą właściwości `TextRun`.

**Najczęściej zadawane pytania**

**Q:** *Czy mogę zastosować ten sam styl do wielu akapitów jednocześnie?*  
**A:** Tak, możesz przeiterować kolekcję akapitów i przypisać wspólny obiekt `ParagraphStyle`.

**Q:** *Czy można zmienić domyślną czcionkę dla całego notatnika?*  
**A:** Oczywiście. Ustaw domyślny styl akapitu na poziomie notatnika, a wszystkie nowe strony będą go dziedziczyć.

**Q:** *Czy te API działają z zaszyfrowanymi plikami OneNote?*  
**A:** API może otworzyć pliki chronione hasłem, jeśli podasz hasło deszyfrujące podczas ładowania.

**Q:** *Co się stanie, jeśli ustawiam rozmiar czcionki przekraczający limity OneNote?*  
**A:** Aspose.Note ograniczy rozmiar do najbliższej obsługiwanej wartości, zapewniając, że dokument pozostanie prawidłowy.

**Q:** *Czy mogę podglądnąć stylizowaną notatkę przed zapisaniem?*  
**A:** Tak, możesz wyrenderować dokument do HTML lub PDF, aby natychmiast zobaczyć efekt wizualny.
