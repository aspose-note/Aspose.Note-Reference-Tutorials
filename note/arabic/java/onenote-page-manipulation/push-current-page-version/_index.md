# كيفية حفظ نسخة صفحة OneNote – دفع نسخة الصفحة الحالية في OneNote

## المقدمة

في هذا البرنامج التعليمي ستكتشف **كيفية حفظ صفحات OneNote** عن طريق دفع نسخة الصفحة الحالية باستخدام Aspose.Note for Java. سواء كنت بحاجة إلى الحفاظ على سجل تدقيق كامل أو ببساطة إدارة تاريخ الإصدارات، فإن الخطوات أدناه توضح لك كيفية تحميل ملف OneNote، إضافة إدخالات التاريخ، استنساخ صفحة، وتحديث نسخة OneNote برمجيًا.

## إجابات سريعة
- **ماذا يعني “دفع نسخة الصفحة الحالية”?** يضيف لقطة من الصفحة الحالية إلى تاريخ إصدارات المستند.  
- **لماذا نستخدم Aspose.Note for Java؟** يوفر واجهة برمجة تطبيقات pure‑Java للتعامل مع ملفات OneNote دون الحاجة إلى Microsoft Office.  
- **هل أحتاج إلى ترخيص لتجربة هذا؟** يمكن تنزيل نسخة تجريبية مجانية، لكن الترخيص مطلوب للاستخدام في الإنتاج.  
- **هل يمكنني أتمتة إصدار النسخ للعديد من الصفحات؟** نعم، يمكنك التكرار عبر الصفحات واستدعاء نفس الواجهة البرمجية لكل منها.  
- **هل الملف المحفوظ متوافق مع أحدث إصدارات OneNote؟** تحافظ Aspose.Note على التوافق مع صيغ OneNote الحالية.

## ما هو “كيفية حفظ OneNote” مع تاريخ الإصدارات؟
حفظ OneNote مع تاريخ الإصدارات يعني تخزين كل تعديل كإدخال منفصل بحيث يمكنك الرجوع إلى الوراء أو مراجعة التغييرات لاحقًا. تجعل فئة `PageHistory` في Aspose.Note هذا الأمر بسيطًا.

## لماذا دفع نسخة الصفحة الحالية؟
- **قابلية التدقيق:** يتم تسجيل كل تغيير، مما يلبي متطلبات الامتثال.  
- **التعاون:** يمكن لأعضاء الفريق رؤية من غير الذي غير ماذا ومتى.  
- **الأمان:** يمكن استعادة المحتوى الذي تم الكتابة فوقه عن طريق الخطأ من التاريخ.

## المتطلبات المسبقة

قبل أن نبدأ، تأكد من وجود ما يلي:

1. معرفة أساسية ببرمجة Java.  
2. تثبيت Java Development Kit (JDK) على جهازك.  
3. مكتبة Aspose.Note for Java – قم بتنزيلها من [here](https://releases.aspose.com/note/java/).  
4. مستند OneNote تجريبي (مثال: `Sample1.one`) تريد إنشاء نسخة منه.

## استيراد الحزم

أولاً، استورد الفئات المطلوبة حتى تتمكن من العمل مع مستندات OneNote وتاريخها.

```java
import java.io.IOException;

import com.aspose.note.Document;
import com.aspose.note.Page;
import com.aspose.note.PageHistory;
```

## الخطوة 1: تحميل مستند OneNote

تحميل ملف OneNote هو الخطوة الأولى في **كيفية إضافة التاريخ**. تقوم الواجهة البرمجية بقراءة ملف `.one` إلى كائن `Document`.

```java
String dataDir = "Your Document Directory";
Document document = new Document(dataDir + "Sample1.one");
```

> **نصيحة:** يجب أن يشير `dataDir` إلى المجلد الذي يحتوي على ملف OneNote الخاص بك. عدل اسم الملف إذا كنت تعمل على مستند مختلف.

## الخطوة 2: الحصول على الصفحة الحالية وتاريخها

لإدارة تاريخ الإصدارات تحتاج إلى مرجع للصفحة التي تريد إنشاء نسخة منها وكائن `PageHistory` المرتبط بها.

```java
Page page = document.getFirstChild();
PageHistory pageHistory = document.getPageHistory(page);
```

> **لماذا هذا مهم:** تقوم `getFirstChild()` بجلب الصفحة الأولى (يمكنك التكرار للحصول على الأخرى)، وتُعطيك `getPageHistory(page)` الحاوية التي تُخزن فيها لقطات النسخ.

## الخطوة 3: دفع نسخة الصفحة الحالية

الآن نحن **كيفية استنساخ الصفحة** ودفعها إلى التاريخ. الاستنساخ يخلق نسخة عميقة، مما يضمن أن اللقطة مستقلة عن التعديلات المستقبلية.

```java
pageHistory.addItem(page.deepClone());
```

> **نصيحة احترافية:** استخدام `deepClone()` يضمن أن جميع العناصر المتداخلة (النص، الصور، الجداول) تُلتقط في إدخال النسخة.

## الخطوة 4: حفظ المستند

أخيرًا، **تحديث نسخة OneNote** عن طريق حفظ المستند. سيحتوي الملف الجديد على إدخال التاريخ المضاف.

```java
document.save(dataDir + "PushCurrentPageVersion_out.one");
```

عند فتح `PushCurrentPageVersion_out.one` في OneNote، سترى تاريخ الإصدارات المتاح عبر واجهة المستخدم.

## المشكلات الشائعة وكيفية تجنبها

- **عدم وجود أذونات كتابة:** تأكد من أن دليل الإخراج قابل للكتابة؛ وإلا ستطلق `save()` استثناءً.  
- **مسار ملف غير صحيح:** تحقق مرة أخرى أن `dataDir` ينتهي بفاصل مسار (`/` أو `\`).  
- **المستندات الكبيرة:** بالنسبة لملفات OneNote الكبيرة جدًا، فكر في استنساخ الصفحات التي تحتاج إلى إنشاء نسخة منها فقط لتقليل استهلاك الذاكرة.

## الخاتمة

أنت الآن تعرف **كيفية حفظ صفحات OneNote** عن طريق دفع النسخة الحالية، مما يتيح لك **إدارة تاريخ الإصدارات** و**تحديث نسخة OneNote** باستخدام Aspose.Note for Java. يمكن دمج هذا النهج في خطوط تقارير آلية، حلول النسخ الاحتياطي، أو أدوات التحرير التعاوني.
