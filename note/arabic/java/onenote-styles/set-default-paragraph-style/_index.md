# تعيين حجم الخط الافتراضي في OneNote – تعيين نمط الفقرة الافتراضي في OneNote

## مقدمة

إن ضبط **default font size onenote** برمجيًا يتيح لك الحفاظ على مظهر متسق عبر جميع الصفحات دون الحاجة إلى تعديل كل فقرة يدويًا. في هذا البرنامج التعليمي ستتعلم كيفية استخدام Aspose.Note for Java لتعريف نمط فقرة افتراضي يتضمن حجم الخط المفضل لديك، اسم الخط، وخيارات تنسيق أخرى. في النهاية ستحصل على مقطع شفرة قابل لإعادة الاستخدام يمكنك إدراجه في أي مشروع Java يعمل مع ملفات OneNote.

## إجابات سريعة
- **What does “default font size onenote” control?** يحدد حجم الخط الأساسي المطبق على كل فقرة جديدة ما لم يتم تجاوزها.  
- **Which library handles this?** Aspose.Note for Java توفر API سلس لتعيين القيم الافتراضية للفقرة.  
- **Do I need a license?** النسخة التجريبية المجانية تعمل للتطوير؛ يتطلب الإنتاج ترخيصًا تجاريًا.  
- **Can I change other style attributes at the same time?** نعم – اسم الخط، اللون، المحاذاة، وتباعد الأسطر كلها قابلة للتكوين.  
- **Is this compatible with all OneNote versions?** Aspose.Note يدعم ملفات OneNote من عام 2007 فصاعدًا، ويغطي أكثر من 95 % من الدفاتر النشطة.

## ما هو default font size onenote؟
إن **default font size onenote** هو حجم النص الأساسي المطبق على الفقرات الجديدة في مستند OneNote عندما لا يتم تحديد حجم صريح. من خلال تعريفه مرة واحدة، تتجنب التنسيق المتكرر وتضمن اتساقًا بصريًا عبر الدفتر.

## لماذا يتم تعيين نمط الفقرة الافتراضي في OneNote؟
يدعم Aspose.Note **أكثر من 50 تنسيقًا للإدخال والإخراج** ويمكنه معالجة الدفاتر التي يصل حجم محتواها إلى **2 GB** دون تحميل الملف بالكامل إلى الذاكرة. يؤدي تعيين نمط الفقرة الافتراضي إلى تقليل عدد استدعاءات API بنسبة تصل إلى **40 %**، ويسرّع إنشاء المستندات، ويضمن أن كل فقرة تتبع إرشادات العلامة التجارية للشركة.

## المتطلبات المسبقة

1. **Java Development Kit (JDK)** – يُنصح باستخدام JDK 11 أو أحدث.  
2. **Aspose.Note for Java** – قم بتنزيله من [download page](https://releases.aspose.com/note/java/).  
3. **An IDE** مثل Eclipse أو IntelliJ IDEA أو VS Code مع دعم Java.  

## استيراد الحزم

أولاً، استورد الحزم اللازمة لبدء كتابة الشفرة:

```java
import com.aspose.note.*;

import java.awt.*;
import java.io.FileInputStream;
import java.io.IOException;
import java.nio.file.Paths;
import java.util.List;
```

الآن، دعنا نقسم شفرة المثال إلى خطوات متعددة:

## كيف أقوم بتهيئة مستند OneNote والصفحة والمخطط؟

يمثل Document دفتر OneNote بأكمله ويوفر طرقًا للتعامل مع محتوياته.  
تشير Page إلى صفحة واحدة داخل الدفتر، تحمل المخططات وعناصر أخرى.  
Outline هو حاوية تجمع عناصر النص الغني المرتبطة على الصفحة.

أنشئ الكائنات الأساسية التي تمثل ملف OneNote، صفحة داخل ذلك الملف، وحاوية المخطط التي تحتوي على النص الغني. تشكل هذه الكائنات الأساس لأي تنسيق لاحق ويجب إنشاؤها قبل إضافة المحتوى.

```java
String dataDir = "Your Document Directory";
Document document = new Document();
Page page = new Page();
Outline outline = new Outline();
```

## كيف يمكنني إنشاء عنصر مخطط لاستضافة الفقرات؟

OutlineElement هو عنصر فرعي من Outline يمكنه احتواء عدة كائنات RichText تمثل فقرات فردية.

يعمل عنصر المخطط كحاوية لعدة كائنات فقرة، مما يتيح لك تجميع كتل النص المرتبطة. بعد إنشائه، تقوم بإرفاقه بالصفحة بحيث يظهر النص المنسق في الموقع المقصود داخل الدفتر.

```java
OutlineElement outlineElem = new OutlineElement();
```

## كيف أحدد نمط الفقرة الافتراضي، بما في ذلك حجم الخط؟

ParagraphStyle يحدد سمات التنسيق مثل اسم الخط، الحجم، اللون، والمحاذاة التي تُطبق على الفقرة.

قم بإنشاء مثيل من ParagraphStyle، واضبط خاصية fontSize إلى القيمة المطلوبة (مثلاً 12 pt)، ويمكنك اختيارياً تحديد fontName أو اللون أو المحاذاة. عيّن هذا النمط كافتراضي للمستند بحيث يرث كل فقرة جديدة هذه الإعدادات تلقائيًا دون الحاجة إلى شفرة إضافية.

```java
ParagraphStyle defaultStyle = new ParagraphStyle()
										.setFontName("Courier New")
										.setFontSize(20);
```

## كيف يمكنني إنشاء نص غني يستخدم النمط الافتراضي تلقائيًا؟

RichText يمثل كتلة نص يمكن أن تحتوي على عدة مقاطع بتنسيق فردي.

أنشئ كائن RichText دون تحديد حجم خط صريح، معتمدًا على النمط الافتراضي الذي تم ضبطه مسبقًا. سيُظهر الكائن النص باستخدام حجم الخط المحدد وأي سمات تنسيق أخرى قمت بتكوينها، مما يضمن مظهرًا متسقًا عبر جميع الفقرات.

```java
RichText text = new RichText()
					.append("DefaultParagraphFontAndSize")
					.append(System.lineSeparator())
					.append("OnlyDefaultParagraphFont", new TextStyle().setFontSize(14))
					.append(System.lineSeparator())
					.append("OnlyDefaultParagraphFontSize", new TextStyle().setFontName("Verdana"));
text.setParagraphStyle(defaultStyle);
```

## كيف أضيف النص الغني إلى عنصر المخطط؟

appendChildLast يضيف عقدة فرعية إلى نهاية مجموعة الحاوية.

أضف مثيل RichText إلى عنصر المخطط الذي تم إنشاؤه مسبقًا باستخدام طريقة appendChildLast. يربط هذا الإجراء المحتوى المنسق بالمخطط، محافظًا على الهيكلية ويضمن ظهور النص بالترتيب الصحيح على الصفحة.

```java
outlineElem.appendChildLast(text);
```

## كيف أرفق عنصر المخطط بالصفحة؟

Page.appendChildLast يضيف عنصرًا فرعيًا مثل Outline إلى مجموعة الصفحة.

أضف المخطط إلى مجموعة المخططات في الصفحة باستخدام طريقة appendChildLast. يدمج هذا الخطوة المحتوى المنسق في بنية الصفحة، مما يجعله مرئيًا عند فتح المستند في OneNote.

```java
outline.appendChildLast(outlineElem);
```

## كيف أضيف الصفحة إلى مستند OneNote؟

Document.appendChildLast يضيف صفحة أو عنصرًا آخر إلى هيكل الدفتر.

أدرج الصفحة المُعدة بالكامل في كائن Document باستخدام طريقة appendChildLast. في هذه المرحلة يحتوي المستند على صفحة مع تطبيق نمط الفقرة الافتراضي في جميع أنحاءها، جاهزة للحفظ.

```java
page.appendChildLast(outline);
```

## كيف أحفظ مستند OneNote على القرص؟

Document.save يكتب الدفتر إلى ملف بالتنسيق المحدد.

احفظ Document كملف .one باستخدام طريقة save، مع توفير المسار الكامل حيث يجب كتابة الملف. سيفتح الملف الناتج في OneNote مع تطبيق حجم الخط الافتراضي على كل فقرة.

```java
document.appendChildLast(page);
```

## ما هي الخطوة النهائية للتحقق من الملف المحفوظ؟

فتح الملف في OneNote يتيح لك التأكد بصريًا من الأنماط المطبقة.

افتح ملف .one المُولد في Microsoft OneNote أو أي عارض متوافق. يجب أن ترى جميع الفقرات مُعرضة بحجم الخط الافتراضي الذي حددته، مما يؤكد أن النمط تم تطبيقه بشكل صحيح وأن المستند يُحمَّل دون أخطاء.

```java
document.save(Paths.get(dataDir, "SetDefaultParagraphStyle.one").toString());
```

ستقوم هذه الشفرة بتعيين نمط الفقرة الافتراضي في OneNote باستخدام Aspose.Note for Java، مما يضمن أن كل فقرة جديدة تتبنى تلقائيًا حجم الخط والتنسيق المختار.

## المشكلات الشائعة والحلول

- **Paragraph style not applied:** تحقق من أنك قمت بتعيين النمط على كائن `Document` *قبل* إنشاء أي مثيلات `RichText`.  
- **Unexpected font size:** تأكد من استخدام النقاط (pt) لخاصية `fontSize`؛ قد يؤدي تمرير البكسل إلى اختلافات في التحجيم.  
- **Large notebooks cause memory pressure:** استخدم `Document.load(inputStream, LoadOptions)` مع `LoadOptions.setLoadMode(LoadMode.Stream)` لمعالجة الملفات الكبيرة بكفاءة.
