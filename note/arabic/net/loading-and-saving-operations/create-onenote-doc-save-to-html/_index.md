# حفظ OneNote كـ HTML باستخدام Aspose.Note

## مقدمة

إذا كنت بحاجة إلى **حفظ OneNote كـ HTML** للنشر على الويب أو للتقارير أو للأرشفة، فإن Aspose.Note لـ .NET يوفر لك حلاً نظيفًا يعتمد على الكود فقط ويعمل على Windows وLinux وmacOS دون الحاجة إلى Microsoft Office. في هذا البرنامج التعليمي ستتعلم كيفية إنشاء مستند OneNote من الصفر، ومعالجة صفحاته، وتصدير دفتر الملاحظات بالكامل—أو صفحات مختارة—إلى HTML باستخدام مجموعة متنوعة من الخيارات. يوضح الدليل كل خطوة، ويشرح سبب أهمية كل إعداد، ويعرض سيناريوهات واقعية حيث يبرز تصدير HTML، مثل تضمين محتوى OneNote في بوابة قاعدة معرفة أو إنشاء وثائق ثابتة.

## إجابات سريعة
- **ما هي الطريقة الأساسية للتصدير؟** استدعِ `document.Save("output.html", SaveFormat.Html)` على كائن `Aspose.Note.Document`.  
- **هل يمكنني تصدير صفحة واحدة؟** نعم—استخدم `document.Save(pageRange, "page.html", SaveFormat.Html)`.  
- **هل يتم تضمين الموارد (الصور، CSS) تلقائيًا؟** بشكل افتراضي يتم حفظها كملفات منفصلة؛ يمكنك تضمينها في تدفق الذاكرة باستخدام `HtmlSaveOptions`.  
- **هل .NET Core مدعوم؟** بالتأكيد—Aspose.Note يعمل مع .NET 5 و .NET 6 و .NET Core 3.1+.  
- **هل أحتاج إلى ترخيص للتطوير؟** نسخة تجريبية مجانية لمدة 30 يومًا تكفي للاختبار؛ الترخيص يزيل علامات التقييم.

## ما هو “حفظ onenote كـ html”؟
**“حفظ OneNote كـ HTML”** يشير إلى عملية تحويل دفتر ملاحظات OneNote، المخزن بصيغة ثنائية مملوكة، إلى ملفات HTML قياسية يمكن لأي متصفح ويب عرضها. يحافظ التحويل على الهيكل الهرمي للدفتر، وتنسيق النص، والصور، والجداول، والموارد الأخرى، مع ترجمة العناصر الخاصة بـ OneNote إلى وسوم HTML مكافئة وأنماط CSS. يتيح ذلك للمطورين عرض محتوى OneNote على المواقع، دمجه في أنظمة إدارة المحتوى، أو أرشفته بصيغة يمكن الوصول إليها عالميًا دون الحاجة إلى عميل OneNote.

## لماذا تحويل OneNote إلى HTML؟
يمكن لـ Aspose.Note التصدير إلى **أكثر من 50 صيغة إخراج** (بما في ذلك PDF و DOCX و PNG و HTML) ومعالجة دفاتر ملاحظات **حتى 500 صفحة** دون تحميل الملف بالكامل في الذاكرة، بفضل بنية البث. تحويله إلى HTML يتيح لك:

1. **نشر المحتوى على الويب** دون الحاجة إلى عميل OneNote.  
2. **التكامل مع أنظمة إدارة المحتوى** التي تقبل HTML فقط.  
3. **استخراج نص قابل للبحث** لمحركات الفهرسة.  
4. **إعادة استخدام موارد مستوى الصفحة** (الصور، الخطوط) في مشاريع ويب أخرى.

## كيفية حفظ onenote كـ html – خطوة بخطوة
يتضمن حفظ OneNote كـ HTML ثلاث إجراءات أساسية: تحميل أو إنشاء دفتر ملاحظات، تكوين خيارات التصدير المطلوبة، واستدعاء طريقة `Save` المناسبة. أولاً، تنشئ أو تحمّل كائن `Document`، ثم قد تضبط `HtmlSaveOptions` للتحكم في معالجة الموارد، إنشاء CSS، والترميز. أخيرًا، تستدعي `document.Save` مع مسار ملف أو تدفق، مع إمكانية تحديد `PageRange` لتحديد نطاق التصدير. يضمن هذا النهج منهجيًا الحصول على نتائج متسقة عبر بيئات مختلفة ويسمح لك بتخصيص المخرجات وفقًا لاحتياجات مشروعك.

### الخطوة 1: إنشاء مستند OneNote بخيارات افتراضية
في هذا المثال الأول نقوم بإنشاء كائن `Document` جديد، نضيف صفحة واحدة بعنوان، ونحفظ دفتر الملاحظات بالكامل إلى HTML باستخدام `HtmlSaveOptions` الافتراضية. لا توجد معلمات إضافية مطلوبة، مما يجعلها أسرع طريقة لـ **حفظ OneNote كـ HTML**.

```csharp
using System;
using System.Drawing;
using System.Globalization;
using System.IO;

using Aspose.Note.Saving;
using Aspose.Note.Saving.Html;

```

### الخطوة 2: تصدير نطاق صفحات محدد إلى HTML
غالبًا ما تحتاج إلى جزء من الصفحات فقط—مثلاً تصدير فصل من ملاحظات الاجتماع. بتمرير كائن `PageRange` إلى `Save`، يمكنك استهداف الصفحات 1‑3 مع ترك البقية دون تعديل.

```csharp
public static void CreateAndSaveToHTMLUsingDefaultOptions()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateOneNoteDocAndSaveToHTML_out.html");
    doc.Save(outputPath);

    Console.WriteLine("\nOneNote document created successfully.\nFile saved at " + outputPath);
}
```

### الخطوة 3: حفظ كـ HTML إلى Memory Stream مع موارد مدمجة
عندما تريد حمولة HTML ذاتية الاحتواء (مثلاً للاستجابات عبر API)، قم بتضمين CSS والخطوط والصور مباشرة في `MemoryStream`. يتيح لك `HtmlSaveOptions` ضبط `EmbedResources` إلى `true`، مما ينتج سلسلة HTML مضغوطة.

```csharp
public static void CreateAndSavePageRange()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateAndSavePageRange_out.html");
    doc.Save(outputPath, new HtmlSaveOptions { PageCount = 1, PageIndex = 0 });

    Console.WriteLine("\nOneNote document created successfully and saved as page range.\nFile saved at " + outputPath);
}
```

### الخطوة 4: حفظ كـ HTML إلى ملفات مع موارد في ملفات منفصلة
إذا كنت تفضّل إخراج مجلد ويب تقليدي—ملف HTML ومجلد للملفات الثابتة—عطل التضمين وحدد مجلد الإخراج. ينتج هذا النهج ملفات مصدر أنظف ويسهل التخزين المؤقت للصور أو أوراق الأنماط.

```csharp
public static void SaveAsHTMLToMemoryStreamWithEmbeddedResources()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportEmbedded,
        ExportFonts = ResourceExportType.ExportEmbedded,
        ExportImages = ResourceExportType.ExportEmbedded,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to a memory stream
    var memoryStream = new MemoryStream();
    document.Save(memoryStream, options);
}
```

### الخطوة 5: حفظ كـ HTML باستخدام ردود نداء لإدارة الموارد
السيناريوهات المتقدمة، مثل تخزين الصور في قاعدة بيانات أو سحابة، تتطلب معالجة مخصصة. من خلال توفير تفويض `ResourceSavingCallback`، تتحكم تمامًا في طريقة حفظ كل مورد، بينما يواصل الـ API إنشاء مراجع HTML صحيحة.

```csharp
public static void SaveAsHTMLToFileWithResourcesInSeparateFiles()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportAsStream,
        ExportFonts = ResourceExportType.ExportAsStream,
        ExportImages = ResourceExportType.ExportAsStream,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to HTML file with resources stored in separate files
    document.Save(Path.Combine(dataDir, "document_out.html"), options);
}
```

## المشكلات الشائعة والحلول

| **الصور تظهر مكسورة** | تم حفظ الموارد في مجلد مختلف عن ملف HTML. | تأكد من أن `HtmlSaveOptions.ResourcesFolder` يطابق المجلد الذي تشير إليه، أو فعّل `EmbedResources`. |
| **أنماط CSS مفقودة** | تم تعطيل إنشاء CSS الافتراضي. | عيّن `HtmlSaveOptions.GenerateCss = true` (الإعداد الافتراضي) أو دمج CSS يدويًا. |
| **دفاتر ملاحظات كبيرة تسبب نفاد الذاكرة** | استخدام مُنشئ `Document` غير المتدفق يحمل الملف بالكامل. | استخدم `Document.Load(stream, LoadOptions)` مع `LoadOptions.LoadFromMemory = true` لتدفق الصفحات. |
| **مشكلات ترميز HTML للأحرف غير ASCII** | تم تعيين `Encoding` غير صحيح على `HtmlSaveOptions`. | عيّن `HtmlSaveOptions.Encoding = Encoding.UTF8`. |

## الخاتمة

باتباع الأنماط المذكورة أعلاه، يمكنك **حفظ OneNote كـ HTML** بسطر واحد من الكود أو ضبط التصدير لتلبية متطلبات مؤسسية معقدة. سواء كنت تحتاج إلى لقطة سريعة لصفحة واحدة، دفتر ملاحظات كامل للنشر على الويب، أو خط أنابيب موارد مخصص، فإن Aspose.Note يوفر المرونة والأداء والموثوقية عبر المنصات التي تحتاجها. دمج هذه المقاطع في حلول .NET الحالية سيمكنك من تقديم محتوى OneNote لأي مستهلك يعتمد على المتصفح دون عبء عميل OneNote الأصلي.

```csharp
public static void SaveAsHTMLToMemoryStreamWithCallBacksToSaveResources()
{
    // Specify the saving callbacks configuration
    var savingCallbacks = new UserSavingCallbacks()
    {
        RootFolder = "documentFolder",
        CssFolder = "css",
        KeepCssStreamOpened = true,
        ImagesFolder = "images",
        FontsFolder = "fonts"
    };

    // Specify HTML saving options
    var options = new HtmlSaveOptions
    {
        FontFaceTypes = FontFaceType.Ttf,
        CssSavingCallback = savingCallbacks,
        FontSavingCallback = savingCallbacks,
        ImageSavingCallback = savingCallbacks
    };

    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Save the document to HTML format with resources managed by user-defined callbacks
    using (var stream = File.Create(Path.Combine(savingCallbacks.RootFolder, "document.html")))
    {
        document.Save(stream, options);
    }

    // Manually append data to the CSS stream
    using (var writer = new StreamWriter(savingCallbacks.CssStream))
    {
        writer.WriteLine();
        writer.WriteLine("/* This line is appended to stream manually by user */");
    }
}
```
