# Simpan OneNote sebagai HTML menggunakan Aspose.Note

## Pendahuluan

Jika Anda perlu **menyimpan OneNote sebagai HTML** untuk penerbitan web, pelaporan, atau pengarsipan, Aspose.Note untuk .NET memberikan solusi bersih yang hanya menggunakan kode dan berfungsi di Windows, Linux, serta macOS tanpa memerlukan Microsoft Office. Dalam tutorial ini Anda akan belajar cara membuat dokumen OneNote dari awal, memanipulasi halamannya, dan mengekspor seluruh notebook—atau halaman yang dipilih—ke HTML menggunakan berbagai opsi. Panduan ini menuntun Anda melalui setiap langkah, menjelaskan mengapa setiap pengaturan penting, dan menampilkan skenario dunia nyata di mana ekspor HTML bersinar, seperti menyematkan konten OneNote dalam portal basis pengetahuan atau menghasilkan dokumentasi statis.

## Jawaban Cepat
- **Apa metode utama untuk mengekspor?** Panggil `document.Save("output.html", SaveFormat.Html)` pada instance `Aspose.Note.Document`.  
- **Apakah saya dapat mengekspor satu halaman?** Ya—gunakan `document.Save(pageRange, "page.html", SaveFormat.Html)`.  
- **Apakah sumber daya (gambar, CSS) disematkan secara otomatis?** Secara default mereka disimpan sebagai file terpisah; Anda dapat menyematkannya ke dalam stream memori dengan `HtmlSaveOptions`.  
- **Apakah .NET Core didukung?** Tentu—Aspose.Note bekerja dengan .NET 5, .NET 6, dan .NET Core 3.1+.  
- **Apakah saya memerlukan lisensi untuk pengembangan?** Versi percobaan gratis 30‑hari dapat digunakan untuk pengujian; lisensi menghilangkan watermark evaluasi.

## Apa itu “save onenote as html”?

**“Save OneNote as HTML”** mengacu pada proses mengonversi notebook OneNote, yang disimpan dalam format biner proprietari, menjadi file HTML standar yang dapat dirender oleh browser web mana pun. Konversi ini mempertahankan struktur hierarki notebook, pemformatan teks, gambar, tabel, dan sumber daya lainnya, sambil menerjemahkan elemen khusus OneNote ke dalam tag HTML dan gaya CSS yang setara. Hal ini memungkinkan pengembang menampilkan konten OneNote di situs web, mengintegrasikannya ke dalam sistem manajemen konten, atau mengarsipkannya dalam format yang dapat diakses secara universal tanpa memerlukan klien OneNote.

## Mengapa mengonversi OneNote ke HTML?

Aspose.Note dapat mengekspor ke **lebih dari 50 format output** (termasuk PDF, DOCX, PNG, dan HTML) dan memproses notebook dengan **hingga 500 halaman** tanpa memuat seluruh file ke memori, berkat arsitektur streaming‑nya. Mengonversi ke HTML memungkinkan Anda:

1. **Menerbitkan konten di web** tanpa memerlukan klien OneNote.  
2. **Mengintegrasikan dengan sistem manajemen konten** yang hanya menerima HTML.  
3. **Mengekstrak teks yang dapat dicari** untuk mesin indeks.  
4. **Menggunakan kembali sumber daya tingkat halaman** (gambar, font) dalam proyek web lainnya.

## Cara menyimpan onenote sebagai html – Langkah demi Langkah

Menyimpan OneNote sebagai HTML melibatkan tiga tindakan inti: memuat atau membuat notebook, mengonfigurasi opsi ekspor yang diinginkan, dan memanggil metode `Save` yang tepat. Pertama, Anda membuat atau memuat objek `Document`, kemudian dapat menyesuaikan `HtmlSaveOptions` untuk mengontrol penanganan sumber daya, pembuatan CSS, dan pengkodean. Akhirnya, panggil `document.Save` dengan jalur file atau stream, opsional menyertakan `PageRange` untuk membatasi ekspor. Pendekatan sistematis ini memastikan hasil yang konsisten di berbagai lingkungan dan memungkinkan Anda menyesuaikan output sesuai kebutuhan proyek.

### Langkah 1: Buat Dokumen OneNote dengan Opsi Default

Dalam contoh pertama ini kami menginstansiasi `Document` baru, menambahkan satu halaman dengan judul, dan menyimpan seluruh notebook ke HTML menggunakan `HtmlSaveOptions` default. Tidak ada parameter tambahan yang diperlukan, menjadikannya cara tercepat untuk **menyimpan OneNote sebagai HTML**.

```csharp
using System;
using System.Drawing;
using System.Globalization;
using System.IO;

using Aspose.Note.Saving;
using Aspose.Note.Saving.Html;

```

### Langkah 2: Ekspor Rentang Halaman Tertentu ke HTML

Seringkali Anda hanya membutuhkan subset halaman—misalnya, mengekspor satu bab catatan rapat. Dengan melewatkan objek `PageRange` ke `Save`, Anda dapat menargetkan halaman 1‑3 sementara yang lainnya tetap tidak tersentuh.

```csharp
public static void CreateAndSaveToHTMLUsingDefaultOptions()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateOneNoteDocAndSaveToHTML_out.html");
    doc.Save(outputPath);

    Console.WriteLine("\nOneNote document created successfully.\nFile saved at " + outputPath);
}
```

### Langkah 3: Simpan sebagai HTML ke Memory Stream dengan Sumber Daya yang Disematkan

Ketika Anda menginginkan satu payload HTML yang berdiri sendiri (mis., untuk respons API), sematkan CSS, font, dan gambar langsung ke dalam `MemoryStream`. `HtmlSaveOptions` memungkinkan Anda mengaktifkan `EmbedResources` menjadi `true`, menghasilkan string HTML yang kompak.

```csharp
public static void CreateAndSavePageRange()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateAndSavePageRange_out.html");
    doc.Save(outputPath, new HtmlSaveOptions { PageCount = 1, PageIndex = 0 });

    Console.WriteLine("\nOneNote document created successfully and saved as page range.\nFile saved at " + outputPath);
}
```

### Langkah 4: Simpan sebagai HTML ke File dengan Sumber Daya dalam File Terpisah

Jika Anda lebih suka output folder web klasik—file HTML plus folder aset—nonaktifkan penyematan dan tentukan direktori output. Pendekatan ini menghasilkan file sumber yang lebih bersih dan memudahkan caching gambar atau stylesheet.

```csharp
public static void SaveAsHTMLToMemoryStreamWithEmbeddedResources()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportEmbedded,
        ExportFonts = ResourceExportType.ExportEmbedded,
        ExportImages = ResourceExportType.ExportEmbedded,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to a memory stream
    var memoryStream = new MemoryStream();
    document.Save(memoryStream, options);
}
```

### Langkah 5: Simpan sebagai HTML Menggunakan Callback untuk Mengelola Sumber Daya

Skenario lanjutan, seperti menyimpan gambar di basis data atau bucket cloud, memerlukan penanganan khusus. Dengan menyediakan delegasi `ResourceSavingCallback`, Anda mengontrol cara setiap sumber daya disimpan, sementara API tetap menghasilkan referensi HTML yang tepat.

```csharp
public static void SaveAsHTMLToFileWithResourcesInSeparateFiles()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportAsStream,
        ExportFonts = ResourceExportType.ExportAsStream,
        ExportImages = ResourceExportType.ExportAsStream,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to HTML file with resources stored in separate files
    document.Save(Path.Combine(dataDir, "document_out.html"), options);
}
```

## Masalah Umum dan Solusinya

| Masalah | Mengapa Terjadi | Solusi |
|-------|----------------|-----|
| **Images appear broken** | Resources were saved to a different folder than the HTML file. | Ensure `HtmlSaveOptions.ResourcesFolder` matches the folder you reference, or enable `EmbedResources`. |
| **CSS styles are missing** | Default CSS generation was disabled. | Set `HtmlSaveOptions.GenerateCss = true` (default) or embed CSS manually. |
| **Large notebooks cause OutOfMemory** | Using the non‑streaming `Document` constructor loads the whole file. | Use `Document.Load(stream, LoadOptions)` with `LoadOptions.LoadFromMemory = true` to stream pages. |
| **HTML encoding issues for non‑ASCII characters** | Wrong `Encoding` set on `HtmlSaveOptions`. | Set `HtmlSaveOptions.Encoding = Encoding.UTF8`. |
| **Saving fails on Linux** | Missing native dependencies for image processing. | Install `libgdiplus` on the Linux host. |

```csharp
public static void SaveAsHTMLToMemoryStreamWithCallBacksToSaveResources()
{
    // Specify the saving callbacks configuration
    var savingCallbacks = new UserSavingCallbacks()
    {
        RootFolder = "documentFolder",
        CssFolder = "css",
        KeepCssStreamOpened = true,
        ImagesFolder = "images",
        FontsFolder = "fonts"
    };

    // Specify HTML saving options
    var options = new HtmlSaveOptions
    {
        FontFaceTypes = FontFaceType.Ttf,
        CssSavingCallback = savingCallbacks,
        FontSavingCallback = savingCallbacks,
        ImageSavingCallback = savingCallbacks
    };

    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Save the document to HTML format with resources managed by user-defined callbacks
    using (var stream = File.Create(Path.Combine(savingCallbacks.RootFolder, "document.html")))
    {
        document.Save(stream, options);
    }

    // Manually append data to the CSS stream
    using (var writer = new StreamWriter(savingCallbacks.CssStream))
    {
        writer.WriteLine();
        writer.WriteLine("/* This line is appended to stream manually by user */");
    }
}
```
