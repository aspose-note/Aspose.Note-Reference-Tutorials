## Cara membuat tabel OneNote secara programatik?

`Document` mewakili file notebook OneNote yang dimuat ke memori.  
`Page.getTables().add()` membuat Table baru pada halaman dengan baris dan kolom yang ditentukan.

Muat instance `Document`, tambahkan `Page`, lalu panggil `Page.getTables().add()` dengan jumlah baris dan kolom yang diinginkan. Setelah dibuat Anda dapat mengatur nilai sel, menerapkan border, dan mengatur gaya tabel—semua melalui panggilan Java yang fluently. Pola dua langkah ini (buat → konfigurasi) memungkinkan Anda membangun tabel secara instan, bahkan untuk notebook multi‑halaman.
