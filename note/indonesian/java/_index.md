## Pertanyaan yang Sering Diajukan

**Q: Bisakah saya menggunakan Aspose.Note untuk Java dalam proyek komersial?**  
A: Ya. Lisensi komersial yang sah diperlukan untuk penggunaan produksi, tetapi versi percobaan gratis tersedia untuk evaluasi.

**Q: Versi Java mana yang didukung?**  
A: Perpustakaan mendukung Java 8, 11, dan rilis LTS yang lebih baru.

**Q: Bagaimana cara menambahkan hyperlink ke halaman OneNote?**  
A: Gunakan kelas `Hyperlink` yang disediakan oleh Aspose.Note untuk mendefinisikan URL dan melampirkannya ke node `RichText`.

**Q: Apakah memungkinkan untuk mengatur teks alternatif untuk gambar demi aksesibilitas?**  
A: Tentu saja. Objek `Image` memiliki properti `AltText` yang dapat Anda atur secara programatis.

**Q: Apa saja tip kinerja untuk notebook besar?**  
A: Aktifkan lisensi metered, gunakan kembali instance `Document` bila memungkinkan, dan alirkan sumber daya besar alih-alih memuatnya sepenuhnya ke memori.
