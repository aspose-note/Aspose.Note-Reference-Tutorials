# OneNote में इमेज में Alt Text कैसे जोड़ें Java का उपयोग करके

## परिचय

इस ट्यूटोरियल में, आप **कैसे इमेज में alt** टेक्स्ट जोड़ें OneNote दस्तावेज़ों में Java और Aspose.Note API का उपयोग करके, यह जानेंगे। वैकल्पिक टेक्स्ट (alt text) जोड़ने से स्क्रीन‑रीडर उपयोगकर्ताओं के लिए पहुँच में सुधार होता है और आपके कंटेंट की समग्र समावेशिता बढ़ती है। इस गाइड के अंत तक, आप Java में इमेज का alt टेक्स्ट सेट कर सकेंगे, जिससे आपके OneNote फ़ाइलें पहुँच मानकों के अधिक अनुरूप बन जाएँगी।

## त्वरित उत्तर
- **कौन सी लाइब्रेरी आवश्यक है?** Aspose.Note for Java  
- **इस ट्यूटोरियल का मुख्य कीवर्ड क्या है?** how to add alt  
- **क्या उत्पादन के लिए लाइसेंस चाहिए?** हाँ, एक व्यावसायिक लाइसेंस आवश्यक है (एक मुफ्त ट्रायल उपलब्ध है)।  
- **क्या मैं कई इमेज में alt टेक्स्ट जोड़ सकता हूँ?** बिल्कुल – प्रत्येक इमेज के लिए चरण दोहराएँ।  
- **कौन सा Java संस्करण समर्थित है?** Java 8 या उससे ऊपर।

## OneNote के संदर्भ में “how to add alt” क्या है?

Alt टेक्स्ट जोड़ना मतलब इमेज के लिए एक पाठ्य विवरण प्रदान करना है जिसे सहायक तकनीकों द्वारा पढ़ा जा सके। यह विवरण उन उपयोगकर्ताओं को मदद करता है जो इमेज नहीं देख सकते, उसकी उद्देश्य को समझने में।

## OneNote में इमेज में alt टेक्स्ट क्यों जोड़ें?

- **पहुँच:** WCAG दिशानिर्देशों को पूरा करता है और दृश्य बाधित उपयोगकर्ताओं के अनुभव को बेहतर बनाता है।  
- **खोजयोग्यता:** सर्च इंजन विवरण को इंडेक्स कर सकते हैं, जिससे आपका कंटेंट अधिक खोज योग्य बनता है।  
- **पेशेवरता:** समावेशी डिज़ाइन के प्रति प्रतिबद्धता दर्शाता है।

## पूर्वापेक्षाएँ

ट्यूटोरियल शुरू करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित पूर्वापेक्षाएँ हैं:

1. Java Development Kit (JDK) – संस्करण 8 या बाद का।  
2. Aspose.Note for Java लाइब्रेरी – इसे [here](https://releases.aspose.com/note/java/) से डाउनलोड करें।  
3. IntelliJ IDEA या Eclipse जैसे IDE।  
4. Java प्रोग्रामिंग का बुनियादी ज्ञान।

## पैकेज इम्पोर्ट करें

पहले, आवश्यक पैकेज को अपने Java प्रोजेक्ट में इम्पोर्ट करें ताकि आप Aspose.Note की कार्यक्षमताओं का उपयोग कर सकें।

```java
import java.io.IOException;

import com.aspose.note.Document;
import com.aspose.note.Image;
import com.aspose.note.Page;
```

अब, Java और Aspose.Note का उपयोग करके OneNote दस्तावेज़ में **वैकल्पिक टेक्स्ट इमेज** जोड़ने की प्रक्रिया को चरण‑दर‑चरण समझते हैं।

## OneNote में इमेज में Alt Text कैसे जोड़ें Java का उपयोग करके

### चरण 1: दस्तावेज़ डायरेक्टरी सेट करें

```java
String dataDir = "Your Document Directory";
```

`"Your Document Directory"` को उस फ़ोल्डर के पाथ से बदलें जिसमें आपका स्रोत इमेज है और जहाँ आउटपुट फ़ाइल सहेजी जाएगी।

### चरण 2: एक Document ऑब्जेक्ट बनाएं

```java
Document document = new Document();
```

यह एक नया, खाली OneNote दस्तावेज़ बनाता है।

### चरण 3: एक Page ऑब्जेक्ट बनाएं

```java
Page page = new Page();
```

एक पेज वह होस्ट होगा जहाँ हम इमेज जोड़ने वाले हैं।

### चरण 4: पेज में इमेज जोड़ें

```java
Image image = new Image(null, dataDir + "image.jpg");
```

`Image` कंस्ट्रक्टर निर्दिष्ट पाथ से इमेज फ़ाइल लोड करता है।

### चरण 5: वैकल्पिक टेक्स्ट शीर्षक सेट करें

```java
image.setAlternativeTextTitle("ImageAlternativeText Title");
```

यहाँ हम **alt टेक्स्ट** जोड़ते हैं जो इमेज के लिए एक संक्षिप्त शीर्षक के रूप में कार्य करता है।

### चरण 6: वैकल्पिक टेक्स्ट विवरण सेट करें

```java
image.setAlternativeTextDescription("ImageAlternativeText Description");
```

यह विवरण अधिक विस्तृत व्याख्या प्रदान करता है—स्क्रीन रीडर्स के लिए आदर्श।

### चरण 7: इमेज को पेज में जोड़ें

```java
page.appendChildLast(image);
```

इमेज (अब alt टेक्स्ट के साथ समृद्ध) पेज में जोड़ी गई है।

### चरण 8: पेज को दस्तावेज़ में जोड़ें

```java
document.appendChildLast(page);
```

पेज को OneNote दस्तावेज़ से संलग्न करें।

### चरण 9: दस्तावेज़ सहेजें

```java
document.save(dataDir + "AlternativeText_out.one");
```

दस्तावेज़ को इमेज में एम्बेडेड वैकल्पिक टेक्स्ट के साथ सहेजा जाता है।

## सामान्य समस्याएँ और समाधान

- **FileNotFoundException:** सुनिश्चित करें कि `dataDir` सही फ़ोल्डर की ओर इशारा कर रहा है और `image.jpg` मौजूद है।  
- **NullPointerException on image:** इमेज पाथ वैध है और फ़ाइल भ्रष्ट नहीं है, यह सुनिश्चित करें।  
- **License errors:** एक वैध Aspose.Note लाइसेंस फ़ाइल का उपयोग करें या मूल्यांकन के लिए ट्रायल मोड में चलाएँ।

## अक्सर पूछे जाने वाले प्रश्न

**प्रश्न: क्या मैं एक ही दस्तावेज़ में कई इमेज में alt टेक्स्ट जोड़ सकता हूँ?**  
उत्तर: हाँ, प्रत्येक इमेज के लिए चरण 4‑6 दोहराएँ।

**प्रश्न: कौन से इमेज फ़ॉर्मेट्स में alt टेक्स्ट जोड़ना समर्थित है?**  
उत्तर: Aspose.Note JPEG, PNG, GIF, BMP और कई अन्य सामान्य फ़ॉर्मेट्स को सपोर्ट करता है।

**प्रश्न: क्या सेट किए गए alt टेक्स्ट को बाद में संशोधित या हटाया जा सकता है?**  
उत्तर: बिल्कुल। `setAlternativeTextTitle("")` या `setAlternativeTextDescription("")` को कॉल करके मान साफ़ करें, या नई स्ट्रिंग्स प्रदान करके अपडेट करें।

**प्रश्न: क्या Aspose.Note अन्य भाषाओं के लिए भी API प्रदान करता है?**  
उत्तर: हाँ, लाइब्रेरी .NET, C++ और Python के लिए भी उपलब्ध है।

**प्रश्न: मैं Aspose.Note का ट्रायल संस्करण कहाँ डाउनलोड कर सकता हूँ?**  
उत्तर: आप मुफ्त ट्रायल [here](https://releases.aspose.com/) से प्राप्त कर सकते हैं।

## निष्कर्ष

इस चरण‑दर‑चरण गाइड का पालन करके, अब आप **कैसे इमेज में alt** टेक्स्ट जोड़ें OneNote में Java का उपयोग करके, जानते हैं। `add alternative text image` को लागू करने से आपके दस्तावेज़ अधिक सुलभ ही नहीं, बल्कि उनकी खोजयोग्यता और समग्र गुणवत्ता भी बढ़ती है। विभिन्न शीर्षकों और विवरणों के साथ प्रयोग करने में संकोच न करें ताकि प्रत्येक इमेज का अर्थ सर्वोत्तम रूप से व्यक्त हो सके।
