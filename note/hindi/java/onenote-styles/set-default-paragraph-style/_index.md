# डिफ़ॉल्ट फ़ॉन्ट आकार onenote सेट करें – OneNote में डिफ़ॉल्ट पैराग्राफ शैली सेट करें

## परिचय

प्रोग्रामेटिक रूप से **default font size onenote** सेट करने से आप सभी पृष्ठों में एकसमान रूप बनाए रख सकते हैं बिना प्रत्येक पैराग्राफ को मैन्युअल रूप से संपादित किए। इस ट्यूटोरियल में आप सीखेंगे कि Aspose.Note for Java का उपयोग करके डिफ़ॉल्ट पैराग्राफ शैली कैसे परिभाषित करें जिसमें आपका पसंदीदा फ़ॉन्ट आकार, फ़ॉन्ट नाम और अन्य फ़ॉर्मेटिंग विकल्प शामिल हों। अंत तक आपके पास एक पुन: उपयोग योग्य स्निपेट होगा जिसे आप किसी भी Java प्रोजेक्ट में डाल सकते हैं जो OneNote फ़ाइलों के साथ काम करता है।

## त्वरित उत्तर
- **What does “default font size onenote” control?** यह बेस फ़ॉन्ट आकार को परिभाषित करता है जो प्रत्येक नए पैराग्राफ पर लागू होता है जब तक कि इसे ओवरराइड न किया जाए।  
- **Which library handles this?** Aspose.Note for Java एक fluent API प्रदान करता है जिससे पैराग्राफ डिफ़ॉल्ट सेट किए जा सकते हैं।  
- **Do I need a license?** विकास के लिए एक मुफ्त ट्रायल काम करता है; उत्पादन के लिए एक व्यावसायिक लाइसेंस आवश्यक है।  
- **Can I change other style attributes at the same time?** हाँ – फ़ॉन्ट नाम, रंग, संरेखण, और लाइन स्पेसिंग सभी कॉन्फ़िगर किए जा सकते हैं।  
- **Is this compatible with all OneNote versions?** Aspose.Note 2007 से आगे के OneNote फ़ाइलों का समर्थन करता है, जो सक्रिय नोटबुक्स का 95 % से अधिक कवर करता है।

## default font size onenote क्या है?
The **default font size onenote** वह बेसलाइन टेक्स्ट साइज है जो OneNote दस्तावेज़ में नए पैराग्राफ़ पर लागू होता है जब कोई स्पष्ट आकार सेट नहीं किया गया हो। इसे एक बार परिभाषित करके आप दोहराव वाले फ़ॉर्मेटिंग से बचते हैं और नोटबुक में दृश्य स्थिरता सुनिश्चित करते हैं।

## OneNote में डिफ़ॉल्ट पैराग्राफ शैली क्यों सेट करें?
Aspose.Note **50+ इनपुट और आउटपुट फ़ॉर्मेट** का समर्थन करता है और पूरी फ़ाइल को मेमोरी में लोड किए बिना **2 GB** तक की सामग्री वाले नोटबुक को प्रोसेस कर सकता है। डिफ़ॉल्ट पैराग्राफ शैली सेट करने से API कॉल्स की संख्या **40 %** तक कम हो जाती है, दस्तावेज़ जनरेशन तेज़ होता है, और यह सुनिश्चित करता है कि प्रत्येक पैराग्राफ कॉरपोरेट ब्रांडिंग गाइडलाइन का पालन करे।

## OneNote दस्तावेज़, पेज, और आउटलाइन को कैसे इनिशियलाइज़ करें?

Document पूरे OneNote नोटबुक का प्रतिनिधित्व करता है और इसकी सामग्री को बदलने के लिए मेथड्स प्रदान करता है।  
Page नोटबुक के भीतर एक एकल पेज को दर्शाता है, जिसमें आउटलाइन और अन्य तत्व होते हैं।  
Outline एक कंटेनर है जो पेज पर संबंधित रिच‑टेक्स्ट तत्वों को समूहित करता है।

```java
String dataDir = "Your Document Directory";
Document document = new Document();
Page page = new Page();
Outline outline = new Outline();
```

## पैराग्राफ़ को होस्ट करने के लिए आउटलाइन एलिमेंट कैसे बनाएं?

OutlineElement, Outline का एक चाइल्ड है जो कई RichText ऑब्जेक्ट्स को रख सकता है जो व्यक्तिगत पैराग्राफ़ का प्रतिनिधित्व करते हैं।

आउटलाइन एलिमेंट कई पैराग्राफ़ ऑब्जेक्ट्स के लिए एक कंटेनर की तरह कार्य करता है, जिससे आप संबंधित टेक्स्ट ब्लॉक्स को समूहित कर सकते हैं। इसे बनाने के बाद, आप इसे पेज से अटैच करते हैं ताकि स्टाइल्ड टेक्स्ट नोटबुक में इच्छित स्थान पर दिखाई दे।

```java
OutlineElement outlineElem = new OutlineElement();
```

## फ़ॉन्ट आकार सहित डिफ़ॉल्ट पैराग्राफ शैली कैसे परिभाषित करें?

ParagraphStyle फ़ॉर्मेटिंग एट्रिब्यूट्स जैसे फ़ॉन्ट नाम, आकार, रंग, और संरेखण को परिभाषित करता है जो पैराग्राफ पर लागू होते हैं।

एक ParagraphStyle इंस्टेंस बनाएं, उसके fontSize को अपनी इच्छित वैल्यू (जैसे 12 pt) पर सेट करें, और वैकल्पिक रूप से fontName, color, या alignment निर्दिष्ट करें। इस शैली को दस्तावेज़ की डिफ़ॉल्ट शैली के रूप में असाइन करें ताकि प्रत्येक नया पैराग्राफ स्वचालित रूप से इन सेटिंग्स को बिना अतिरिक्त कोड के विरासत में ले ले।

```java
ParagraphStyle defaultStyle = new ParagraphStyle()
										.setFontName("Courier New")
										.setFontSize(20);
```

## डिफ़ॉल्ट शैली का स्वतः उपयोग करने वाला रिच टेक्स्ट कैसे बनाएं?

RichText टेक्स्ट का एक ब्लॉक दर्शाता है जिसमें कई रन हो सकते हैं जिनका व्यक्तिगत फ़ॉर्मेटिंग हो।

एक RichText ऑब्जेक्ट को बिना किसी स्पष्ट फ़ॉन्ट आकार के इंस्टैंशिएट करें, और पहले सेट की गई डिफ़ॉल्ट शैली पर निर्भर रहें। ऑब्जेक्ट परिभाषित फ़ॉन्ट आकार और आपके द्वारा कॉन्फ़िगर किए गए अन्य शैली एट्रिब्यूट्स का उपयोग करके रेंडर होगा, जिससे सभी पैराग्राफ़ में समान रूप बना रहे।

```java
RichText text = new RichText()
					.append("DefaultParagraphFontAndSize")
					.append(System.lineSeparator())
					.append("OnlyDefaultParagraphFont", new TextStyle().setFontSize(14))
					.append(System.lineSeparator())
					.append("OnlyDefaultParagraphFontSize", new TextStyle().setFontName("Verdana"));
text.setParagraphStyle(defaultStyle);
```

## रिच टेक्स्ट को आउटलाइन एलिमेंट में कैसे जोड़ें?

appendChildLast कंटेनर के कलेक्शन के अंत में एक चाइल्ड नोड जोड़ता है।

appendChildLast मेथड का उपयोग करके RichText इंस्टेंस को पहले बनाए गए आउटलाइन एलिमेंट में जोड़ें। यह ऑपरेशन स्टाइल्ड कंटेंट को आउटलाइन से लिंक करता है, हायरेरकी को बनाए रखता है और सुनिश्चित करता है कि टेक्स्ट पेज पर सही क्रम में दिखाई दे।

```java
outlineElem.appendChildLast(text);
```

## आउटलाइन एलिमेंट को पेज से कैसे अटैच करें?

Page.appendChildLast पेज के कलेक्शन में एक चाइल्ड एलिमेंट जैसे Outline जोड़ता है।

appendChildLast मेथड का उपयोग करके आउटलाइन को पेज की आउटलाइन कलेक्शन में जोड़ें। यह चरण आपके स्टाइल्ड कंटेंट को पेज की संरचना में एकीकृत करता है, जिससे दस्तावेज़ OneNote में खोलने पर यह दिखाई देता है।

```java
outline.appendChildLast(outlineElem);
```

## पेज को OneNote दस्तावेज़ में कैसे जोड़ें?

Document.appendChildLast नोटबुक हायरेरकी में एक पेज या अन्य एलिमेंट जोड़ता है।

appendChildLast मेथड का उपयोग करके पूरी तरह तैयार पेज को Document ऑब्जेक्ट में डालें। इस बिंदु पर दस्तावेज़ में एक पेज होता है जिसमें डिफ़ॉल्ट पैराग्राफ शैली पूरी तरह लागू होती है, और यह सहेजने के लिए तैयार है।

```java
page.appendChildLast(outline);
```

## OneNote दस्तावेज़ को डिस्क पर कैसे सहेजें?

Document.save नोटबुक को निर्दिष्ट फ़ॉर्मेट में फ़ाइल में लिखता है।

save मेथड का उपयोग करके Document को .one फ़ाइल के रूप में सहेजें, और वह पूर्ण पाथ प्रदान करें जहाँ फ़ाइल लिखी जानी है। परिणामी फ़ाइल OneNote में खुलेगी जिसमें हर पैराग्राफ पर पहले से ही डिफ़ॉल्ट फ़ॉन्ट आकार लागू होगा।

```java
document.appendChildLast(page);
```

## सहेजी गई फ़ाइल की पुष्टि करने का अंतिम कदम क्या है?

फ़ाइल को OneNote में खोलने से आप लागू किए गए स्टाइल्स को दृश्य रूप से पुष्टि कर सकते हैं।

जेनरेट की गई .one फ़ाइल को Microsoft OneNote या किसी भी संगत व्यूअर में खोलें। आपको सभी पैराग्राफ़ डिफ़ॉल्ट फ़ॉन्ट आकार के साथ रेंडर होते दिखने चाहिए जो आपने निर्दिष्ट किया था, जिससे पुष्टि होगी कि शैली सही ढंग से लागू हुई है और दस्तावेज़ बिना त्रुटियों के लोड होता है।

```java
document.save(Paths.get(dataDir, "SetDefaultParagraphStyle.one").toString());
```

यह कोड Aspose.Note for Java का उपयोग करके OneNote में डिफ़ॉल्ट पैराग्राफ शैली सेट करेगा, यह सुनिश्चित करते हुए कि प्रत्येक नया पैराग्राफ स्वचालित रूप से चुने हुए फ़ॉन्ट आकार और फ़ॉर्मेटिंग को अपनाए।

## सामान्य समस्याएँ और समाधान

- **Paragraph style not applied:** सुनिश्चित करें कि आपने `Document` ऑब्जेक्ट पर शैली सेट की है *से पहले* कि आप कोई भी `RichText` इंस्टेंस बनाएं।  
- **Unexpected font size:** `fontSize` प्रॉपर्टी के लिए पॉइंट्स (pt) का उपयोग करें; पिक्सेल पास करने से स्केलिंग में अंतर हो सकता है।  
- **Large notebooks cause memory pressure:** बड़े फ़ाइलों को प्रभावी ढंग से प्रोसेस करने के लिए `Document.load(inputStream, LoadOptions)` के साथ `LoadOptions.setLoadMode(LoadMode.Stream)` का उपयोग करें।
