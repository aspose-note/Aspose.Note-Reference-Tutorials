# नोटबुक को फ्लैटन कैसे करें – OneNote को PNG में बदलें

## परिचय

इस ट्यूटोरियल में आप **नोटबुक को फ्लैटन करने** की प्रक्रिया सीखेंगे, जिसमें OneNote नोटबुक को Aspose.Note for Java का उपयोग करके उच्च‑गुणवत्ता वाले PNG इमेज में बदलेंगे। चाहे आपको रिपोर्ट में नोटबुक का स्नैपशॉट एम्बेड करना हो, सहयोगियों के साथ केवल‑पढ़ने योग्य दृश्य साझा करना हो, या अनुपालन योग्य इमेज को आर्काइव करना हो, यह चरण‑दर‑चरण गाइड आपको दिखाएगा कि कैसे OneNote को इमेज के रूप में सेव करें जबकि रिज़ॉल्यूशन को नियंत्रित किया जा सके।

## त्वरित उत्तर
- **“flatten notebook” का क्या अर्थ है?** यह सभी पेज तत्वों को एक रास्टर इमेज में मिलाता है।  
- **कौन सा फ़ॉर्मेट उपयोग किया जाता है?** डिफ़ॉल्ट रूप से PNG उपयोग होता है, जो लॉस‑लेस क्वालिटी देता है।  
- **क्या मैं DPI बदल सकता हूँ?** हाँ, `ImageSaveOptions` पर `setResolution` का उपयोग करें।  
- **क्या मुझे लाइसेंस चाहिए?** मूल्यांकन के लिए एक फ्री ट्रायल काम करता है; उत्पादन के लिए एक व्यावसायिक लाइसेंस आवश्यक है।  
- **क्या यह सभी OS पर समर्थित है?** Java API जहाँ भी Java चलता है, वहाँ चलती है।

## OneNote को PNG में बदलना क्या है?

OneNote को PNG में बदलने से नोटबुक के प्रत्येक पेज की बिटमैप प्रतिनिधित्व बनता है, जिसमें टेक्स्ट, ड्रॉइंग और एम्बेडेड ऑब्जेक्ट्स एक ही इमेज फ़ाइल में संरक्षित रहते हैं। यह विशेष रूप से दस्तावेज़ीकरण, प्रस्तुतियों, या अनुपालन आर्काइव के लिए उपयोगी है।

## Aspose.Note के साथ OneNote को PNG में क्यों बदलें?

- **पूर्ण फ़िडेलिटी** – सभी दृश्य तत्व बरकरार रहते हैं।  
- **सिंगल‑फ़ाइल आउटपुट** – कई पेज फ़ाइलों का प्रबंधन नहीं करना पड़ता।  
- **कस्टमाइज़ेबल रिज़ॉल्यूशन** – गुणवत्ता आवश्यकताओं को पूरा करने के लिए DPI समायोजित करें।  
- **ऑफ़िस इंस्टॉलेशन नहीं आवश्यक** – Microsoft OneNote से पूरी तरह स्वतंत्र रूप से काम करता है।

## पूर्वापेक्षाएँ

शुरू करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

1. आपके सिस्टम पर Java Development Kit (JDK) स्थापित हो।  
2. Aspose.Note for Java लाइब्रेरी डाउनलोड करके अपने Java प्रोजेक्ट में सेट अप करें। आप लाइब्रेरी [यहाँ](https://releases.aspose.com/note/java/) से डाउनलोड कर सकते हैं।  
3. Java प्रोग्रामिंग का बुनियादी ज्ञान।

## पैकेज इम्पोर्ट करें

शुरू करने के लिए, आपको Aspose.Note for Java से आवश्यक पैकेज इम्पोर्ट करने होंगे:

```java
import java.io.IOException;
import com.aspose.note.ImageSaveOptions;
import com.aspose.note.Notebook;
import com.aspose.note.NotebookImageSaveOptions;
import com.aspose.note.SaveFormat;
```

## चरण‑दर‑चरण गाइड

### चरण 1: दस्तावेज़ डायरेक्टरी सेट करें

सबसे पहले, वह डायरेक्टरी निर्धारित करें जहाँ आपका नोटबुक फ़ाइल स्थित है:

```java
String dataDir = "Your Document Directory";
```

`"Your Document Directory"` को अपने नोटबुक फ़ाइल के पथ से बदलें।

### चरण 2: नोटबुक लोड करें

अगला, `Notebook` क्लास का उपयोग करके नोटबुक फ़ाइल लोड करें:

```java
Notebook notebook = new Notebook(dataDir + "test.onetoc2");
```

`"test.onetoc2"` को अपनी नोटबुक फ़ाइल के नाम से बदलें।

### चरण 3: इमेज सेव विकल्प सेट करें (set image resolution java)

अब, नोटबुक को इमेज के रूप में सेव करने के विकल्प सेट करें। हम सेव फ़ॉर्मेट PNG निर्धारित करेंगे और रिज़ॉल्यूशन 400 DPI सेट करेंगे:

```java
NotebookImageSaveOptions saveOptions = new NotebookImageSaveOptions(SaveFormat.Png);
ImageSaveOptions documentSaveOptions = saveOptions.getDocumentSaveOptions();
documentSaveOptions.setResolution(400);
```

आप अपनी आवश्यकताओं के अनुसार रिज़ॉल्यूशन समायोजित कर सकते हैं। यहाँ आप आउटपुट क्वालिटी को नियंत्रित करने के लिए **इमेज रिज़ॉल्यूशन सेट** करते हैं।

### चरण 4: नोटबुक को फ्लैटन करें (how to flatten notebook)

सभी तत्वों को एक ही इमेज में फ्लैटन करने के लिए, `flatten` विकल्प को `true` सेट करें:

```java
saveOptions.setFlatten(true);
```

`flatten` को `true` सेट करने से सुनिश्चित होता है कि परिणामी PNG आपके नोटबुक का सटीक लेआउट बनाए रखेगा।

### चरण 5: इमेज सेव करें (save onenote as image)

अंत में, नोटबुक को फ्लैटन इमेज के रूप में सेव करें:

```java
notebook.save(dataDir + "ExportImageasFlattenedNotebook_out.png", saveOptions);
```

`"ExportImageasFlattenedNotebook_out.png"` को अपनी इच्छित आउटपुट फ़ाइल नाम से बदलें। यह चरण **OneNote को इमेज के रूप में सेव** करता है जिसे आप कहीं भी साझा या एम्बेड कर सकते हैं।

## सामान्य उपयोग केस

- **दस्तावेज़ीकरण:** तकनीकी मैनुअल या उपयोगकर्ता गाइड में नोटबुक इमेज एम्बेड करें।  
- **प्रस्तुतियाँ:** PowerPoint में उच्च‑रिज़ॉल्यूशन PNG स्लाइड का उपयोग करें बिना फ़ॉन्ट या ऑब्जेक्ट संगतता की चिंता के।  
- **आर्काइविंग:** अनुपालन ऑडिट के लिए नोटबुक का केवल‑पढ़ने योग्य स्नैपशॉट संग्रहित करें।

## समस्या निवारण टिप्स

- **फ़ाइल नहीं मिली:** `dataDir` पथ को दोबारा जांचें और सुनिश्चित करें कि `.onetoc2` फ़ाइल मौजूद है।  
- **कम गुणवत्ता वाली इमेज:** `documentSaveOptions.setResolution(600);` बदलकर DPI बढ़ाएँ।  
- **घटित तत्व:** सुनिश्चित करें कि `saveOptions.setFlatten(true);` सक्षम है; अन्यथा कुछ लेयर अलग रह सकते हैं।

## अक्सर पूछे जाने वाले प्रश्न

**प्रश्न 1: क्या मैं आउटपुट इमेज का रिज़ॉल्यूशन समायोजित कर सकता हूँ?**  
हाँ, आप `setResolution` मेथड पैरामीटर को बदलकर अपनी आवश्यकताओं के अनुसार रिज़ॉल्यूशन समायोजित कर सकते हैं।

**प्रश्न 2: क्या Aspose.Note निर्यात के लिए अन्य इमेज फ़ॉर्मेट्स को सपोर्ट करता है?**  
हाँ, Aspose.Note नोटबुक निर्यात के लिए PNG, JPEG, BMP आदि जैसे विभिन्न इमेज फ़ॉर्मेट्स को सपोर्ट करता है।

**प्रश्न 3: क्या मैं आउटपुट इमेज को आगे कस्टमाइज़ कर सकता हूँ?**  
हाँ, Aspose.Note आउटपुट इमेज को कस्टमाइज़ करने के लिए विस्तृत विकल्प प्रदान करता है, जिसमें पेज साइज, ओरिएंटेशन, और क्वालिटी सेटिंग्स शामिल हैं।

**प्रश्न 4: क्या Aspose.Note for Java का ट्रायल वर्ज़न उपलब्ध है?**  
हाँ, आप एक फ्री ट्रायल वर्ज़न [यहाँ](https://releases.aspose.com/) से प्राप्त कर सकते हैं।

**प्रश्न 5: Aspose.Note for Java के लिए सपोर्ट कहाँ मिल सकता है?**  
आप Aspose.Note फ़ोरम पर सपोर्ट और संसाधन [यहाँ](https://forum.aspose.com/c/note/28) पा सकते हैं।
