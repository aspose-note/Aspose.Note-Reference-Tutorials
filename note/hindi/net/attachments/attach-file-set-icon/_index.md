# Aspose.Note में कस्टम आइकन सेट करें और फ़ाइल संलग्न करें

## परिचय

यदि आपको OneNote पेज में किसी अटैचमेंट के लिए **set custom icon** सेट करने की आवश्यकता है, तो Aspose.Note for .NET इसे आसान बनाता है। फ़ाइल संलग्न करके और उसकी आइकन के रूप में एक छवि असाइन करके, आप अधिक समृद्ध, अधिक सूचनात्मक नोट्स बना सकते हैं जो बिल्कुल वही दिखते हैं जैसा आप चाहते हैं। इस ट्यूटोरियल में हम पूरी प्रक्रिया को चरणबद्ध तरीके से देखेंगे—एक नई दस्तावेज़ से शुरू करके, एक अटैचमेंट जोड़ना, एक कस्टम JPEG आइकन लागू करना, और अंत में OneNote फ़ाइल को सहेजना।

## त्वरित उत्तर
- **“set custom icon” क्या है?** यह आपको डिफ़ॉल्ट अटैचमेंट थंबनेल को आपके द्वारा प्रदान की गई छवि से बदलने की अनुमति देता है।  
- **क्या मैं कई फ़ाइलें संलग्न कर सकता हूँ?** हाँ, प्रत्येक फ़ाइल के लिए अटैचमेंट चरणों को दोहराएँ।  
- **आइकन के लिए कौन से इमेज फ़ॉर्मेट समर्थित हैं?** कोई भी .NET‑संगत फ़ॉर्मेट जैसे JPEG, PNG, BMP, या GIF।  
- **क्या मुझे लाइसेंस चाहिए?** उत्पादन उपयोग के लिए एक वैध Aspose.Note लाइसेंस आवश्यक है; मूल्यांकन के लिए एक मुफ्त ट्रायल काम करता है।  
- **OneNote फ़ाइल को कैसे सहेजें?** `Document.Save()` का उपयोग करें और *.one* फ़ाइल पाथ निर्दिष्ट करें।

## Aspose.Note में “set custom icon” क्या है?
कस्टम आइकन सेट करने का मतलब है कि OneNote पेज में संलग्न फ़ाइल को दर्शाने के लिए एक विशिष्ट छवि (जैसे JPEG) असाइन करना। यह उपयोगकर्ताओं के लिए दृश्य संकेतों को सुधारता है और फ़ाइल प्रकार के लोगो या ब्रांड इमेजरी दिखाने के लिए उपयोग किया जा सकता है।

## अटैचमेंट्स के लिए कस्टम आइकन क्यों सेट करें?
- **बेहतर दृश्य संदर्भ:** उपयोगकर्ता तुरंत संलग्न फ़ाइल के प्रकार या उद्देश्य को पहचान लेते हैं।  
- **ब्रांड स्थिरता:** सभी संबंधित दस्तावेज़ों के लिए अपने कंपनी के लोगो को आइकन के रूप में उपयोग करें।  
- **उन्नत उपयोगकर्ता अनुभव:** नोट्स को अधिक परिष्कृत और पेशेवर बनाता है, विशेष रूप से साझा नोटबुक्स में।

## आवश्यकताएँ
- C# प्रोग्रामिंग का बुनियादी ज्ञान।  
- Aspose.Note for .NET लाइब्रेरी स्थापित।  
- Visual Studio (या कोई भी .NET‑संगत IDE) विकास के लिए तैयार।

## नेमस्पेस इम्पोर्ट करें
पहले, आवश्यक नेमस्पेस को स्कोप में लाएँ ताकि आप फ़ाइलों, छवियों और OneNote ऑब्जेक्ट्स के साथ काम कर सकें।

```csharp
using System.IO;
using Aspose.Note;
using System;
using System.Collections.Generic;
using System.Drawing.Imaging;
```

## फ़ाइल संलग्न करने और उसका आइकन सेट करने के लिए चरण‑दर‑चरण गाइड

### चरण 1: एक Document ऑब्जेक्ट बनाएं
एक नई `Document` इंस्टेंस वह OneNote फ़ाइल दर्शाती है जिसे आप बनाएँगे।

```csharp
Document doc = new Document();
```

### चरण 2: एक Page ऑब्जेक्ट इनिशियलाइज़ करें
प्रत्येक OneNote फ़ाइल में एक या अधिक पेज होते हैं। यहाँ हम पहला पेज बनाते हैं।

```csharp
Aspose.Note.Page page = new Aspose.Note.Page(doc);
```

### चरण 3: एक Outline ऑब्जेक्ट इनिशियलाइज़ करें
Outlines पेज पर कंटेंट ब्लॉक्स के कंटेनर के रूप में कार्य करते हैं।

```csharp
Outline outline = new Outline(doc);
```

### चरण 4: एक OutlineElement ऑब्जेक्ट इनिशियलाइज़ करें
यह एलिमेंट संलग्न फ़ाइल और उसके कस्टम आइकन को रखेगा।

```csharp
OutlineElement outlineElem = new OutlineElement(doc);
```

### चरण 5: आइकन इमेज पढ़ें और AttachedFile ऑब्जेक्ट इनिशियलाइज़ करें
हम इमेज स्ट्रीम (कस्टम आइकन) खोलते हैं और उस फ़ाइल की ओर इशारा करते हैं जिसे हम एम्बेड करना चाहते हैं।

```csharp
string dataDir = "Your Document Directory";
using (var stream = File.OpenRead(dataDir + "icon.jpg"))
{
    AttachedFile attachedFile = new AttachedFile(doc, dataDir + "attachment.txt", stream, ImageFormat.Jpeg);
}
```

> **Pro tip:** यदि आप PNG आइकन पसंद करते हैं तो `ImageFormat.Jpeg` को `ImageFormat.Png` से बदलें।

### चरण 6: Attached File को OutlineElement में जोड़ें
अब फ़ाइल (उसके आइकन के साथ) OutlineElement की चाइल्ड बन जाती है।

```csharp
outlineElem.AppendChildLast(attachedFile);
```

### चरण 7: OutlineElement को Outline में जोड़ें
यह एलिमेंट को Outline कंटेनर के भीतर नेस्ट करता है।

```csharp
outline.AppendChildLast(outlineElem);
```

### चरण 8: Outline को Page में जोड़ें
पूरा Outline पदानुक्रम Page में संलग्न करें।

```csharp
page.AppendChildLast(outline);
```

### चरण 9: Page को Document में जोड़ें
पूरा किया गया पेज दस्तावेज़ संरचना में जोड़ें।

```csharp
doc.AppendChildLast(page);
```

### चरण 10: OneNote दस्तावेज़ को सहेजें
अंत में, दस्तावेज़ को डिस्क पर *.one* फ़ाइल के रूप में लिखें।

```csharp
dataDir = dataDir + "AttachFileAndSetIcon_out.one";
doc.Save(dataDir);
```

## सामान्य उपयोग केस
- **कॉन्ट्रैक्ट एम्बेड करना:** एक PDF संलग्न करें और कॉन्ट्रैक्ट‑लोगो आइकन का उपयोग करें।  
- **कोड स्निपेट्स साझा करना:** एक `.txt` फ़ाइल को कस्टम “code” आइकन के साथ संलग्न करें।  
- **प्रोजेक्ट डॉक्यूमेंटेशन:** डिज़ाइन फ़ाइलें संलग्न करें और फ़ाइल प्रकार से मेल खाने वाला थंबनेल सेट करें।

## सामान्य समस्याएँ और समाधान
| समस्या | समाधान |
|-------|----------|
| **आइकन नहीं दिख रहा** | `ImageFormat` जो आपने पास किया है, उसके साथ इमेज फ़ॉर्मेट मेल खाता है यह सत्यापित करें (जैसे JPEG बनाम PNG)। |
| **फ़ाइल पाथ त्रुटियाँ** | गुम स्लैश समस्याओं से बचने के लिए `Path.Combine(dataDir, "file.ext")` का उपयोग करें। |
| **बड़ी अटैचमेंट्स प्रदर्शन में देरी करती हैं** | फ़ाइल को पहले संपीड़ित करें या कई छोटे अटैचमेंट्स में विभाजित करें। |

## अक्सर पूछे जाने वाले प्रश्न

**प्रश्न:** क्या मैं Aspose.Note for .NET का उपयोग करके एक ही नोट में कई फ़ाइलें संलग्न कर सकता हूँ?  
**उत्तर:** हाँ। प्रत्येक फ़ाइल के लिए “Read the Icon Image and Initialize the AttachedFile Object” ब्लॉक को दोहराएँ और प्रत्येक `AttachedFile` को उसी `OutlineElement` में जोड़ें या अलग-अलग एलिमेंट बनाएं।

**प्रश्न:** क्या फ़ाइल अटैचमेंट्स के लिए कस्टम आइकन सेट करना संभव है?  
**उत्तर:** बिल्कुल। `AttachedFile` कन्स्ट्रक्टर आपको इमेज स्ट्रीम पास करने और इमेज फ़ॉर्मेट निर्दिष्ट करने की अनुमति देता है, जिससे आप आइकन पर पूर्ण नियंत्रण रख सकते हैं।

**प्रश्न:** क्या Aspose.Note आइकन सेट करने के लिए अन्य इमेज फ़ॉर्मेट्स का समर्थन करता है?  
**उत्तर:** हाँ। JPEG के अलावा, आप PNG, BMP, GIF, या `System.Drawing.Imaging.ImageFormat` द्वारा समर्थित कोई भी फ़ॉर्मेट उपयोग कर सकते हैं।

**प्रश्न:** क्या मैं बाहरी URLs से फ़ाइलें संलग्न कर सकता हूँ?  
**उत्तर:** जबकि Aspose.Note स्थानीय स्ट्रीम्स के साथ काम करता है, आप `HttpClient` के माध्यम से फ़ाइल डाउनलोड कर सकते हैं, उसे `MemoryStream` में संग्रहीत करें, और फिर उस स्ट्रीम को `AttachedFile` को पास कर सकते हैं।

**प्रश्न:** फ़ाइल अटैचमेंट्स के लिए कोई आकार सीमा है क्या?  
**उत्तर:** Aspose.Note स्वयं कोई कठोर सीमा नहीं लगाता, लेकिन बहुत बड़ी फ़ाइलें मेमोरी उपयोग और प्रदर्शन को प्रभावित कर सकती हैं। बड़ी अटैचमेंट्स को संपीड़ित करने पर विचार करें।
