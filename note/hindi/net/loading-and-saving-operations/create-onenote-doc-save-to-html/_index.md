# Aspose.Note का उपयोग करके OneNote को HTML के रूप में सहेजें

## परिचय

यदि आपको वेब प्रकाशन, रिपोर्टिंग या अभिलेखीय उद्देश्यों के लिए **OneNote को HTML के रूप में सहेजने** की आवश्यकता है, तो Aspose.Note for .NET आपको एक साफ़, केवल‑कोड समाधान प्रदान करता है जो Windows, Linux और macOS पर Microsoft Office की आवश्यकता के बिना काम करता है। इस ट्यूटोरियल में आप सीखेंगे कि कैसे शून्य से OneNote दस्तावेज़ बनाएं, उसकी पृष्ठों को संशोधित करें, और विभिन्न विकल्पों का उपयोग करके पूरे नोटबुक—या चयनित पृष्ठों—को HTML में निर्यात करें। यह गाइड आपको प्रत्येक चरण से गुज़रता है, बताता है कि प्रत्येक सेटिंग क्यों महत्वपूर्ण है, और वास्तविक‑दुनिया के परिदृश्य दिखाता है जहाँ HTML निर्यात चमकता है, जैसे कि OneNote सामग्री को नॉलेज‑बेस पोर्टल में एम्बेड करना या स्थिर दस्तावेज़ बनाना।

## त्वरित उत्तर
- **निर्यात करने की मुख्य विधि क्या है?** एक `Aspose.Note.Document` इंस्टेंस पर `document.Save("output.html", SaveFormat.Html)` कॉल करें।  
- **क्या मैं एकल पृष्ठ निर्यात कर सकता हूँ?** हाँ—`document.Save(pageRange, "page.html", SaveFormat.Html)` का उपयोग करें।  
- **क्या संसाधन (छवियां, CSS) स्वचालित रूप से एम्बेड होते हैं?** डिफ़ॉल्ट रूप से वे अलग फ़ाइलों के रूप में सहेजे जाते हैं; आप उन्हें `HtmlSaveOptions` के साथ मेमोरी स्ट्रीम में एम्बेड कर सकते हैं।  
- **.NET Core समर्थित है?** बिल्कुल—Aspose.Note .NET 5, .NET 6, और .NET Core 3.1+ के साथ काम करता है।  
- **क्या विकास के लिए लाइसेंस की आवश्यकता है?** एक मुफ्त 30‑दिन का ट्रायल परीक्षण के लिए काम करता है; लाइसेंस मूल्यांकन वॉटरमार्क हटाता है।

## “OneNote को HTML के रूप में सहेजना” क्या है?

**“OneNote को HTML के रूप में सहेजना”** वह प्रक्रिया है जिसमें OneNote नोटबुक, जो एक स्वामित्व बाइनरी फ़ॉर्मेट में संग्रहीत होती है, को मानक HTML फ़ाइलों में परिवर्तित किया जाता है जिन्हें कोई भी वेब ब्राउज़र रेंडर कर सकता है। यह रूपांतरण नोटबुक की पदानुक्रमित संरचना, टेक्स्ट फ़ॉर्मेटिंग, छवियां, तालिकाएं और अन्य संसाधनों को संरक्षित करता है, जबकि OneNote‑विशिष्ट तत्वों को समकक्ष HTML टैग और CSS स्टाइल में बदलता है। इससे डेवलपर्स OneNote सामग्री को वेबसाइटों पर प्रदर्शित कर सकते हैं, इसे कंटेंट‑मैनेजमेंट सिस्टम में एकीकृत कर सकते हैं, या इसे सार्वभौमिक रूप से सुलभ फ़ॉर्मेट में अभिलेखित कर सकते हैं बिना OneNote क्लाइंट की आवश्यकता के।

## OneNote को HTML में बदलने का कारण क्या है?

Aspose.Note **50+ आउटपुट फ़ॉर्मेट** (PDF, DOCX, PNG, HTML आदि) में निर्यात कर सकता है और **500 पृष्ठ** तक की नोटबुक को पूरी फ़ाइल को मेमोरी में लोड किए बिना प्रोसेस करता है, इसके स्ट्रीमिंग आर्किटेक्चर के कारण। HTML में बदलने से आप:

1. **वेब पर सामग्री प्रकाशित कर सकते हैं** बिना OneNote क्लाइंट की आवश्यकता के।  
2. **कंटेंट‑मैनेजमेंट सिस्टम** के साथ एकीकृत कर सकते हैं जो केवल HTML स्वीकार करता है।  
3. **खोज योग्य टेक्स्ट** को निकालकर इंडेक्सिंग इंजन में उपयोग कर सकते हैं।  
4. **पृष्ठ‑स्तर के संसाधनों** (छवियां, फ़ॉन्ट) को अन्य वेब प्रोजेक्ट्स में पुन: उपयोग कर सकते हैं।

## OneNote को HTML के रूप में सहेजने के चरण-दर-चरण निर्देश

OneNote को HTML के रूप में सहेजने में तीन मुख्य कार्य शामिल हैं: नोटबुक लोड या बनाना, इच्छित निर्यात विकल्प कॉन्फ़िगर करना, और उपयुक्त `Save` मेथड को कॉल करना। पहले आप एक `Document` ऑब्जेक्ट बनाते या लोड करते हैं, फिर `HtmlSaveOptions` को समायोजित करके रिसोर्स हैंडलिंग, CSS जनरेशन और एन्कोडिंग नियंत्रित कर सकते हैं। अंत में, आप `document.Save` को फ़ाइल पाथ या स्ट्रीम के साथ कॉल करते हैं, वैकल्पिक रूप से `PageRange` निर्दिष्ट करके निर्यात को सीमित कर सकते हैं। यह व्यवस्थित दृष्टिकोण विभिन्न वातावरणों में सुसंगत परिणाम सुनिश्चित करता है और आपको प्रोजेक्ट की आवश्यकताओं के अनुसार आउटपुट को अनुकूलित करने की अनुमति देता है।

### चरण 1: डिफ़ॉल्ट विकल्पों के साथ OneNote दस्तावेज़ बनाएं

इस पहले उदाहरण में हम एक नया `Document` इंस्टैंस बनाते हैं, एक शीर्षक के साथ एक पृष्ठ जोड़ते हैं, और डिफ़ॉल्ट `HtmlSaveOptions` का उपयोग करके पूरे नोटबुक को HTML में सहेजते हैं। कोई अतिरिक्त पैरामीटर आवश्यक नहीं है, जिससे यह **OneNote को HTML के रूप में सहेजने** का सबसे तेज़ तरीका बनता है।

```csharp
using System;
```

### चरण 2: विशिष्ट पृष्ठ रेंज को HTML में निर्यात करें

अक्सर आपको पृष्ठों का एक उपसमुच्चय चाहिए—उदाहरण के लिए, मीटिंग नोट्स के एक अध्याय को निर्यात करना। `Save` को `PageRange` ऑब्जेक्ट पास करके आप पृष्ठ 1‑3 को लक्ष्य बना सकते हैं जबकि बाकी को अनछुआ छोड़ सकते हैं।

```csharp
public static void CreateAndSaveToHTMLUsingDefaultOptions()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateOneNoteDocAndSaveToHTML_out.html");
    doc.Save(outputPath);

    Console.WriteLine("\nOneNote document created successfully.\nFile saved at " + outputPath);
}
```

### चरण 3: एम्बेडेड रिसोर्सेज़ के साथ मेमोरी स्ट्रीम में HTML सहेजें

जब आपको एकल स्व-समाहित HTML पेलोड चाहिए (जैसे API प्रतिक्रियाओं के लिए), तो CSS, फ़ॉन्ट और छवियों को सीधे `MemoryStream` में एम्बेड करें। `HtmlSaveOptions` आपको `EmbedResources` को `true` करने की अनुमति देता है, जिससे एक कॉम्पैक्ट HTML स्ट्रिंग बनती है।

```csharp
public static void CreateAndSavePageRange()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateAndSavePageRange_out.html");
    doc.Save(outputPath, new HtmlSaveOptions { PageCount = 1, PageIndex = 0 });

    Console.WriteLine("\nOneNote document created successfully and saved as page range.\nFile saved at " + outputPath);
}
```

### चरण 4: संसाधनों को अलग फ़ाइलों में रखकर फ़ाइलों में HTML सहेजें

यदि आप क्लासिक वेब‑फ़ोल्डर आउटपुट—HTML फ़ाइल प्लस एक एसेट फ़ोल्डर—को पसंद करते हैं, तो एम्बेडिंग को निष्क्रिय करें और आउटपुट डायरेक्टरी निर्दिष्ट करें। यह दृष्टिकोण स्रोत फ़ाइलों को साफ़ रखता है और छवियों या स्टाइलशीट्स की कैशिंग को आसान बनाता है।

```csharp
public static void SaveAsHTMLToMemoryStreamWithEmbeddedResources()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportEmbedded,
        ExportFonts = ResourceExportType.ExportEmbedded,
        ExportImages = ResourceExportType.ExportEmbedded,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to a memory stream
    var memoryStream = new MemoryStream();
    document.Save(memoryStream, options);
}
```

### चरण 5: रिसोर्सेज़ को प्रबंधित करने के लिए कॉलबैक्स के साथ HTML सहेजें

उन्नत परिदृश्य, जैसे कि छवियों को डेटाबेस या क्लाउड बकेट में संग्रहीत करना, कस्टम हैंडलिंग की मांग करते हैं। `ResourceSavingCallback` डेलीगेट प्रदान करके आप प्रत्येक रिसोर्स को कैसे सहेजना है, उस पर पूर्ण नियंत्रण रखते हैं, जबकि API सही HTML रेफ़रेंसेज़ उत्पन्न करता रहता है।

```csharp
public static void SaveAsHTMLToFileWithResourcesInSeparateFiles()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportAsStream,
        ExportFonts = ResourceExportType.ExportAsStream,
        ExportImages = ResourceExportType.ExportAsStream,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to HTML file with resources stored in separate files
    document.Save(Path.Combine(dataDir, "document_out.html"), options);
}
```

## अक्सर पूछे जाने वाले प्रश्न

**Q: क्या मैं सहेजे गए HTML फ़ाइल की उपस्थिति को अनुकूलित कर सकता हूँ?**  
A: हाँ—`HtmlSaveOptions` द्वारा उत्पन्न CSS को संशोधित करें या `HtmlSaveOptions.CustomCss` के माध्यम से अपना स्वयं का स्टाइलशीट प्रदान करें।

**Q: क्या Aspose.Note HTML के अलावा अन्य फ़ॉर्मैट में रूपांतरण का समर्थन करता है?**  
A: बिल्कुल; आप PDF, DOCX, PNG, JPEG आदि में रूपांतरण कर सकते हैं—30 से अधिक आउटपुट फ़ॉर्मैट समर्थित हैं।

**Q: क्या Aspose.Note .NET Core एप्लिकेशन के साथ संगत है?**  
A: हाँ—.NET Core 3.1, .NET 5, .NET 6 और बाद के संस्करणों के लिए पूर्ण समर्थन।

**Q: क्या मैं Aspose.Note का उपयोग करके OneNote दस्तावेज़ों से टेक्स्ट और छवियां निकाल सकता हूँ?**  
A: `Page.GetText()` के साथ साधारण टेक्स्ट और `Page.GetResources(ResourceType.Image)` के माध्यम से छवियां प्राप्त कर सकते हैं।

**Q: क्या Aspose.Note सुविधाओं के परीक्षण के लिए ट्रायल संस्करण उपलब्ध है?**  
A: हाँ, आप एक मुफ्त ट्रायल [यहाँ](https://releases.aspose.com/) से डाउनलोड कर सकते हैं।

## निष्कर्ष

उपरोक्त पैटर्न का पालन करके आप एक पंक्ति के कोड में **OneNote को HTML के रूप में सहेज** सकते हैं या जटिल एंटरप्राइज़ आवश्यकताओं को पूरा करने के लिए निर्यात को फाइन‑ट्यून कर सकते हैं। चाहे आपको एक त्वरित एक‑पृष्ठ स्नैपशॉट चाहिए, वेब प्रकाशन के लिए पूरा नोटबुक चाहिए, या कस्टम रिसोर्स पाइपलाइन चाहिए, Aspose.Note वह लचीलापन, प्रदर्शन और क्रॉस‑प्लेटफ़ॉर्म विश्वसनीयता प्रदान करता है जिसकी आपको आवश्यकता है। इन स्निपेट्स को अपने मौजूदा .NET समाधान में एकीकृत करें, और आप किसी भी ब्राउज़र‑आधारित उपभोक्ता को OneNote सामग्री बिना मूल OneNote क्लाइंट के ओवरहेड के सर्व कर पाएंगे।

**अंतिम अपडेट:** 2026-06-20  
**परीक्षण किया गया:** Aspose.Note 24.11 for .NET  
**लेखक:** Aspose

```csharp
public static void SaveAsHTMLToMemoryStreamWithCallBacksToSaveResources()
{
    // Specify the saving callbacks configuration
    var savingCallbacks = new UserSavingCallbacks()
    {
        RootFolder = "documentFolder",
        CssFolder = "css",
        KeepCssStreamOpened = true,
        ImagesFolder = "images",
        FontsFolder = "fonts"
    };

    // Specify HTML saving options
    var options = new HtmlSaveOptions
    {
        FontFaceTypes = FontFaceType.Ttf,
        CssSavingCallback = savingCallbacks,
        FontSavingCallback = savingCallbacks,
        ImageSavingCallback = savingCallbacks
    };

    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Save the document to HTML format with resources managed by user-defined callbacks
    using (var stream = File.Create(Path.Combine(savingCallbacks.RootFolder, "document.html")))
    {
        document.Save(stream, options);
    }

    // Manually append data to the CSS stream
    using (var writer = new StreamWriter(savingCallbacks.CssStream))
    {
        writer.WriteLine();
        writer.WriteLine("/* This line is appended to stream manually by user */");
    }
}
```
