# Εφαρμογή Σκοτεινού Θέματος στο Κείμενο στο OneNote - Aspose.Note

## Εισαγωγή
Αν ψάχνετε να **μετατρέψετε το OneNote σε PDF** ενώ δίνετε στις σημειώσεις σας μια κομψή σκοτεινή εμφάνιση, βρίσκεστε στο σωστό μέρος. Σε αυτό το tutorial θα δούμε πώς να εφαρμόσετε ένα σκοτεινό θέμα, να αλλάξετε το χρώμα του κειμένου και τελικά να αποθηκεύσετε το αρχείο OneNote ως PDF χρησιμοποιώντας το **Aspose.Note for Java**. Στο τέλος θα έχετε ένα γυαλισμένο, υψηλής αντίθεσης έγγραφο που είναι πιο φιλικό στα μάτια και έτοιμο για κοινή χρήση.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει “OneNote σε PDF”;** Μετατρέπει ένα αρχείο OneNote . one σε έγγραφο PDF διατηρώντας τη διάταξη.  
- **Μπορώ να εφαρμόσω σκοτεινό θέμα κατά τη μετατροπή;** Ναι – μπορείτε να ορίσετε το φόντο της σελίδας σε μαύρο και να αλλάξετε το ανοιχτό κείμενο σε λευκό.  
- **Ποια βιβλιοθήκη διαχειρίζεται τη μετατροπή;** Το Aspose.Note for Java παρέχει ένα απλό API για τη μορφοποίηση και την εξαγωγή PDF.  
- **Χρειάζομαι άδεια για παραγωγική χρήση;** Απαιτείται έγκυρη άδεια Aspose.Note για μη‑δοκιμαστικές εγκαταστάσεις.  
- **Ποια έκδοση Java υποστηρίζεται;** Οποιοδήποτε JDK 8+ λειτουργεί με την πιο πρόσφατη έκδοση του Aspose.Note.

## Τι είναι το “OneNote σε PDF” με Σκοτεινό Θέμα;
Η μετατροπή OneNote σε PDF δημιουργεί μια φορητή, μόνο για ανάγνωση έκδοση των σημειώσεών σας. Η προσθήκη σκοτεινού θέματος σημαίνει ορισμό μαύρου φόντου σελίδας και εξασφάλιση ότι όλο το κείμενο είναι ανοιχτόχρωμο, βελτιώνοντας την αναγνωσιμότητα σε συνθήκες χαμηλού φωτισμού και προσφέροντας μοντέρνα εμφάνιση.

## Εισαγωγή Πακέτων
Για να ξεκινήσετε, εισάγετε τα απαραίτητα πακέτα στο έργο Java σας:

```java
import com.aspose.note.*;
import java.awt.*;
import java.io.IOException;
import java.nio.file.Paths;
```

## Βήμα 1: Ορισμός Χρώματος Φόντου Σελίδας
Αρχικά, αλλάζουμε το φόντο της σελίδας σε μαύρο – αυτό είναι το βήμα **set page background java** που δημιουργεί τη βάση του σκοτεινού θέματος.

```java
// The path to the documents directory.
String dataDir = "Your Document Directory";
// Load the document into Aspose.Note.
Document doc = new Document(Paths.get(dataDir, "Aspose.one").toString());
for (Page page: doc)
{
    page.setBackgroundColor(Color.BLACK);
}
```

## Βήμα 2: Προσαρμογή Χρώματος Κειμένου
Στη συνέχεια, σαρώσουμε κάθε κόμβο `RichText`. Εάν το υπάρχον χρώμα γραμματοσειράς είναι πολύ σκούρο (ή σχεδόν αόρατο σε μαύρο φόντο), το αλλάζουμε σε λευκό. Αυτό είναι η λογική **change text color java** / **adjust font color java**.

```java
for (RichText node: doc.getChildNodes(RichText.class))
{
    Color c = node.getParagraphStyle().getFontColor();
    if (c.getAlpha() <= 10 || Math.abs(c.getRed() - Color.BLACK.getRed()) + Math.abs(c.getGreen() - Color.BLACK.getGreen()) + Math.abs(c.getBlue() - Color.BLACK.getBlue()) <= 30)
    {
        node.getParagraphStyle().setFontColor(Color.WHITE);
    }
}
```

## Βήμα 3: Αποθήκευση Εγγράφου ως PDF
Τέλος, εξάγουμε το μορφοποιημένο αρχείο OneNote σε PDF – το βήμα **save onenote pdf**.

```java
doc.save(Paths.get(dataDir, "AsposeDarkTheme.pdf").toString());
```

Επαναλάβετε αυτά τα βήματα για κάθε έγγραφο και θα έχετε εφαρμόσει επιτυχώς ένα σκοτεινό θέμα στο κείμενο του OneNote και το έχετε εξάγει ως PDF χρησιμοποιώντας το Aspose.Note for Java.
