# Μετατροπή OneNote σε PDF και Διαχείριση Μετρημένης Άδειας σε Java

## Εισαγωγή

Σε αυτό το tutorial θα ανακαλύψετε πώς να **μετατρέψετε το OneNote σε PDF** χρησιμοποιώντας μια μετρημένη άδεια με τη βιβλιοθήκη Aspose.Note for Java. Η μετρημένη άδεια σας επιτρέπει να παρακολουθείτε την κατανάλωση πίστωσης σε πραγματικό χρόνο, δίνοντάς σας την ευελιξία να πληρώνετε μόνο για ό,τι χρησιμοποιείτε. Θα περάσουμε από τη διαδικασία από την ρύθμιση των κλειδιών άδειας μέχρι τη φόρτωση ενός αρχείου OneNote, τη μετατροπή του σε PDF και τον έλεγχο της χρήσης πίστωσης.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει η “μετατροπή OneNote σε PDF”;** Μετατρέπει ένα αρχείο .one σε ένα φορητό έγγραφο PDF.  
- **Ποια βιβλιοθήκη διαχειρίζεται τη μετατροπή;** Η Aspose.Note for Java παρέχει ένα απλό API για αυτήν την εργασία.  
- **Χρειάζομαι άδεια για τη μετατροπή;** Ναι, απαιτείται μια μετρημένη άδεια για χρήση σε παραγωγή.  
- **Πώς μπορώ να παρακολουθήσω τη χρήση;** Χρησιμοποιήστε `Metered.getConsumptionCredit()` και `Metered.getConsumptionQuantity()`.  
- **Απαιτείται κάποια επιπλέον ρύθμιση;** Απλώς ένα Java JDK και το αρχείο JAR της Aspose.Note.

## Τι είναι η “μετατροπή OneNote σε PDF”; 

Η μετατροπή OneNote σε PDF δημιουργεί μια στατική, ευρέως υποστηριζόμενη αναπαράσταση των σελίδων του σημειωματάριου σας. Τα PDF είναι ιδανικά για κοινή χρήση, εκτύπωση ή αρχειοθέτηση, επειδή διατηρούν τη διάταξη, τις γραμματοσειρές και τις εικόνες σε όλες τις πλατφόρμες.

## Γιατί να χρησιμοποιήσετε μια μετρημένη άδεια για αυτήν τη μετατροπή; 

Μια μετρημένη άδεια χρεώνει βάσει της πραγματικής κατανάλωσης πίστωσης αντί για σταθερό τέλος. Αυτό το μοντέλο είναι τέλειο όταν έχετε περιοδικές ανάγκες μετατροπής ή θέλετε να διατηρείτε τα κόστη προβλέψιμα, ενώ εξακολουθείτε να έχετε πρόσβαση σε όλη τη δύναμη της Aspose.Note.

## Προαπαιτούμενα

Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε:

1. **Java Development Kit (JDK)** – οποιαδήποτε πρόσφατη έκδοση (συνιστάται JDK 11+). Κατεβάστε το από [εδώ](https://www.oracle.com/java/technologies/javase-jdk11-downloads.html).  
2. **Aspose.Note for Java library** – αποκτήστε το τελευταίο JAR από την [επίσημη ιστοσελίδα](https://releases.aspose.com/note/java/).  

> **Pro tip:** Προσθέστε το JAR της Aspose.Note στο classpath του έργου σας ή χρησιμοποιήστε ένα εργαλείο κατασκευής όπως Maven/Gradle για τη διαχείριση της εξάρτησης.

## Εισαγωγή Πακέτων

Πρώτα, εισάγετε τις κλάσεις που θα χρειαστείτε. Διατηρήστε αυτό το μπλοκ ακριβώς όπως φαίνεται — χωρίς αλλαγές στον κώδικα.

```java
import com.aspose.note.Document;
import com.aspose.note.Metered;



import java.nio.file.Paths;
```

## Βήμα 1: Ρύθμιση της Μετρημένης Άδειας

```java
Metered metered = new Metered();
metered.setMeteredKey("MyPublicKey", "MyPrivateKey");
```

Εδώ δημιουργούμε ένα αντικείμενο `Metered` και ενσωματώνουμε τα δημόσια/ιδιωτικά κλειδιά που λάβατε από την Aspose. Αυτό το βήμα ενεργοποιεί τη λειτουργία μετρημένης άδειας για όλες τις επόμενες λειτουργίες, συμπεριλαμβανομένης της κλήσης **convert OneNote to PDF**.

## Βήμα 2: Φόρτωση του Εγγράφου OneNote και Μετατροπή σε PDF

```java
String dataDir = "Your Document Directory";
Document doc = new Document(Paths.get(dataDir,"Sample1.one").toString());
doc.save(Paths.get(dataDir,"MeteredLicense.pdf").toString());
```

- **Φόρτωση**: `Document` διαβάζει το αρχείο `.one` που βρίσκεται στο `dataDir`. Αυτός είναι ο κλασικός τρόπος για **φόρτωση εγγράφου OneNote** για επεξεργασία.  
- **Μετατροπή & Αποθήκευση**: Η κλήση `save` με επέκταση `.pdf` μετατρέπει αυτόματα **.one σε .pdf**. Η λειτουργία επίσης **αποθηκεύει PDF από OneNote** στον ίδιο φάκελο. Αντικαταστήστε το `"Your Document Directory"` με την πραγματική διαδρομή στο σύστημά σας.

## Βήμα 3: Έλεγχος Κατανάλωσης Πίστωσης Πριν και Μετά τη Μετατροπή

```java
System.out.println(String.format("Credit before operation: %.02f", Metered.getConsumptionCredit()));
System.out.println(String.format("Consumption quantity before operation: %.02f", Metered.getConsumptionQuantity()));
```

Αυτές οι δύο γραμμές εκτυπώνουν το υπόλοιπο πίστωσης και την ποσότητα πίστωσης που χρησιμοποιήθηκε από τη μετατροπή. Εκτελώντας τις ίδιες κλήσεις μετά την ενέργεια `save` θα δείτε τις ενημερωμένες τιμές, επιβεβαιώνοντας ότι η μετρημένη άδεια παρακολουθεί σωστά τη χρήση.
