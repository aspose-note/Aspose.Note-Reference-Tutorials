# Αποθήκευση σε Δυαδική Εικόνα Χρησιμοποιώντας τη Μέθοδο Otsu στο OneNote

## Εισαγωγή

Σε αυτό το εκπαιδευτικό υλικό, θα ανακαλύψετε **πώς να αποθηκεύσετε OneNote** έγγραφα ως δυαδικές εικόνες χρησιμοποιώντας τη μέθοδο Otsu με το Aspose.Note for Java. Η μετατροπή ενός αρχείου OneNote σε εικόνα ασπρόμαυρη είναι χρήσιμη για pipelines επεξεργασίας εικόνας, προεπεξεργασία OCR, ή όταν απλώς χρειάζεστε μια ελαφριά οπτική αναπαράσταση των σημειώσεών σας.

## Γρήγορες Απαντήσεις
- **Τι κάνει η μέθοδος Otsu;** Καθορίζει αυτόματα το βέλτιστο κατώφλι για τη μετατροπή μιας εικόνας σε κλίμακα γκρι σε ασπρόμαυρη (δυαδική) εικόνα.  
- **Ποια μορφή χρησιμοποιείται για την έξοδο;** Η PNG είναι η προεπιλογ επειδή διατηρεί την ποιότητα χωρίς απώλειες.  
- **Χρειάζομαι άδεια για την εκτέλεση του κώδικα;** Μια δωρεάν δοκιμή λειτουργεί για ανάπτυξη· απαιτείται εμπορική άδεια για παραγωγή.  
- **Μπορώ να αλλάξω την έξοδο σε άλλη μορφή;** Ναι – απλώς αντικαταστήστε το `SaveFormat.Png` με άλλη υποστηριζόμενη μορφή.  
- **Είναι κατάλληλο για OCR;** Απόλυτα – οι δυαδικές εικόνες βελτιώνουν την ακρίβεια του OCR αφαιρώντας τον θόρυβο κλίμακας γκρι.

## Τι είναι η Μέθοδος Otsu;
Η μέθοδος Otsu αναλύει το ιστόγραμμα μιας εικόνας σε κλίμακα γκρι και επιλέγει ένα κατώφλι που ελαχιστοποιεί την ενδο‑τάξη διασπορά, διαχωρίζοντας αποτελεσματικά το προσκήνιο (μαύρο) από το φόντο (άσπρο). Αυτό την καθιστά ιδανική για τη δημιουργία εξόδων **black white image java** από σελίδες OneNote.

## Γιατί να αποθηκεύσετε το OneNote ως PNG;
- **Καθολική συμβατότητα:** Η PNG λειτουργεί σε προγράμματα περιήγησης, κινητές εφαρμογές και επιτραπέζια εργαλεία.  
- **Συμπίεση χωρίς απώλειες:** Δεν υπάρχει υποβάθμιση ποιότητας, κάτι που είναι κρίσιμο για επεξεργασία σε επόμενα στάδια.  
- **Έτοιμο για OCR:** Τα δυαδικά PNG είναι η προτιμώμενη είσοδος για τις περισσότερες μηχανές OCR.

## Προαπαιτούμενα
1. Βασικές γνώσεις προγραμματισμού Java.  
2. Εγκατεστημένο JDK (Java Development Kit).  
3. Βιβλιοθήκη Aspose.Note for Java προστιθέμενη στο έργο σας (Maven/Gradle ή χειροκίνητο JAR).  

## Εισαγωγή Πακέτων
Για να ξεκινήσετε, εισάγετε τις απαιτούμενες κλάσεις Aspose.Note και τις βοηθητικές βιβλιοθήκες I/O της Java.

```java
import com.aspose.note.*;
import java.io.IOException;
```

## Βήμα 1: Φόρτωση του Εγγράφου OneNote
Πρώτα, δείξτε στο φάκελο που περιέχει το αρχείο `.one` σας και φορτώστε το στο αντικείμενο `Document`.

```java
String dataDir = "Your Document Directory";
// Load the document into Aspose.Note.
Document oneFile = new Document(dataDir + "Aspose.one");
```

## Βήμα 2: Διαμόρφωση Δυαδικοποίησης με Otsu
Δημιουργήστε ένα στιγμιότυπο `ImageBinarizationOptions` και ενημερώστε το Aspose.Note να χρησιμοποιεί τον αλγόριθμο Otsu.

```java
dataDir = dataDir + "SaveToBinaryImageUsingOtsuMethod_out.png";
ImageBinarizationOptions binarizationOptions = new ImageBinarizationOptions();
binarizationOptions.setBinarizationMethod(BinarizationMethod.Otsu);
```

## Βήμα 3: Ορισμός Επιλογών Αποθήκευσης Εικόνας (PNG, Ασπρόμαυρο)
Ορίστε πώς θα αποθηκευτεί η εικόνα. Εδώ επιλέγουμε PNG, εξαναγκάζουμε λειτουργία χρώματος ασπρόμαυρο και συνδέουμε τις επιλογές δυαδικοποίησης.

```java
ImageSaveOptions options = new ImageSaveOptions(SaveFormat.Png);
options.setColorMode(ColorMode.BlackAndWhite);
options.setBinarizationOptions(binarizationOptions);
```

## Βήμα 4: Αποθήκευση του Εγγράφου ως Δυαδική Εικόνα
Τέλος, γράψτε το δυαδικό PNG στο δίσκο χρησιμοποιώντας τις επιλογές που προετοιμάσαμε.

```java
// Save the document.
oneFile.save(dataDir, options);
```

## Κοινά Προβλήματα & Συμβουλές
- **Αρχείο δεν βρέθηκε:** Επαληθεύστε ότι το `dataDir` τελειώνει με διαχωριστικό διαδρομής (`/` ή `\\`) πριν προσαρτήσετε το όνομα του αρχείου.  
- **Κενή έξοδος:** Βεβαιωθείτε ότι η πηγή σελίδα OneNote περιέχει περιεχόμενο· κενές σελίδες θα παράγουν κενό PNG.  
- **Απόδοση:** Για μεγάλα σημειωματάρια, επεξεργαστείτε τις σελίδες ξεχωριστά ώστε η χρήση μνήμης να παραμένει χαμηλή.

Τώρα γνωρίζετε **πώς να αποθηκεύσετε OneNote** ως δυαδική εικόνα PNG χρησιμοποιώντας τη μέθοδο Otsu στη Java. Αυτή η προσέγγιση είναι ιδανική για τη δημιουργία πόρων **black white image java** για OCR, αρχειοθέτηση ή οποιοδήποτε σενάριο όπου απαιτείται ένα ελαφρύ οπτικό αντίγραφο μιας σελίδας OneNote.

## Πρόσθετες Συχνές Ερωτήσεις

**Ε: Πώς αλλάζω τη μορφή εξόδου από PNG σε JPEG;**  
Α: Αντικαταστήστε το `SaveFormat.Png` με `SaveFormat.Jpeg` στον κατασκευαστή `ImageSaveOptions`.

**Ε: Υπάρχει τρόπος να ορίσω προσαρμοσμένο DPI για την εξαγόμενη εικόνα;**  
Α: Ναι, χρησιμοποιήστε `options.setResolution(double dpi)` πριν καλέσετε `save`.

**Ε: Μπορώ να επεξεργαστώ πολλές σελίδες OneNote σε βρόχο;**  
Α: Σίγουρα – επαναλάβετε πάνω στο `Document.getPages()` και εφαρμόστε την ίδια λογική αποθήκευσης σε κάθε σελίδα.
