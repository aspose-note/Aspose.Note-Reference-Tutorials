# Λήψη Αναθεωρήσεων Σελίδων στο OneNote - Aspose.Note

## Εισαγωγή

## Γρήγορες Απαντήσεις
- **Τι επιστρέφει η “get last modified time”;** Η χρονική σήμανση της πιο πρόσφατης επεξεργασίας σε μια σελίδα OneNote.  
- **Ποια κλάση παρέχει το ιστορικό αναθεωρήσεων;** `PageHistory` μέσω `Document.getPageHistory(Page)`.  
- **Χρειάζομαι άδεια για αυτή τη λειτουργία;** Ναι, απαιτείται έγκυρη άδεια Aspose.Note για χρήση σε παραγωγή.  
- **Ποια έκδοση της Java υποστηρίζεται;** Java 8 ή νεότερη (JDK 8+).  
- **Μπορώ να φιλτράρω τις αναθεωρήσεις ανά συγγραφέα;** Μπορείτε να διαβάσετε την ιδιότητα `Author` κάθε αντικειμένου `Page` και να εφαρμόσετε το δικό σας φίλτρο.

## Τι είναι η “get last modified time” στο OneNote;

Η **τελευταία ημερομηνία τροποποίησης** είναι ένα πεδίο μεταδεδομένων που αποθηκεύεται με κάθε σελίδα και καταγράφει πότε η σελίδα άλλαξε τελευταία φορά. Το Aspose.Note εκθέτει αυτήν την τιμή μέσω της μεθόδου `Page.getLastModifiedTime()`, καθιστώντας εύκολο το να εμφανίσετε ή να καταγράψετε τις ημερομηνίες αλλαγής.

## Γιατί να ανακτήσετε τις αναθεωρήσεις σελίδας;

- **Ιχνηλάτες ελέγχου:** Διατηρήστε ένα αρχείο του ποιος άλλαξε τι και πότε.  
- **Σύγκριση εκδόσεων:** Δημιουργήστε λειτουργίες που συγκρίνουν δύο αναθεωρήσεις δίπλα‑δίπλα.  
- **Ενδείξεις συνεργασίας χρηστών:** Κατανοήστε τα πρότυπα επεξεργασίας σε κοινόχρηστα σημειωματάρια.

## Πώς να Λάβετε την Τελευταία Ημερομηνία Τροποποίησης μιας Σελίδας OneNote

### Βήμα 1: Ορισμός Καταλόγου Εγγράφου
Ορίστε τον κατάλογο όπου βρίσκεται το έγγραφο OneNote σας.

```java
String dataDir = "Your Document Directory";
```

### Βήμα 2: Φόρτωση του Εγγράφου
Φορτώστε το έγγραφο OneNote στο Aspose.Note.

```java
Document doc = new Document(dataDir + "Sample1.one");
```

### Βήμα 3: Λήψη Πρώτης Σελίδας
Ανακτήστε την πρώτη σελίδα από το έγγραφο.

```java
Page firstPage = doc.getFirstChild();
```

### Βήμα 4: Λήψη Αναθεωρήσεων Σελίδας
Αποκτήστε το ιστορικό αναθεωρήσεων της σελίδας.

```java
PageHistory revisions = doc.getPageHistory(firstPage);
```

### Βήμα 5: Διάσχιση Αναθεωρήσεων Σελίδας
Επανάληψη στη λίστα των αναθεωρήσεων σελίδας και ανάκτηση σχετικών πληροφοριών, συμπεριλαμβανομένης της **τελευταίας ημερομηνίας τροποποίησης**.

```java
for (Page pageRevision : revisions) {
    System.out.println("LastModifiedTime: " + pageRevision.getLastModifiedTime());
    System.out.println("CreationTime: " + pageRevision.getCreationTime());
    System.out.println("Title: " + pageRevision.getTitle());
    System.out.println("Level: " + pageRevision.getLevel());
    System.out.println("Author: " + pageRevision.getAuthor());
    System.out.println();
}
```

## Κοινά Προβλήματα και Λύσεις
- **Null `PageHistory`:** Βεβαιωθείτε ότι το έγγραφο περιέχει πραγματικά αναθεωρήσεις· διαφορετικά το `getPageHistory` μπορεί να επιστρέψει `null`.  
- **Διαφορές ζώνης ώρας:** Το `getLastModifiedTime()` επιστρέφει ένα `java.util.Date` στη προεπιλεγμένη ζώνη ώρας του συστήματος. Μετατρέψτε σε UTC αν χρειάζεται.  
- **Άδεια δεν φορτώθηκε:** Χωρίς έγκυρη άδεια, το Aspose.Note μπορεί να λειτουργεί σε λειτουργία αξιολόγησης, περιορίζοντας τον αριθμό των σελίδων που επεξεργάζονται.
