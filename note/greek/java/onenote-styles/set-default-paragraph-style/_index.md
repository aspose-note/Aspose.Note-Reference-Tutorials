# Ορισμός Προεπιλεγμένου Μεγέθους Γραμματοσειράς onenote – Ορισμός Προεπιλεγμένου Στυλ Παραγράφου στο OneNote

## Εισαγωγή

Ο προγραμματιστικός ορισμός του **default font size onenote** σας επιτρέπει να διατηρήσετε μια συνεπή εμφάνιση σε όλες τις σελίδες χωρίς να επεξεργάζεστε χειροκίνητα κάθε παράγραφο. Σε αυτό το σεμινάριο θα μάθετε πώς να χρησιμοποιήσετε το Aspose.Note for Java για να ορίσετε ένα προεπιλεγμένο στυλ παραγράφου που περιλαμβάνει το προτιμώμενο μέγεθος γραμματοσειράς, το όνομα γραμματοσειράς και άλλες επιλογές μορφοποίησης. Στο τέλος θα έχετε ένα επαναχρησιμοποιήσιμο απόσπασμα κώδικα που μπορείτε να ενσωματώσετε σε οποιοδήποτε έργο Java που εργάζεται με αρχεία OneNote.

## Γρήγορες Απαντήσεις
- **Τι ελέγχει το “default font size onenote”;** Ορίζει το βασικό μέγεθος γραμματοσειράς που εφαρμόζεται σε κάθε νέα παράγραφο εκτός εάν παρακαμφθεί.  
- **Ποια βιβλιοθήκη διαχειρίζεται αυτό;** Το Aspose.Note for Java παρέχει μια ευέλικτη API για τον ορισμό προεπιλεγμένων παραγράφων.  
- **Χρειάζομαι άδεια;** Μια δωρεάν δοκιμή λειτουργεί για ανάπτυξη· απαιτείται εμπορική άδεια για παραγωγή.  
- **Μπορώ να αλλάξω και άλλα χαρακτηριστικά στυλ ταυτόχρονα;** Ναι – το όνομα γραμματοσειράς, το χρώμα, η στοίχιση και η απόσταση γραμμών είναι όλα ρυθμιζόμενα.  
- **Είναι συμβατό με όλες τις εκδόσεις του OneNote;** Το Aspose.Note υποστηρίζει αρχεία OneNote από το 2007 και μετά, καλύπτοντας πάνω από 95 % των ενεργών σημειωματάριων.

## Τι είναι το default font size onenote;
Το **default font size onenote** είναι το βασικό μέγεθος κειμένου που εφαρμόζεται σε νέες παραγράφους σε ένα έγγραφο OneNote όταν δεν έχει οριστεί ρητά μέγεθος. Ορίζοντάς το μία φορά, αποφεύγετε την επαναλαμβανόμενη μορφοποίηση και εξασφαλίζετε οπτική συνέπεια σε όλο το σημειωματάριο.

## Γιατί να ορίσετε προεπιλεγμένο στυλ παραγράφου στο OneNote;
Το Aspose.Note υποστηρίζει **πάνω από 50 μορφές εισόδου και εξόδου** και μπορεί να επεξεργαστεί σημειωματάρια με περιεχόμενο έως **2 GB** χωρίς να φορτώνει ολόκληρο το αρχείο στη μνήμη. Ο ορισμός προεπιλεγμένου στυλ παραγράφου μειώνει τον αριθμό των κλήσεων API έως και **40 %**, επιταχύνει τη δημιουργία εγγράφων και εγγυάται ότι κάθε παράγραφος συμμορφώνεται με τις εταιρικές οδηγίες branding.

## Προαπαιτούμενα

1. **Java Development Kit (JDK)** – Συνιστάται JDK 11 ή νεότερο.  
2. **Aspose.Note for Java** – κατεβάστε το από τη [σελίδα λήψης](https://releases.aspose.com/note/java/).  
3. **Ένα IDE** όπως το Eclipse, IntelliJ IDEA ή VS Code με υποστήριξη Java.  

## Εισαγωγή Πακέτων

Πρώτα, εισάγετε τα απαραίτητα πακέτα για να ξεκινήσετε τον κώδικα:

```java
import com.aspose.note.*;

import java.awt.*;
import java.io.FileInputStream;
import java.io.IOException;
import java.nio.file.Paths;
import java.util.List;
```

Τώρα, ας αναλύσουμε τον κώδικα παραδείγματος σε πολλαπλά βήματα:

## Πώς να αρχικοποιήσω το έγγραφο OneNote, τη σελίδα και το περίγραμμα;

Το Document αντιπροσωπεύει ολόκληρο το σημειωματάριο OneNote και παρέχει μεθόδους για τη διαχείριση του περιεχομένου του.  
Η Page αντιστοιχεί σε μια μεμονωμένη σελίδα μέσα στο σημειωματάριο, περιέχοντας outlines και άλλα στοιχεία.  
Το Outline είναι ένας container που ομαδοποιεί σχετιζόμενα στοιχεία πλούσιου κειμένου σε μια σελίδα.

Δημιουργήστε τα βασικά αντικείμενα που αντιπροσωπεύουν ένα αρχείο OneNote, μια σελίδα μέσα σε αυτό το αρχείο και το container outline που περιέχει πλούσιο κείμενο. Αυτά τα αντικείμενα αποτελούν τη βάση για οποιαδήποτε περαιτέρω μορφοποίηση και πρέπει να δημιουργηθούν πριν προστεθεί περιεχόμενο.

```java
String dataDir = "Your Document Directory";
Document document = new Document();
Page page = new Page();
Outline outline = new Outline();
```

## Πώς μπορώ να δημιουργήσω ένα στοιχείο outline για να φιλοξενήσει παραγράφους;

Το OutlineElement είναι παιδί του Outline που μπορεί να περιέχει πολλαπλά αντικείμενα RichText που αντιπροσωπεύουν μεμονωμένες παραγράφους.

Το στοιχείο outline λειτουργεί ως container για πολλαπλά αντικείμενα παραγράφου, επιτρέποντάς σας να ομαδοποιήσετε σχετιζόμενα τμήματα κειμένου. Μετά τη δημιουργία του, το συνδέετε με τη σελίδα ώστε το μορφοποιημένο κείμενο να εμφανίζεται στην προβλεπόμενη θέση μέσα στο σημειωματάριο.

```java
OutlineElement outlineElem = new OutlineElement();
```

## Πώς ορίζω το προεπιλεγμένο στυλ παραγράφου, συμπεριλαμβανομένου του μεγέθους γραμματοσειράς;

Το ParagraphStyle ορίζει χαρακτηριστικά μορφοποίησης όπως το όνομα γραμματοσειράς, το μέγεθος, το χρώμα και την ευθυγράμμιση που εφαρμόζονται σε μια παράγραφο.

Δημιουργήστε μια παρουσία του ParagraphStyle, ορίστε το fontSize στην επιθυμητή τιμή (π.χ., 12 pt) και προαιρετικά καθορίστε το fontName, το χρώμα ή την ευθυγράμμιση. Αναθέστε αυτό το στυλ ως προεπιλογή του εγγράφου ώστε κάθε νέα παράγραφος να κληρονομεί αυτόματα αυτές τις ρυθμίσεις χωρίς επιπλέον κώδικα.

```java
ParagraphStyle defaultStyle = new ParagraphStyle()
										.setFontName("Courier New")
										.setFontSize(20);
```

## Πώς μπορώ να δημιουργήσω πλούσιο κείμενο που χρησιμοποιεί αυτόματα το προεπιλεγμένο στυλ;

Το RichText αντιπροσωπεύει ένα τμήμα κειμένου που μπορεί να περιέχει πολλαπλές ακολουθίες με ατομική μορφοποίηση.

Δημιουργήστε ένα αντικείμενο RichText χωρίς να καθορίσετε ρητό μέγεθος γραμματοσειράς, βασιζόμενοι στο προεπιλεγμένο στυλ που ορίστηκε προηγουμένως. Το αντικείμενο θα αποδοθεί χρησιμοποιώντας το καθορισμένο μέγεθος γραμματοσειράς και τυχόν άλλα χαρακτηριστικά στυλ που έχετε ρυθμίσει, εξασφαλίζοντας συνεπή εμφάνιση σε όλες τις παραγράφους.

```java
RichText text = new RichText()
					.append("DefaultParagraphFontAndSize")
					.append(System.lineSeparator())
					.append("OnlyDefaultParagraphFont", new TextStyle().setFontSize(14))
					.append(System.lineSeparator())
					.append("OnlyDefaultParagraphFontSize", new TextStyle().setFontName("Verdana"));
text.setParagraphStyle(defaultStyle);
```

## Πώς να προσαρτήσω το πλούσιο κείμενο στο στοιχείο outline;

Η μέθοδος appendChildLast προσθέτει ένα παιδικό κόμβο στο τέλος της συλλογής ενός container.

Προσθέστε την παρουσία RichText στο προηγουμένως δημιουργημένο στοιχείο outline χρησιμοποιώντας τη μέθοδο appendChildLast. Αυτή η λειτουργία συνδέει το μορφοποιημένο περιεχόμενο με το outline, διατηρώντας την ιεραρχία και εξασφαλίζοντας ότι το κείμενο εμφανίζεται με τη σωστή σειρά στη σελίδα.

```java
outlineElem.appendChildLast(text);
```

## Πώς να συνδέσω το στοιχείο outline με τη σελίδα;

Η μέθοδος Page.appendChildLast προσθέτει ένα παιδικό στοιχείο όπως ένα Outline στη συλλογή της σελίδας.

Προσθέστε το outline στη συλλογή των outlines της σελίδας χρησιμοποιώντας τη μέθοδο appendChildLast. Αυτό το βήμα ενσωματώνει το μορφοποιημένο περιεχόμενό σας στη δομή της σελίδας, καθιστώντας το ορατό όταν το έγγραφο ανοίξει στο OneNote.

```java
outline.appendChildLast(outlineElem);
```

## Πώς να προσθέσω τη σελίδα στο έγγραφο OneNote;

Η μέθοδος Document.appendChildLast προσθέτει μια σελίδα ή άλλο στοιχείο στην ιεραρχία του σημειωματάριου.

Εισάγετε τη πλήρως προετοιμασμένη σελίδα στο αντικείμενο Document χρησιμοποιώντας τη μέθοδο appendChildLast. Σε αυτό το σημείο το έγγραφο περιέχει μια σελίδα με το προεπιλεγμένο στυλ παραγράφου που εφαρμόζεται παντού, έτοιμη για αποθήκευση.

```java
page.appendChildLast(outline);
```

## Πώς να αποθηκεύσω το έγγραφο OneNote στο δίσκο;

Η μέθοδος Document.save γράφει το σημειωματάριο σε ένα αρχείο στην καθορισμένη μορφή.

Αποθηκεύστε το Document ως αρχείο .one χρησιμοποιώντας τη μέθοδο save, παρέχοντας τη πλήρη διαδρομή όπου πρέπει να γραφτεί το αρχείο. Το παραγόμενο αρχείο θα ανοίξει στο OneNote με το προεπιλεγμένο μέγεθος γραμματοσειράς ήδη εφαρμοσμένο σε κάθε παράγραφο.

```java
document.appendChildLast(page);
```

## Ποιο είναι το τελικό βήμα για την επαλήθευση του αποθηκευμένου αρχείου;

Το άνοιγμα του αρχείου στο OneNote σας επιτρέπει να επιβεβαιώσετε οπτικά τα εφαρμοσμένα στυλ.

Ανοίξτε το παραγόμενο αρχείο .one στο Microsoft OneNote ή σε οποιονδήποτε συμβατό προβολέα. Θα πρέπει να δείτε όλες τις παραγράφους να αποδίδονται με το προεπιλεγμένο μέγεθος γραμματοσειράς που καθορίσατε, επιβεβαιώνοντας ότι το στυλ εφαρμόστηκε σωστά και ότι το έγγραφο φορτώνει χωρίς σφάλματα.

```java
document.save(Paths.get(dataDir, "SetDefaultParagraphStyle.one").toString());
```

Αυτός ο κώδικας θα ορίσει το προεπιλεγμένο στυλ παραγράφου στο OneNote χρησιμοποιώντας το Aspose.Note for Java, διασφαλίζοντας ότι κάθε νέα παράγραφος υιοθετεί αυτόματα το επιλεγμένο μέγεθος γραμματοσειράς και μορφοποίηση.

## Συνηθισμένα Προβλήματα και Λύσεις

- **Το στυλ παραγράφου δεν εφαρμόζεται:** Βεβαιωθείτε ότι έχετε ορίσει το στυλ στο αντικείμενο `Document` *πριν* δημιουργήσετε οποιεσδήποτε εμφανίσεις `RichText`.  
- **Απρόσμενο μέγεθος γραμματοσειράς:** Βεβαιωθείτε ότι χρησιμοποιείτε μονάδες σημείου (pt) για την ιδιότητα `fontSize`; η χρήση pixel μπορεί να προκαλέσει διαφορές κλιμάκωσης.  
- **Μεγάλα σημειωματάρια προκαλούν πίεση μνήμης:** Χρησιμοποιήστε `Document.load(inputStream, LoadOptions)` με `LoadOptions.setLoadMode(LoadMode.Stream)` για να επεξεργαστείτε μεγάλα αρχεία αποδοτικά.
