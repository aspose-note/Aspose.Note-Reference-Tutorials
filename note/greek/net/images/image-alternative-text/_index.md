# Πώς να Προσθέσετε Εναλλακτικό Κείμενο σε Εικόνες στο Aspose.Note

## Εισαγωγή

Αν χρειάζεστε **πώς να προσθέσετε εναλλακτικό κείμενο** για εικόνες στα έγγραφα τύπου OneNote, βρίσκεστε στο σωστό μέρος. Αυτό το tutorial σας καθοδηγεί βήμα προς βήμα για την προσθήκη εναλλακτικού κειμένου (και τίτλου και περιγραφής) σε μια εικόνα χρησιμοποιώντας το Aspose.Note για .NET. Η προσθήκη εναλλακτικού κειμένου όχι μόνο ενισχύει την προσβασιμότητα για χρήστες με προγράμματα ανάγνωσης οθόνης, αλλά βελτιώνει επίσης το SEO για οποιοδήποτε περιεχόμενο που δημοσιεύεται στο web και ενσωματώνει αυτές τις εικόνες.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει “alt text”;** Μια κειμενική περιγραφή που αντιπροσωπεύει μια εικόνα όταν δεν μπορεί να εμφανιστεί.  
- **Γιατί να χρησιμοποιήσετε το Aspose.Note για εναλλακτικό κείμενο;** Παρέχει ένα απλό API για να ορίσετε τόσο τον τίτλο όσο και την περιγραφή προγραμματιστικά.  
- **Ποια είναι τα προαπαιτούμενα;** Περιβάλλον ανάπτυξης .NET, Visual Studio, και εγκατεστημένο το Aspose.Note για .NET.  
- **Μπορώ να προσθέσω εναλλακτικό κείμενο σε υπάρχουσες εικόνες;** Ναι – μπορείτε να φορτώσετε ένα αντικείμενο εικόνας, να ορίσετε τις ιδιότητές του και να αποθηκεύσετε το έγγραφο.  
- **Πού αποθηκεύεται το αποτέλεσμα;** Στη διαδρομή που καθορίζετε με `document.Save(...)`.

## Τι είναι το “πώς να προσθέσετε εναλλακτικό κείμενο” στο Aspose.Note;

Η προσθήκη εναλλακτικού κειμένου σημαίνει την ανάθεση των ιδιοτήτων `AlternativeTextTitle` και `AlternativeTextDescription` ενός αντικειμένου `Image`. Αυτές οι ιδιότητες διαβάζονται από προγράμματα ανάγνωσης οθόνης και άλλες βοηθητικές τεχνολογίες για να μεταφέρουν το νόημα της εικόνας.

## Γιατί να προσθέσετε εναλλακτικό κείμενο σε εικόνα στο έγγραφό σας;

- **Συμμόρφωση προσβασιμότητας** – πληροί τις οδηγίες WCAG και Section 508.  
- **Βελτιωμένο SEO** – οι μηχανές αναζήτησης ευρετηριάζουν το περιγραφικό κείμενο.  
- **Καλύτερη εμπειρία χρήστη** – οι χρήστες με απενεργοποιημένες εικόνες εξακολουθούν να κατανοούν το περιεχόμενο.

## Προαπαιτούμενα

Πριν ξεκινήσετε, βεβαιωθείτε ότι έχετε:

- Βασικές γνώσεις C# και ανάπτυξης .NET.  
- Εγκατεστημένο το Visual Studio στον υπολογιστή σας.  
- Το Aspose.Note για .NET να είναι ληφθέν και αναφερθεί στο έργο σας – μπορείτε να το αποκτήσετε [εδώ](https://releases.aspose.com/note/net/).  
- Ένα αρχείο εικόνας (π.χ., `image.jpg`) που θέλετε να ενσωματώσετε.

## Εισαγωγή Namespaces

Πρώτα, συμπεριλάβετε τα namespaces που απαιτούνται για τη διαχείριση αρχείων και τα αντικείμενα Aspose.Note.

```csharp
using System.IO;
using Aspose.Note;
using System.Collections.Generic;
using System.Drawing;
using System;
```

## Βήμα 1: Αρχικοποίηση Εγγράφου και Σελίδας

Δημιουργήστε ένα νέο αντικείμενο `Document` και προσθέστε μια `Page` όπου θα τοποθετηθεί η εικόνα.

```csharp
var document = new Document();
var page = new Page(document);
```

## Βήμα 2: Φόρτωση της Εικόνας

Καθορίστε το φάκελο που περιέχει την εικόνα σας και δημιουργήστε ένα αντικείμενο `Image`.

```csharp
string dataDir = "Your Document Directory";
var image = new Image(document, dataDir + "image.jpg");
```

## Βήμα 3: Ορισμός Εναλλακτικού Κειμένου

Εδώ **προσθέτουμε εναλλακτικό κείμενο στην εικόνα** συμπληρώνοντας και τα πεδία τίτλου και περιγραφής.

```csharp
image.AlternativeTextTitle = "This is an image's title!";
image.AlternativeTextDescription = "And this is an image's description!";
```

## Βήμα 4: Προσθήκη Εικόνας στη Σελίδα

Τώρα **προσθέτουμε την εικόνα στη σελίδα** χρησιμοποιώντας τη μέθοδο `AppendChildLast`.

```csharp
page.AppendChildLast(image);
```

## Βήμα 5: Αποθήκευση Εγγράφου

Καθορίστε το όνομα του αρχείου εξόδου και αποθηκεύστε το έγγραφο OneNote.

```csharp
dataDir = dataDir + "ImageAlternativeText_out.one";
document.Save(dataDir);
```

## Βήμα 6: Εμφάνιση Μηνύματος Επιτυχίας

Ένα απλό μήνυμα κονσόλας επιβεβαιώνει ότι η λειτουργία ολοκληρώθηκε με επιτυχία.

```csharp
Console.WriteLine("\nImage alternative text setup successfully.\nFile saved at " + dataDir); 
```

## Συνηθισμένα Προβλήματα και Λύσεις

| Πρόβλημα | Αιτία | Διόρθωση |
|----------|-------|----------|
| **Το εναλλακτικό κείμενο δεν εμφανίζεται** | `AlternativeTextTitle` ή `AlternativeTextDescription` άδεια | Βεβαιωθείτε ότι και οι δύο ιδιότητες έχουν οριστεί πριν την αποθήκευση. |
| **Το αρχείο δεν βρέθηκε** | Λάθος διαδρομή `dataDir` | Χρησιμοποιήστε απόλυτη διαδρομή ή επαληθεύστε ότι ο σχετικός φάκελος υπάρχει. |
| **Εξαίρεση κατά την αποθήκευση** | Ανεπαρκή δικαιώματα εγγραφής | Εκτελέστε το Visual Studio ως διαχειριστής ή επιλέξτε φάκελο με δυνατότητα εγγραφής. |
