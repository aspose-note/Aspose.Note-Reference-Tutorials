# Ανάκτηση Μεταδεδομένων Εικόνας με Aspose.Note για .NET

Σε αυτό το πρακτικό tutorial θα μάθετε **πώς να εξάγετε μεταδεδομένα εικόνας**—συμπεριλαμβανομένου του πλάτους, του ύψους, του αρχικού μεγέθους, του ονόματος αρχείου και του χρόνου τροποποίησης—από αρχεία Microsoft OneNote χρησιμοποιώντας τη βιβλιοθήκη Aspose.Note για C#. Είτε χρειάζεστε να εμφανίσετε μικρογραφίες εικόνων, να επικυρώσετε τα μεγέθη των εικόνων, είτε να δημιουργήσετε έναν κατάλογο ενσωματωμένων πόρων, η εξαγωγή αυτών των πληροφοριών προγραμματιστικά σας εξοικονομεί αμέτρητα χειροκίνητα βήματα.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει “εξαγωγή μεταδεδομένων εικόνας”;** Ανάκτηση ιδιοτήτων όπως διαστάσεις, όνομα αρχείου και χρονικές σφραγίδες από εικόνες που αποθηκεύονται μέσα σε έγγραφο OneNote.  
- **Ποια βιβλιοθήκη το διαχειρίζεται;** Aspose.Note for .NET.  
- **Χρειάζομαι άδεια;** Μια δωρεάν δοκιμή λειτουργεί για ανάπτυξη· απαιτείται εμπορική άδεια για παραγωγή.  
- **Υποστηριζόμενες πλατφόρμες;** .NET Framework 4.6+, .NET Core 3.1+, .NET 5/6+.  
- **Τυπικός χρόνος εκτέλεσης;** Λιγότερο από ένα δευτερόλεπτο για ένα τυπικό αρχείο .one.

## Τι είναι η εξαγωγή μεταδεδομένων εικόνας;
Η εξαγωγή μεταδεδομένων εικόνας σημαίνει προγραμματιστική ανάγνωση των περιγραφικών χαρακτηριστικών (πλάτος, ύψος, αρχικές διαστάσεις, όνομα αρχείου, χρόνος τελευταίας τροποποίησης κ.λπ.) που συνοδεύουν κάθε αντικείμενο εικόνας μέσα σε μια σελίδα OneNote. Αυτά τα δεδομένα είναι χρήσιμα για επικύρωση, αναφορές ή περαιτέρω επεξεργασία εικόνας.

## Γιατί να εξάγετε μεταδεδομένα εικόνας από το OneNote;
- **Αυτοματοποίηση** – Δημιουργήστε εργαλεία που δημιουργούν αυτόματα αποθέματα εικόνων.  
- **Επικύρωση** – Διασφαλίστε ότι οι εικόνες πληρούν τις απαιτήσεις μεγέθους πριν τη δημοσίευση.  
- **Ενσωμάτωση** – Συνδυάστε το περιεχόμενο OneNote με άλλα συστήματα που χρειάζονται λεπτομέρειες εικόνας (CMS, DAM κ.λπ.).  
- **Απόδοση** – Αποφύγετε τη φόρτωση πλήρων δυαδικών εικόνων όταν χρειάζονται μόνο οι διαστάσεις.

## Προαπαιτούμενα
1. **Βασικές αρχές C#** – Θα πρέπει να αισθάνεστε άνετα γράφοντας βασικό κώδικα C#.  
2. **Aspose.Note για .NET** – Κατεβάστε και εγκαταστήστε τη βιβλιοθήκη από την επίσημη ιστοσελίδα [εδώ](https://releases.aspose.com/note/net/).  
3. **Αρχείο OneNote (.one)** – Οποιοδήποτε υπάρχον έγγραφο OneNote που περιέχει εικόνες.

## Εισαγωγή Χώρων Ονομάτων
Πριν ξεκινήσουμε, εισάγετε τους απαιτούμενους χώρους ονομάτων:

```csharp
using System.IO;
using Aspose.Note;
using System.Collections.Generic;
using System;
```

## Βήμα 1: Φόρτωση του εγγράφου OneNote
Πρώτα, φορτώστε το επιλεγμένο αρχείο OneNote σε ένα αντικείμενο `Aspose.Note.Document`. Αυτό είναι το βήμα **φόρτωσης εγγράφου onenote** που προετοιμάζει το API για περαιτέρω ερωτήματα.

```csharp
// The path to the documents directory.
string dataDir = "Your Document Directory";

// Load the document into Aspose.Note.
Document oneFile = new Document(dataDir + "Aspose.one");
```

Αντικαταστήστε το `"Your Document Directory"` με το πραγματικό φάκελο που περιέχει το αρχείο `.one` σας.

## Βήμα 2: Ανάκτηση όλων των κόμβων εικόνας
Τώρα θα **εξάγουμε εικόνες** τραβώντας κάθε κόμβο `Image` από το έγγραφο.

```csharp
// Get all Image nodes
IList<Aspose.Note.Image> images = oneFile.GetChildNodes<Aspose.Note.Image>();
```

Η μέθοδος `GetChildNodes<T>()` σαρώει ολόκληρη τη ιεραρχία του σημειωματάριου και επιστρέφει μια συλλογή αντικειμένων εικόνας.

## Βήμα 3: Επανάληψη σε κάθε εικόνα και **c# λήψη ιδιοτήτων εικόνας**
Θα κάνουμε βρόχο στη συλλογή και θα εκτυπώσουμε τα μεταδεδομένα, συμπεριλαμβανομένων των τιμών **λήψης διαστάσεων εικόνας** και **εξαγωγής αρχικού μεγέθους εικόνας**.

```csharp
foreach (Aspose.Note.Image image in images)
{
    Console.WriteLine("Width: {0}", image.Width);
    Console.WriteLine("Height: {0}", image.Height);
    Console.WriteLine("OriginalWidth: {0}", image.OriginalWidth);
    Console.WriteLine("OriginalHeight: {0}", image.OriginalHeight);
    Console.WriteLine("FileName: {0}", image.FileName);
    Console.WriteLine("LastModifiedTime: {0}", image.LastModifiedTime);
    Console.WriteLine();
}
```

Η έξοδος εμφανίζει το τρέχον πλάτος/ύψος κάθε εικόνας (όπως αποδίδεται στη σελίδα), τις αρχικές διαστάσεις που αποθηκεύονται στο αρχείο, το όνομα αρχείου και τη χρονική σήμανση της τελευταίας τροποποίησης.

## Συχνά Προβλήματα και Λύσεις
| Πρόβλημα | Αιτία | Διόρθωση |
|-------|--------|-----|
| **NullReferenceException** όταν το `images` είναι κενό | Το έγγραφο δεν περιέχει εικόνες | Επαληθεύστε ότι το αρχείο πηγής `.one` περιέχει ενσωματωμένες εικόνες. |
| **Λανθασμένες διαστάσεις** | Οι εικόνες κλιμακώνονται στο OneNote | Χρησιμοποιήστε `OriginalWidth`/`OriginalHeight` για να λάβετε το πραγματικό μέγεθος. |
| **Το FileName είναι κενό** | Η εικόνα επικολλήθηκε από το πρόχειρο | Το API μπορεί να μην έχει όνομα αρχείου· διαχειριστείτε τιμές `null` ή κενές συμβολοσειρές στον κώδικά σας. |

## Συχνές Ερωτήσεις

**Q: Είναι το Aspose.Note συμβατό με όλες τις εκδόσεις του Microsoft OneNote;**  
A: Το Aspose.Note υποστηρίζει μορφές .one, .onepkg και .onetoc2, καλύπτοντας τις περισσότερες εκδόσεις του OneNote από το 2007 και μετά.

**Q: Μπορώ να τροποποιήσω τις ιδιότητες της εικόνας μετά την εξαγωγή;**  
A: Ναι, μπορείτε να αλλάξετε ιδιότητες όπως `Width`, `Height` ή `FileName` και στη συνέχεια να αποθηκεύσετε το έγγραφο.

**Q: Λειτουργεί το Aspose.Note με .NET Core;**  
A: Απόλυτα. Η βιβλιοθήκη είναι πλήρως συμβατή με .NET Core, .NET 5 και .NET 6.

**Q: Υπάρχει διαθέσιμη δωρεάν δοκιμή;**  
A: Ναι, μπορείτε να κατεβάσετε μια δοκιμαστική έκδοση από τον ιστότοπο της Aspose για να εξερευνήσετε όλες τις δυνατότητες.

**Q: Πού μπορώ να βρω πρόσθετη βοήθεια ή υποστήριξη από την κοινότητα;**  
A: Επισκεφθείτε το φόρουμ Aspose.Note [εδώ](https://forum.aspose.com/c/note/28) για συμβουλές, δείγματα κώδικα και απαντήσεις από την κοινότητα.
