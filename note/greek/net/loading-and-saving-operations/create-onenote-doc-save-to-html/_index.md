# Αποθήκευση OneNote ως HTML χρησιμοποιώντας το Aspose.Note

## Εισαγωγή

Αν χρειάζεστε να **αποθηκεύσετε το OneNote ως HTML** για δημοσίευση στο web, αναφορές ή αρχειοθέτηση, το Aspose.Note για .NET σας παρέχει μια καθαρή, μόνο‑κώδικα λύση που λειτουργεί σε Windows, Linux και macOS χωρίς να απαιτείται το Microsoft Office. Σε αυτό το μάθημα θα μάθετε πώς να δημιουργήσετε ένα έγγραφο OneNote από το μηδέν, να διαχειριστείτε τις σελίδες του και να εξάγετε ολόκληρο το σημειωματάριο—ή επιλεγμένες σελίδες—σε HTML χρησιμοποιώντας διάφορες επιλογές. Ο οδηγός σας καθοδηγεί βήμα προς βήμα, εξηγεί γιατί κάθε ρύθμιση είναι σημαντική και παρουσιάζει πραγματικά σενάρια όπου η εξαγωγή σε HTML ξεχωρίζει, όπως η ενσωμάτωση περιεχομένου OneNote σε portal γνώσεων ή η δημιουργία στατικής τεκμηρίωσης.

## Γρήγορες Απαντήσεις
- **Ποια είναι η κύρια μέθοδος εξαγωγής;** Κλήστε `document.Save("output.html", SaveFormat.Html)` σε ένα αντικείμενο `Aspose.Note.Document`.  
- **Μπορώ να εξάγω μια μόνο σελίδα;** Ναι—χρησιμοποιήστε `document.Save(pageRange, "page.html", SaveFormat.Html)`.  
- **Ενσωματώνονται αυτόματα οι πόροι (εικόνες, CSS);** Από προεπιλογή αποθηκεύονται ως ξεχωριστά αρχεία· μπορείτε να τα ενσωματώσετε σε ένα memory stream με `HtmlSaveOptions`.  
- **Υποστηρίζεται το .NET Core;** Απόλυτα—το Aspose.Note λειτουργεί με .NET 5, .NET 6 και .NET Core 3.1+.  
- **Χρειάζομαι άδεια για ανάπτυξη;** Μια δωρεάν δοκιμή 30 ημερών λειτουργεί για δοκιμές· μια άδεια αφαιρεί τα υδατογράμματα αξιολόγησης.

## Τι είναι η «αποθήκευση onenote ως html»;

**«Save OneNote as HTML»** αναφέρεται στη διαδικασία μετατροπής ενός σημειωματάριου OneNote, το οποίο αποθηκεύεται σε ιδιόκτητη δυαδική μορφή, σε τυπικά αρχεία HTML που μπορούν να εμφανιστούν από οποιονδήποτε web browser. Η μετατροπή διατηρεί τη ιεραρχική δομή του σημειωματάριου, τη μορφοποίηση του κειμένου, τις εικόνες, τους πίνακες και άλλους πόρους, ενώ μετατρέπει τα ειδικά στοιχεία του OneNote σε ισοδύναμες ετικέτες HTML και στυλ CSS. Αυτό επιτρέπει στους προγραμματιστές να εμφανίζουν περιεχόμενο OneNote σε ιστοσελίδες, να το ενσωματώνουν σε συστήματα διαχείρισης περιεχομένου ή να το αρχειοθετούν σε μια καθολικά προσβάσιμη μορφή χωρίς να απαιτείται ο πελάτης OneNote.

## Γιατί να μετατρέψετε το OneNote σε HTML;

Το Aspose.Note μπορεί να εξάγει σε **50+ μορφές εξόδου** (συμπεριλαμβανομένων PDF, DOCX, PNG και HTML) και επεξεργάζεται σημειωματάρια με **μέχρι 500 σελίδες** χωρίς να φορτώνει ολόκληρο το αρχείο στη μνήμη, χάρη στην αρχιτεκτονική streaming. Η μετατροπή σε HTML σας επιτρέπει:

1. **Δημοσιεύστε περιεχόμενο στο web χωρίς να απαιτείται ο πελάτης OneNote.**  
2. **Ενσωματώστε σε συστήματα διαχείρισης περιεχομένου που δέχονται μόνο HTML.**  
3. **Εξάγετε κείμενο αναζητήσιμο για μηχανές ευρετηρίασης.**  
4. **Επαναχρησιμοποιήστε πόρους επιπέδου σελίδας (εικόνες, γραμματοσειρές) σε άλλα web projects.**

## Πώς να αποθηκεύσετε το onenote ως html – Βήμα προς Βήμα

Η αποθήκευση του OneNote ως HTML περιλαμβάνει τρεις βασικές ενέργειες: φόρτωση ή δημιουργία ενός σημειωματάριου, διαμόρφωση των επιλογών εξαγωγής και κλήση της κατάλληλης μεθόδου `Save`. Αρχικά, δημιουργείτε ή φορτώνετε ένα αντικείμενο `Document`, στη συνέχεια προσαρμόζετε το `HtmlSaveOptions` για να ελέγξετε τη διαχείριση πόρων, τη δημιουργία CSS και την κωδικοποίηση. Τέλος, καλείτε `document.Save` με διαδρομή αρχείου ή stream, προαιρετικά καθορίζοντας ένα `PageRange` για περιορισμό της εξαγωγής. Αυτή η συστηματική προσέγγιση εξασφαλίζει συνεπή αποτελέσματα σε διαφορετικά περιβάλλοντα και σας επιτρέπει να προσαρμόσετε την έξοδο στις ανάγκες του έργου σας.

### Βήμα 1: Δημιουργία εγγράφου OneNote με προεπιλεγμένες επιλογές

Σε αυτό το πρώτο παράδειγμα δημιουργούμε ένα νέο `Document`, προσθέτουμε μια σελίδα με τίτλο και αποθηκεύουμε ολόκληρο το σημειωματάριο σε HTML χρησιμοποιώντας τις προεπιλεγμένες `HtmlSaveOptions`. Δεν απαιτούνται επιπλέον παράμετροι, καθιστώντας το ο πιο γρήγορος τρόπος για **αποθήκευση OneNote ως HTML**.

```csharp
using System;
using System.Drawing;
using System.Globalization;
using System.IO;

using Aspose.Note.Saving;
using Aspose.Note.Saving.Html;

```

### Βήμα 2: Εξαγωγή συγκεκριμένου εύρους σελίδων σε HTML

Συχνά χρειάζεται μόνο ένα υποσύνολο σελίδων—π.χ., η εξαγωγή ενός κεφαλαίου σημειώσεων συνάντησης. Με τη μεταβίβαση ενός αντικειμένου `PageRange` στη `Save`, μπορείτε να στοχεύσετε τις σελίδες 1‑3 ενώ οι υπόλοιπες παραμένουν αμετάβλητες.

```csharp
public static void CreateAndSaveToHTMLUsingDefaultOptions()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateOneNoteDocAndSaveToHTML_out.html");
    doc.Save(outputPath);

    Console.WriteLine("\nOneNote document created successfully.\nFile saved at " + outputPath);
}
```

### Βήμα 3: Αποθήκευση ως HTML σε Memory Stream με ενσωματωμένους πόρους

Όταν θέλετε ένα μοναδικό, αυτό‑συμπαγές HTML payload (π.χ., για απαντήσεις API), ενσωματώστε CSS, γραμματοσειρές και εικόνες απευθείας σε ένα `MemoryStream`. Το `HtmlSaveOptions` σας επιτρέπει να θέσετε το `EmbedResources` σε `true`, παράγοντας μια συμπαγή HTML συμβολοσειρά.

```csharp
public static void CreateAndSavePageRange()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateAndSavePageRange_out.html");
    doc.Save(outputPath, new HtmlSaveOptions { PageCount = 1, PageIndex = 0 });

    Console.WriteLine("\nOneNote document created successfully and saved as page range.\nFile saved at " + outputPath);
}
```

### Βήμα 4: Αποθήκευση ως HTML σε αρχεία με πόρους σε ξεχωριστά αρχεία

Αν προτιμάτε την κλασική έξοδο web‑folder—αρχείο HTML συν φάκελο περιουσιακών στοιχείων—απενεργοποιήστε την ενσωμάτωση και ορίστε έναν φάκελο εξόδου. Αυτή η προσέγγιση προσφέρει πιο καθαρά αρχεία πηγής και ευκολότερη προσωρινή αποθήκευση εικόνων ή φύλλων στυλ.

```csharp
public static void SaveAsHTMLToMemoryStreamWithEmbeddedResources()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportEmbedded,
        ExportFonts = ResourceExportType.ExportEmbedded,
        ExportImages = ResourceExportType.ExportEmbedded,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to a memory stream
    var memoryStream = new MemoryStream();
    document.Save(memoryStream, options);
}
```

### Βήμα 5: Αποθήκευση ως HTML χρησιμοποιώντας callbacks για διαχείριση πόρων

Σύνθετα σενάρια, όπως η αποθήκευση εικόνων σε βάση δεδομένων ή σε cloud bucket, απαιτούν προσαρμοσμένη διαχείριση. Παρέχοντας έναν delegate `ResourceSavingCallback`, ελέγχετε ακριβώς πώς κάθε πόρος αποθηκεύεται, ενώ το API συνεχίζει να δημιουργεί σωστές αναφορές HTML.

```csharp
public static void SaveAsHTMLToFileWithResourcesInSeparateFiles()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportAsStream,
        ExportFonts = ResourceExportType.ExportAsStream,
        ExportImages = ResourceExportType.ExportAsStream,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to HTML file with resources stored in separate files
    document.Save(Path.Combine(dataDir, "document_out.html"), options);
}
```

```csharp
public static void SaveAsHTMLToMemoryStreamWithCallBacksToSaveResources()
{
    // Specify the saving callbacks configuration
    var savingCallbacks = new UserSavingCallbacks()
    {
        RootFolder = "documentFolder",
        CssFolder = "css",
        KeepCssStreamOpened = true,
        ImagesFolder = "images",
        FontsFolder = "fonts"
    };

    // Specify HTML saving options
    var options = new HtmlSaveOptions
    {
        FontFaceTypes = FontFaceType.Ttf,
        CssSavingCallback = savingCallbacks,
        FontSavingCallback = savingCallbacks,
        ImageSavingCallback = savingCallbacks
    };

    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Save the document to HTML format with resources managed by user-defined callbacks
    using (var stream = File.Create(Path.Combine(savingCallbacks.RootFolder, "document.html")))
    {
        document.Save(stream, options);
    }

    // Manually append data to the CSS stream
    using (var writer = new StreamWriter(savingCallbacks.CssStream))
    {
        writer.WriteLine();
        writer.WriteLine("/* This line is appended to stream manually by user */");
    }
}
```
