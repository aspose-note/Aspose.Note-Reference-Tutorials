**Q: Can I use this integration in a commercial application?**

**Q: Does the API support importing recurring Outlook tasks?**  
A: Absolutely. Recurrence patterns are preserved when you map the Outlook `Task` object to a OneNote task.

**Q: What version of Java is required?**  
A: Java 8 or higher is recommended for optimal performance and compatibility.

**Q: How do I handle large numbers of tasks?**  
A: Process them in batches and release resources after each batch to avoid memory pressure.

**Q: Is there a way to update a OneNote task when the Outlook task changes?**
