# Guardar OneNote como HTML usando Aspose.Note

## Introducción

Si necesitas **guardar OneNote como HTML** para publicación web, informes o archivado, Aspose.Note para .NET te ofrece una solución limpia, solo código, que funciona en Windows, Linux y macOS sin requerir Microsoft Office. En este tutorial aprenderás a crear un documento OneNote desde cero, manipular sus páginas y exportar todo el cuaderno —o páginas seleccionadas— a HTML usando una variedad de opciones. La guía te acompaña paso a paso, explica por qué cada configuración es importante y muestra escenarios reales donde la exportación a HTML destaca, como incrustar contenido de OneNote en un portal de base de conocimientos o generar documentación estática.

## Respuestas rápidas
- **¿Cuál es el método principal para exportar?** Llame a `document.Save("output.html", SaveFormat.Html)` en una instancia de `Aspose.Note.Document`.  
- **¿Puedo exportar una sola página?** Sí—use `document.Save(pageRange, "page.html", SaveFormat.Html)`.  
- **¿Los recursos (imágenes, CSS) se incrustan automáticamente?** Por defecto se guardan como archivos separados; puedes incrustarlos en un flujo de memoria con `HtmlSaveOptions`.  
- **¿Se admite .NET Core?** Absolutamente—Aspose.Note funciona con .NET 5, .NET 6 y .NET Core 3.1+.  
- **¿Necesito una licencia para desarrollo?** Una prueba gratuita de 30 días funciona para pruebas; una licencia elimina las marcas de agua de evaluación.

## ¿Qué es “guardar onenote como html”?

**“Save OneNote as HTML”** se refiere al proceso de convertir un cuaderno de OneNote, que se almacena en un formato binario propietario, en archivos HTML estándar que pueden ser renderizados por cualquier navegador web. La conversión preserva la estructura jerárquica del cuaderno, el formato de texto, imágenes, tablas y otros recursos, mientras traduce los elementos específicos de OneNote a etiquetas HTML equivalentes y estilos CSS. Esto permite a los desarrolladores mostrar contenido de OneNote en sitios web, integrarlo en sistemas de gestión de contenido o archivarlo en un formato universalmente accesible sin requerir el cliente de OneNote.

## ¿Por qué convertir OneNote a HTML?

Aspose.Note puede exportar a **más de 50 formatos de salida** (incluidos PDF, DOCX, PNG y HTML) y procesa cuadernos con **hasta 500 páginas** sin cargar todo el archivo en memoria, gracias a su arquitectura de transmisión. Convertir a HTML te permite:

1. **Publicar contenido en la web** sin requerir el cliente de OneNote.  
2. **Integrar con sistemas de gestión de contenido** que solo aceptan HTML.  
3. **Extraer texto indexable** para motores de búsqueda.  
4. **Reutilizar recursos a nivel de página** (imágenes, fuentes) en otros proyectos web.

## Cómo guardar onenote como html – Paso a paso

Guardar OneNote como HTML implica tres acciones principales: cargar o crear un cuaderno, configurar las opciones de exportación deseadas y llamar al método `Save` correspondiente. Primero, creas o cargas un objeto `Document`, luego puedes ajustar `HtmlSaveOptions` para controlar el manejo de recursos, la generación de CSS y la codificación. Finalmente, llamas a `document.Save` con una ruta de archivo o un flujo, opcionalmente especificando un `PageRange` para limitar la exportación. Este enfoque sistemático garantiza resultados consistentes en diferentes entornos y te permite adaptar la salida a las necesidades de tu proyecto.

### Paso 1: Crear un documento OneNote con opciones predeterminadas

En este primer ejemplo instanciamos un `Document` nuevo, añadimos una sola página con un título y guardamos todo el cuaderno en HTML usando las `HtmlSaveOptions` predeterminadas. No se requieren parámetros adicionales, lo que lo convierte en la forma más rápida de **guardar OneNote como HTML**.

```csharp
using System;
using System.Drawing;
using System.Globalization;
using System.IO;

using Aspose.Note.Saving;
using Aspose.Note.Saving.Html;

```

### Paso 2: Exportar un rango de páginas específico a HTML

A menudo solo necesitas un subconjunto de páginas—por ejemplo, exportar un capítulo de notas de reunión. Al pasar un objeto `PageRange` a `Save`, puedes apuntar a las páginas 1‑3 mientras dejas el resto sin tocar.

```csharp
public static void CreateAndSaveToHTMLUsingDefaultOptions()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateOneNoteDocAndSaveToHTML_out.html");
    doc.Save(outputPath);

    Console.WriteLine("\nOneNote document created successfully.\nFile saved at " + outputPath);
}
```

### Paso 3: Guardar como HTML en un MemoryStream con recursos incrustados

Cuando deseas una única carga HTML autocontenida (p. ej., para respuestas de API), incrusta CSS, fuentes e imágenes directamente en un `MemoryStream`. `HtmlSaveOptions` te permite establecer `EmbedResources` a `true`, generando una cadena HTML compacta.

```csharp
public static void CreateAndSavePageRange()
{
    // Initialize OneNote document
    Document doc = new Document();
    Page page = doc.AppendChildLast(new Page());

    // Default style for all text in the document.
    ParagraphStyle textStyle = new ParagraphStyle { FontColor = Color.Black, FontName = "Arial", FontSize = 10 };
    page.Title = new Title()
    {
        TitleText = new RichText() { Text = "Title text.", ParagraphStyle = textStyle },
        TitleDate = new RichText() { Text = new DateTime(2011, 11, 11).ToString("D", CultureInfo.InvariantCulture), ParagraphStyle = textStyle },
        TitleTime = new RichText() { Text = "12:34", ParagraphStyle = textStyle }
    };

    // Save into HTML format
    string dataDir = "Your Document Directory";
    string outputPath = Path.Combine(dataDir, "CreateAndSavePageRange_out.html");
    doc.Save(outputPath, new HtmlSaveOptions { PageCount = 1, PageIndex = 0 });

    Console.WriteLine("\nOneNote document created successfully and saved as page range.\nFile saved at " + outputPath);
}
```

### Paso 4: Guardar como HTML en archivos con recursos en archivos separados

Si prefieres la salida clásica de carpeta web—archivo HTML más una carpeta de recursos—desactiva la incrustación y especifica un directorio de salida. Este enfoque produce archivos fuente más limpios y facilita el almacenamiento en caché de imágenes o hojas de estilo.

```csharp
public static void SaveAsHTMLToMemoryStreamWithEmbeddedResources()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportEmbedded,
        ExportFonts = ResourceExportType.ExportEmbedded,
        ExportImages = ResourceExportType.ExportEmbedded,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to a memory stream
    var memoryStream = new MemoryStream();
    document.Save(memoryStream, options);
}
```

### Paso 5: Guardar como HTML usando callbacks para gestionar recursos

Escenarios avanzados, como almacenar imágenes en una base de datos o en un bucket en la nube, requieren un manejo personalizado. Al proporcionar un delegado `ResourceSavingCallback`, controlas exactamente cómo se persiste cada recurso, mientras la API sigue generando referencias HTML correctas.

```csharp
public static void SaveAsHTMLToFileWithResourcesInSeparateFiles()
{
    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Specify HTML saving options
    var options = new HtmlSaveOptions()
    {
        ExportCss = ResourceExportType.ExportAsStream,
        ExportFonts = ResourceExportType.ExportAsStream,
        ExportImages = ResourceExportType.ExportAsStream,
        FontFaceTypes = FontFaceType.Ttf
    };

    // Save the document to HTML file with resources stored in separate files
    document.Save(Path.Combine(dataDir, "document_out.html"), options);
}
```

## Problemas comunes y soluciones

| Problema | Por qué ocurre | Solución |
|----------|----------------|----------|
| **Las imágenes aparecen rotas** | Los recursos se guardaron en una carpeta diferente a la del archivo HTML. | Asegúrate de que `HtmlSaveOptions.ResourcesFolder` coincida con la carpeta que referencias, o habilita `EmbedResources`. |
| **Faltan estilos CSS** | La generación de CSS predeterminada estaba desactivada. | Establece `HtmlSaveOptions.GenerateCss = true` (valor predeterminado) o incrusta CSS manualmente. |
| **Los cuadernos grandes provocan OutOfMemory** | Usar el constructor `Document` no streaming carga todo el archivo. | Usa `Document.Load(stream, LoadOptions)` con `LoadOptions.LoadFromMemory = true` para transmitir páginas. |
| **Problemas de codificación HTML para caracteres no ASCII** | Se estableció una `Encoding` incorrecta en `HtmlSaveOptions`. | Establece `HtmlSaveOptions.Encoding = Encoding.UTF8`. |
| **Falla al guardar en Linux** | Faltan dependencias nativas para el procesamiento de imágenes. | Instala `libgdiplus` en el host Linux. |

```csharp
public static void SaveAsHTMLToMemoryStreamWithCallBacksToSaveResources()
{
    // Specify the saving callbacks configuration
    var savingCallbacks = new UserSavingCallbacks()
    {
        RootFolder = "documentFolder",
        CssFolder = "css",
        KeepCssStreamOpened = true,
        ImagesFolder = "images",
        FontsFolder = "fonts"
    };

    // Specify HTML saving options
    var options = new HtmlSaveOptions
    {
        FontFaceTypes = FontFaceType.Ttf,
        CssSavingCallback = savingCallbacks,
        FontSavingCallback = savingCallbacks,
        ImageSavingCallback = savingCallbacks
    };

    // Load the OneNote document
    string dataDir = "Your Document Directory";
    var document = new Document(Path.Combine(dataDir, "Aspose.one"));

    // Save the document to HTML format with resources managed by user-defined callbacks
    using (var stream = File.Create(Path.Combine(savingCallbacks.RootFolder, "document.html")))
    {
        document.Save(stream, options);
    }

    // Manually append data to the CSS stream
    using (var writer = new StreamWriter(savingCallbacks.CssStream))
    {
        writer.WriteLine();
        writer.WriteLine("/* This line is appended to stream manually by user */");
    }
}
```
