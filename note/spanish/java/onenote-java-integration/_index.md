## ¿Cómo recupero datos de OneNote usando Java?
Use `Notebook.load("my.notebook")` para abrir un cuaderno de OneNote; `Notebook` representa el archivo del cuaderno. Recupera un objeto `Page` (una sola página en el cuaderno) y llama a `page.getAttachments()`, que devuelve una colección de objetos `Attachment` (archivos incrustados) que pueden transmitirse al disco. Esta operación normalmente finaliza en menos de un segundo para cuadernos con menos de 500 páginas y puede procesarse página por página para mantener la memoria por debajo de 100 MB.

## ¿Qué es Aspose.Note para Java?
`Aspose.Note` es una biblioteca pure‑Java que lee, escribe y manipula archivos Microsoft OneNote (.one, .onepkg) sin requerir la aplicación OneNote. Soporta **30+ formatos de archivo** para importación/exportación y puede manejar cuadernos con **hasta 10,000 páginas** mientras mantiene una huella de memoria baja mediante APIs de transmisión.

## ¿Por qué debería adjuntar archivos programáticamente en OneNote?
El adjunto programático te permite incrustar PDFs, imágenes o hojas de cálculo directamente en una nota, preservando el contexto y facilitando mucho el procesamiento posterior (búsqueda, indexación o exportación). Usando la API puedes establecer un ícono personalizado para cada adjunto, lo que mejora la navegación visual para los usuarios finales y elimina pasos manuales propensos a errores.

## Preguntas frecuentes

**Q: ¿Puedo usar Aspose.Note para Java en un proyecto comercial?**  
A: Sí, se requiere una licencia comercial para uso en producción, pero hay una prueba gratuita disponible para evaluación.

**Q: ¿Qué versiones de Java son compatibles?**  
A: La biblioteca soporta Java 8 y versiones posteriores, incluyendo Java 11, 17 y 21.

**Q: ¿Cómo manejo archivos grandes de OneNote al recuperar adjuntos?**  
A: Carga el cuaderno en modo de transmisión o procesa secciones/páginas individualmente para mantener el uso de memoria por debajo de 100 MB.

**Q: ¿Es posible establecer un ícono personalizado para un archivo adjunto?**  
A: Absolutamente – el tutorial “Adjuntar archivo y establecer ícono” te muestra cómo especificar un ícono programáticamente.

**Q: ¿Necesito instalar OneNote en el servidor para usar la API?**  
A: No, Aspose.Note funciona de forma independiente de la aplicación OneNote; lee y escribe formatos de archivo OneNote directamente.
